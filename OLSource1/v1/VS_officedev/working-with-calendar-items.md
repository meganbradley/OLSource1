# Working with Calendar Items
  The \<xref:Microsoft.Office.Interop.Outlook.AppointmentItem> class contains the methods and properties you use to perform tasks with calendar items.  
  
|Task|Procedure|  
|----------|---------------|  
|Create an appointment and add recipients.|[How to: Programmatically Create Appointments](../VS_officedev/how-to--programmatically-create-appointments.md)|  
|Delete an instance of a recurring appointment.|[How to: Programmatically Delete Appointments](../VS_officedev/how-to--programmatically-delete-appointments.md)|  
|Create a Calendar folder and add an Appointment item.|[How to: Programmatically Create a Custom Calendar](../VS_officedev/how-to--programmatically-create-a-custom-calendar.md)|  
|Create a meeting request and send it.|[How to: Programmatically Create a Meeting Request](../VS_officedev/how-to--programmatically-create-a-meeting-request.md)|  
  
 For more information about Outlook tasks and the Outlook object model, see [Outlook Object Model Overview](../VS_officedev/outlook-object-model-overview.md).
