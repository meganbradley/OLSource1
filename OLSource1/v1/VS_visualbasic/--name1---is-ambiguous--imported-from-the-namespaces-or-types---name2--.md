# &#39;&lt;name1&gt;&#39; is ambiguous, imported from the namespaces or types &#39;&lt;name2&gt;&#39;
You have provided a name that is ambiguous and therefore conflicts with another name. The [!INCLUDE[vbprvb](../VS_visualbasic/includes/vbprvb_md.md)] compiler does not have any conflict resolution rules; you must disambiguate names yourself.  
  
 **Error ID:** BC30561  
  
### To correct this error  
  
1.  Disambiguate the name by removing namespace imports.  
  
2.  Fully qualify the name.
