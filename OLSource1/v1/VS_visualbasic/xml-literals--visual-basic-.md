# XML Literals (Visual Basic)
The topics in this section document the syntax of XML literals in [!INCLUDE[vbprvb](../VS_visualbasic/includes/vbprvb_md.md)]. The XML literal syntax enables you to incorporate XML directly in your code.  
  
## In This Section  
  
|Topic|Description|  
|-----------|-----------------|  
|[XML Element Literal](../VS_visualbasic/xml-element-literal--visual-basic-.md)|Describes the syntax for literals that represent \<xref:System.Xml.Linq.XElement> objects.|  
|[XML Document Literal](../VS_visualbasic/xml-document-literal--visual-basic-.md)|Describes the syntax for literals that represent \<xref:System.Xml.Linq.XDocument> objects.|  
|[XML CDATA Literal](../VS_visualbasic/xml-cdata-literal--visual-basic-.md)|Describes the syntax for literals that represent \<xref:System.Xml.Linq.XCData> objects.|  
|[XML Comment Literal](../VS_visualbasic/xml-comment-literal--visual-basic-.md)|Describes the syntax for literals that represent \<xref:System.Xml.Linq.XComment> objects.|  
|[XML Processing Instruction Literal](../VS_visualbasic/xml-processing-instruction-literal--visual-basic-.md)|Describes the syntax for literals that represent \<xref:System.Xml.Linq.XProcessingInstruction> objects.|
