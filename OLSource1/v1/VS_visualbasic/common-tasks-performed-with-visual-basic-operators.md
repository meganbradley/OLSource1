# Common Tasks Performed with Visual Basic Operators
Operators perform many common tasks involving one or more expressions called *operands*.  
  
## Arithmetic and Bit-shift Tasks  
 The following table summarizes the available arithmetic and bit-shift operations.  
  
|||  
|-|-|  
|To|See|  
|Add one numeric value to another|[+ Operator](../VS_visualbasic/--operator--visual-basic-.md)|  
|Subtract one numeric value from another|[- Operator (Visual Basic)](../VS_visualbasic/--operator--visual-basic-1.md)|  
|Reverse the sign of a numeric value|[- Operator (Visual Basic)](../VS_visualbasic/--operator--visual-basic-1.md)|  
|Multiply one numeric value by another|[* Operator](../VS_visualbasic/--operator--visual-basic-.md)|  
|Divide one numeric value into another|[/ Operator (Visual Basic)](../VS_visualbasic/--operator--visual-basic-3.md)|  
|Find the quotient of one numeric value divided by another (without the remainder)|[\ Operator (Visual Basic)](../VS_visualbasic/--operator--visual-basic-2.md)|  
|Find the remainder of one numeric value divided by another (without the quotient)|[Mod Operator](../VS_visualbasic/mod-operator--visual-basic-.md)|  
|Raise one numeric value to the power of another|[^ Operator](../VS_visualbasic/^-operator--visual-basic-.md)|  
|Shift the bit pattern of a numeric value to the left|[<\< Operator](../VS_visualbasic/---operator--visual-basic-.md)|  
|Shift the bit pattern of a numeric value to the right|[>> Operator](../VS_visualbasic/---operator--visual-basic-.md)|
