# Query Keywords (C# Reference)
This section contains the contextual keywords used in query expressions.  
  
## In This Section  
  
|Clause|Description|  
|------------|-----------------|  
|[from](../VS_csharp/from-clause--csharp-reference-.md)|Specifies a data source and a range variable (similar to an iteration variable).|  
|[where](../VS_csharp/where-clause--csharp-reference-.md)|Filters source elements based on one or more Boolean expressions separated by logical AND and OR operators ( `&&` or `&#124;&#124;` ).|  
|[select](../VS_csharp/select-clause--csharp-reference-.md)|Specifies the type and shape that the elements in the returned sequence will have when the query is executed.|  
|[group](../VS_csharp/group-clause--csharp-reference-.md)|Groups query results according to a specified key value.|  
|[into](../VS_csharp/into--csharp-reference-.md)|Provides an identifier that can serve as a reference to the results of a join, group or select clause.|  
|[orderby](../VS_csharp/orderby-clause--csharp-reference-.md)|Sorts query results in ascending or descending order based on the default comparer for the element type.|  
|[join](../VS_csharp/join-clause--csharp-reference-.md)|Joins two data sources based on an equality comparison between two specified matching criteria.|  
|[let](../VS_csharp/let-clause--csharp-reference-.md)|Introduces a range variable to store sub-expression results in a query expression.|  
|[in](../VS_csharp/in--csharp-reference-.md)|Contextual keyword in a [join](../VS_csharp/join-clause--csharp-reference-.md) clause.|  
|[on](../VS_csharp/on--csharp-reference-.md)|Contextual keyword in a [join](../VS_csharp/join-clause--csharp-reference-.md) clause.|  
|[equals](../VS_csharp/equals--csharp-reference-.md)|Contextual keyword in a [join](../VS_csharp/join-clause--csharp-reference-.md) clause.|  
|[by](../VS_csharp/by--csharp-reference-.md)|Contextual keyword in a [group](../VS_csharp/group-clause--csharp-reference-.md) clause.|  
|[ascending](../VS_csharp/ascending--csharp-reference-.md)|Contextual keyword in an [orderby](../VS_csharp/orderby-clause--csharp-reference-.md) clause.|  
|[descending](../VS_csharp/descending--csharp-reference-.md)|Contextual keyword in an [orderby](../VS_csharp/orderby-clause--csharp-reference-.md) clause.|
