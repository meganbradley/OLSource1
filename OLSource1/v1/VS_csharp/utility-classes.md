# Utility Classes
The following MFC-independent utility classes are provided:  
  
-   [CImage](../VS_csharp/cimage-class.md) Provides enhanced bitmap support, including the ability to load and save images in JPEG, GIF, BMP, and Portable Network Graphics (PNG) formats.  
  
-   [CPoint](../VS_csharp/cpoint-class.md) Provides an implementation for storing coordinate (x, y) pairs.  
  
-   [CRect](../VS_csharp/crect-class.md) Provides an implementation for storing coordinates of rectangular areas.  
  
-   [CSize](../VS_csharp/csize-class.md) Provides an implementation for storing distance, relative positions, or paired values.  
  
-   [CString](../VS_csharp/cstringt-class.md) Provides an implementation for storing character strings.  
  
-   [CAdapt](../VS_csharp/cadapt-class.md) A simple template used to wrap classes that redefine the address-of operator.  
  
-   [_U_RECT](../VS_csharp/_u_rect-class.md) An argument adapter class that allows either `RECT` pointers or references to be passed to a function that is implemented in terms of pointers.
