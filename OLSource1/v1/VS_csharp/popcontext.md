# POPCONTEXT
Restores part or all of the current `context` (saved by the [PUSHCONTEXT](../VS_csharp/pushcontext.md) directive). The `context` can be **ASSUMES**, `RADIX`, **LISTING**, **CPU**, or **ALL**.  
  
## Syntax  
  
```  
  
POPCONTEXT context  
```
