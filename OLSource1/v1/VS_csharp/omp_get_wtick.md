# omp_get_wtick
Returns the number of seconds between processor clock ticks.  
  
## Syntax  
  
```  
double omp_get_wtick( );  
```  
  
## Remarks  
 For more information, see [3.3.2 omp_get_wtick Function](../VS_csharp/3.3.2-omp_get_wtick-function.md).
