# Message Options Dialog Box
Use this dialog box to select which messages are listed in [Messages View](../VS_csharp/messages-view.md). To display this dialog box, choose **Log Messages** from the **Spy** menu.  
  
## In This Section  
 [Windows Tab, Message Options Dialog Box](../VS_csharp/windows-tab--message-options-dialog-box.md)  
 Used to select the types of messages to list. Includes the Finder Tool.  
  
 [Messages Tab, Message Options Dialog Box](../VS_csharp/messages-tab--message-options-dialog-box.md)  
 Used to select message types for viewing.  
  
 [Output Tab, Message Options Dialog Box](../VS_csharp/output-tab--message-options-dialog-box.md)  
 Used to specify what message data to display.
