# exp10 Function
Calculates the base-10 exponential of the argument  
  
## Syntax  
  
```  
inline float exp10(  
   float _X  
) restrict(amp);  
inline double exp10(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the base-10 exponential of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
