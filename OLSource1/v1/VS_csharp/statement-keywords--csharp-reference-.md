# Statement Keywords (C# Reference)
Statements are program instructions. Except as described in the topics referenced in the following table, statements are executed in sequence. The following table lists the C# statement keywords. For more information about statements that are not expressed with any keyword, see [Statements (C# Programming Guide)](../VS_csharp/statements--csharp-programming-guide-.md).

|[Iteration statements](../VS_csharp/iteration-statements--csharp-reference-.md)|[do](../VS_csharp/do--csharp-reference-.md), [for](../VS_csharp/for--csharp-reference-.md), [foreach](../VS_csharp/foreach--in--csharp-reference-.md), [in](../VS_csharp/foreach--in--csharp-reference-.md), [while](../VS_csharp/while--csharp-reference-.md)|
