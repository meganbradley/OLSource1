# .LISTALL
Starts listing of all statements.  
  
## Syntax  
  
```  
  
.LISTALL  
  
```  
  
## Remarks  
 Equivalent to the combination of [.LIST](../VS_csharp/.list.md), [.LISTIF](../VS_csharp/.listif.md), and [.LISTMACROALL](../VS_csharp/.listmacroall.md).
