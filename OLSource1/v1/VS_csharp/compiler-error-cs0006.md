# Compiler Error CS0006
Metadata file 'dll_name' could not be found  
  
 The program was compiled and explicitly passed the name of a file that contained metadata; however, the .dll was not found. For more information, see [/reference (C# Compiler Options)](../VS_csharp/-reference--csharp-compiler-options-.md).
