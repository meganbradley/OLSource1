# Performance (LINQ to XML) (C#)
This section provides information about performance in [!INCLUDE[sqltecxlinq](../VS_csharp/includes/sqltecxlinq_md.md)], specifically the performance of functional construction and queries.  
  
## In This Section

 [Atomized XName and XNamespace Objects (LINQ to XML) (C#)](../VS_csharp/atomized-xname-and-xnamespace-objects--linq-to-xml---csharp-.md)
