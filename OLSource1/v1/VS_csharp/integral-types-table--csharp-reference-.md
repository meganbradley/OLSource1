# Integral Types Table (C# Reference)
The following table shows the sizes and ranges of the integral types, which constitute a subset of simple types.  
  
|Type|Range|Size|  
|----------|-----------|----------|  
|[sbyte](../VS_csharp/sbyte--csharp-reference-.md)|-128 to 127|Signed 8-bit integer|  
|[byte](../VS_csharp/byte--csharp-reference-.md)|0 to 255|Unsigned 8-bit integer|  
|[char](../VS_csharp/char--csharp-reference-.md)|U+0000 to U+ffff|Unicode 16-bit character|  
|[short](../VS_csharp/short--csharp-reference-.md)|-32,768 to 32,767|Signed 16-bit integer|  
|[ushort](../VS_csharp/ushort--csharp-reference-.md)|0 to 65,535|Unsigned 16-bit integer|  
|[int](../VS_csharp/int--csharp-reference-.md)|-2,147,483,648 to 2,147,483,647|Signed 32-bit integer|  
|[uint](../VS_csharp/uint--csharp-reference-.md)|0 to 4,294,967,295|Unsigned 32-bit integer|  
|[long](../VS_csharp/long--csharp-reference-.md)|-9,223,372,036,854,775,808 to 9,223,372,036,854,775,807|Signed 64-bit integer|  
|[ulong](../VS_csharp/ulong--csharp-reference-.md)|0 to 18,446,744,073,709,551,615|Unsigned 64-bit integer|  
  
 If the value represented by an integer literal exceeds the range of `ulong`, a compilation error will occur.
