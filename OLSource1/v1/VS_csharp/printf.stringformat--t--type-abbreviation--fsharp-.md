# Printf.StringFormat&lt;&#39;T&gt; Type Abbreviation (F#)
Represents a statically-analyzed format when formatting builds a string. The type parameter indicates the arguments and return type of the format operation. This type is a type abbreviation for [StringFormat<'T,string>](../VS_csharp/printf.stringformat--t--result--type-abbreviation--fsharp-.md).
