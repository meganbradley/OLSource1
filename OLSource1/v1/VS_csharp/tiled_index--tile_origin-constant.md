# tiled_index::tile_origin Constant
Stores an [index](../VS_csharp/index-class.md) object of rank 1, 2, or 3 that represents the global coordinates of the origin of the current tile within a [tiled_extent](../VS_csharp/tiled_extent-class.md) object.  
  
## Syntax  
  
```  
const index<rank> tile_origin  
```  
  
## Requirements  
 **Header:** amp.h  
  
 **Namespace:** Concurrency
