# Set Current Stack Frame Command
Allows you to set a particular stack frame.  
  
## Syntax  
  
```  
Debug.SetCurrentStackFrame index  
```  
  
## Arguments  
 `index`  
 Required. Selects a stack frame by its index.  
  
## Example  
  
```  
>Debug.SetCurrentStackFrame 1  
```
