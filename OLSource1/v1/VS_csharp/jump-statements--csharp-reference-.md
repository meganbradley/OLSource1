# Jump Statements (C# Reference)
Branching is performed using jump statements, which cause an immediate transfer of the program control. The following keywords are used in jump statements:  
  
-   [break](../VS_csharp/break--csharp-reference-.md)  
  
-   [continue](../VS_csharp/continue--csharp-reference-.md)  
  
-   [goto](../VS_csharp/goto--csharp-reference-.md)  
  
-   [return](../VS_csharp/return--csharp-reference-.md)  
  
-   [throw](../VS_csharp/throw--csharp-reference-.md)
