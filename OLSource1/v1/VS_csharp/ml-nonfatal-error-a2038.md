# ML Nonfatal Error A2038
**missing operand for macro operator**  
  
 The assembler found the end of a macro's parameter list immediately after the [!](../VS_csharp/operator-!--masm-run-time-.md) or [%](../VS_csharp/operator--.md) operator.
