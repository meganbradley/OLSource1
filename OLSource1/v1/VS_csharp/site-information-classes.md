# Site Information Classes
The following classes allow an object to communicate with its site:  
  
-   [IObjectWithSiteImpl](../VS_csharp/iobjectwithsiteimpl-class.md) Retrieves and sets a pointer to an object's site. Used for objects that are not controls.  
  
-   [IOleObjectImpl](../VS_csharp/ioleobjectimpl-class.md) Retrieves and sets a pointer to an object's site. Used for controls.
