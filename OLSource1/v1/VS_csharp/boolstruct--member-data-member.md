# BoolStruct::Member Data Member
Supports the WRL infrastructure and is not intended to be used directly from your code.  
  
## Syntax  
  
```  
int Member;  
```  
  
## Remarks  
 Specifies that a [ComPtr](../VS_csharp/comptr-class.md) is, or is not, managing the object lifetime of an interface.  
  
## Requirements  
 **Header:** internal.h  
  
 **Namespace:** Microsoft::WRL::Details
