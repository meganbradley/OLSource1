# Compiler Warning (level 4) C4536
'type name' : type-name exceeds meta-data limit of 'limit' characters  
  
 A type name would be truncated in metadata if it was a managed type. See [C3180](../VS_csharp/compiler-error-c3180.md) for more information.  
  
 This warning is off by default. See [Compiler Warnings That Are Off by Default](../VS_csharp/compiler-warnings-that-are-off-by-default.md) for more information.
