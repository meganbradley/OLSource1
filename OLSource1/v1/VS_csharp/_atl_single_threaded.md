# _ATL_SINGLE_THREADED
Define if all of your objects use the single threading model  
  
## Syntax  
  
```  
  
_ATL_SINGLE_THREADED  
  
```  
  
## Remarks  
 Specifies that the object always runs in the primary COM thread. See [Specifying the Project's Threading Model](../VS_csharp/specifying-the-threading-model-for-a-project--atl-.md) for other threading options, and [Options, ATL Simple Object Wizard](../VS_csharp/options--atl-simple-object-wizard.md) for a description of the threading models available for an ATL object.
