# in (C# Reference)
The `in` contextual keyword is used in three contexts:  
  
-   [foreach](../VS_csharp/foreach--in--csharp-reference-.md) statements  
  
-   [join clauses](../VS_csharp/join-clause--csharp-reference-.md) in query expressions  
  
-   [generic type parameters](../VS_csharp/in--generic-modifier---csharp-reference-.md) in generic interfaces and delegates.
