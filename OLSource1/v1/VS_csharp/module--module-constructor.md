# Module::Module Constructor
Initializes a new instance of the Module class.  
  
## Syntax  
  
```  
Module();  
```  
  
## Remarks  
 This constructor is protected and cannot be called with the `new` keyword. Instead, call either [Module::GetModule](../VS_csharp/module--getmodule-method.md) or [Module::Create](../VS_csharp/module--create-method.md).  
  
## Requirements  
 **Header:** module.h  
  
 **Namespace:** Microsoft::WRL
