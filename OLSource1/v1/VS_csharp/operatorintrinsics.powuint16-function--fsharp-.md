# OperatorIntrinsics.PowUInt16 Function (F#)
This is a library intrinsic. Calls to this function may be generated by uses of the generic [pown](../VS_csharp/operators.pown-^t--function--fsharp-.md) operator on values of type `uint16`.  
  
 **Namespace/Module Path:** Microsoft.FSharp.Core.Operators.OperatorIntrinsics  
  
 **Assembly:** FSharp.Core (in FSharp.Core.dll)  
  
## Syntax  
  
```  
// Signature:  
PowUInt16 : uint16 -> int -> uint16  
  
// Usage:  
PowUInt16 x n  
```  
  
#### Parameters  
 `x`  
 Type: [uint16](../VS_csharp/core.uint16-type-abbreviation--fsharp-.md)  
  
 The value to raise to a power.  
  
 `n`  
 Type: [int](../VS_csharp/core.int-type-abbreviation--fsharp-.md)  
  
 The power to raise the value to.  
  
## Return Value  
 The result of the exponentiation operation.  
  
## Remarks  
 This function is for use by compiled F# code and should not be used directly.
