# acos Function (fast_math)
Calculates the arccosine of the argument  
  
## Syntax  
  
```  
inline float acos(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the arccosine value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
