# SI.UnitNames Namespace (F#)
This namespac contains types for the International System of Units (SI).  
  
 **Namespace/Module Path**: Microsoft.FSharp.Data.UnitSystems.SI  
  
 **Assembly**: FSharp.Core (in FSharp.Core.dll)  
  
## Syntax  
  
```  
module UnitNames  
```  
  
## Remarks  
 This module contains SI units by name. Unit symbols are contained in [SI.UnitSymbols Module (F#)](../VS_csharp/si.unitsymbols-namespace--fsharp-.md).
