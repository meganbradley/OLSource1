# Interface Pointers Classes
The following classes manage a given interface pointer:  
  
-   [CComPtr](../VS_csharp/ccomptr-class.md) Performs automatic reference counting.  
  
-   [CComQIPtr](../VS_csharp/ccomqiptr-class.md) Similar to `CComPtr`, but also performs automatic querying of interfaces.  
  
-   [CInterfaceArray](../VS_csharp/cinterfacearray-class.md) Provides methods useful when constructing an array of COM interface pointers.  
  
-   [CInterfaceList](../VS_csharp/cinterfacelist-class.md) Provides methods useful when constructing a list of COM interface pointers.  
  
-   [CComGITPtr](../VS_csharp/ccomgitptr-class.md) Provides methods for dealing with interface pointers and the global interface table (GIT).
