# Sids::TerminalServer

>  This function cannot be used in applications that execute in the [!INCLUDE[wrt](../VS_csharp/includes/wrt_md.md)].  
  
## Syntax  
  
```  
  
CSid TerminalServer( ) throw(...);  
  
```
