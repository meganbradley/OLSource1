# &lt;para&gt; (Visual C++)
The <para\> tag is for use inside a tag, such as [<summary\>](../VS_csharp/-summary---visual-c---.md), [<remarks\>](../VS_csharp/-remarks---visual-c---.md), or [<returns\>](../VS_csharp/-returns---visual-c---.md), and lets you add structure to the text.  
  
## Syntax  
  
```  
<para>content</para>  
```  
  
#### Parameters  
 `content`  
 The text of the paragraph.  
  
## Remarks  
 Compile with [/doc](../VS_csharp/-doc--process-documentation-comments---c-c---.md) to process documentation comments to a file.  
  
## Example  
 See [<summary\>](../VS_csharp/-summary---visual-c---.md) for an example of using <para\>.
