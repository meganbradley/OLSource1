# .Txt Files as Linker Input
LINK expects various text files as additional input. The [command-file](../VS_csharp/link-command-files.md) specifier (@) and the [Base Address](../VS_csharp/-base--base-address-.md) (/BASE), [/DEF](../VS_csharp/-def--specify-module-definition-file-.md), and [/ORDER](../VS_csharp/-order--put-functions-in-order-.md) options all specify text files. These files can have any extension, not just .txt.
