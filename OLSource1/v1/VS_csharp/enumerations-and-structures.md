# Enumerations and Structures
These are the enumerations and structures used by the DIA SDK.  
  
## In This Section  
  
## Enumerations  
 [BasicType Enumeration](../VS_csharp/basictype.md)  
 Specifies the symbol's basic type.  
  
 [CV_access_e Enumeration](../VS_csharp/cv_access_e.md)  
 Specifies the scope of visibility (access level) of member functions and variables.  
  
 [CV_call_e Enumeration](../VS_csharp/cv_call_e.md)  
 Specifies the calling convention for a function.  
  
 [CV_CFL_LANG Enumeration](../VS_csharp/cv_cfl_lang.md)  
 Specifies the source code language of the application or linked module.  
  
 [CV_CPU_TYPE_e Enumeration](../VS_csharp/cv_cpu_type_e.md)  
 Specifies the target processor.  
  
 [CV_HREG_e Enumeration](../VS_csharp/cv_hreg_e.md)  
 Specifies a target register.  
  
 [DataKind Enumeration](../VS_csharp/datakind.md)  
 Indicates the particular scope of a data value.  
  
 [LocationType Enumeration](../VS_csharp/locationtype.md)  
 Indicates the kind of location information contained in a symbol.  
  
 [MemoryTypeEnum](../VS_csharp/memorytypeenum.md)  
 Specifies the type of memory to access.  
  
 [NameSearchOptions Enumeration](../VS_csharp/namesearchoptions.md)  
 Specifies the search options for symbol and file names.  
  
 [StackFrameTypeEnum](../VS_csharp/stackframetypeenum.md)  
 Specifies the stack frame type.  
  
 [SymTagEnum Enumeration](../VS_csharp/symtagenum.md)  
 Specifies the type of symbol.  
  
 [THUNK_ORDINAL Enumeration](../VS_csharp/thunk_ordinal.md)  
 Designates thunk types.  
  
 [UdtKind](../VS_csharp/udtkind.md)  
 Describes the variety of user-defined type (UDT).  
  
## Structures  
 [DiaAddressMapEntry Structure](../VS_csharp/diaaddressmapentry.md)  
 Describes an entry in an address map.
