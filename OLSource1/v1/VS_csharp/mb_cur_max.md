# MB_CUR_MAX
## Syntax  
  
```  
  
#include <stdlib.h>  
  
```  
  
## Remarks  
 Context: ANSI multibyte- and wide-character conversion functions  
  
 The value of `MB_CUR_MAX` is the maximum number of bytes in a multibyte character for the current locale.
