# Floating-Point Types Table (C# Reference)
The following table shows the precision and approximate ranges for the floating-point types.  
  
|Type|Approximate range|Precision|  
|----------|-----------------------|---------------|  
|[float](../VS_csharp/float--csharp-reference-.md)|±1.5e−45 to ±3.4e38|7 digits|  
|[double](../VS_csharp/double--csharp-reference-.md)|±5.0e−324 to ±1.7e308|15-16 digits|
