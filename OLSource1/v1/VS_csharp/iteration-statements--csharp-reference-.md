# Iteration Statements (C# Reference)
You can create loops by using the iteration statements. Iteration statements cause embedded statements to be executed a number of times, subject to the loop-termination criteria. These statements are executed in order, except when a [jump statement](../VS_csharp/jump-statements--csharp-reference-.md) is encountered.  
  
 The following keywords are used in iteration statements:  
  
-   [do](../VS_csharp/do--csharp-reference-.md)  
  
-   [for](../VS_csharp/for--csharp-reference-.md)  
  
-   [foreach](../VS_csharp/foreach--in--csharp-reference-.md)  
  
-   [in](../VS_csharp/foreach--in--csharp-reference-.md)  
  
-   [while](../VS_csharp/while--csharp-reference-.md)
