# IDiaLineNumber::get_length
Retrieves the number of bytes in a block.  
  
## Syntax  
  
```cpp#  
HRESULT get_length (   
   DWORD* pRetVal  
);  
```  
  
#### Parameters  
 `pRetVal`  
 [out] Returns the number of bytes in a block.  
  
## Return Value  
 If successful, returns `S_OK`. Returns `S_FALSE` if this property is not supported. Otherwise, returns an error code.  
  
## Remarks  
 The block is the length of source code on the line as represented by the [IDiaLineNumber](../VS_csharp/idialinenumber.md) object.
