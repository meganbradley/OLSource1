# END_CATCH
Marks the end of the last **CATCH** or `AND_CATCH` block.  
  
## Syntax  
  
```  
  
END_CATCH  
  
```  
  
## Remarks  
 For more information on the `END_CATCH` macro, see the article [Exceptions](../VS_csharp/exception-handling-in-mfc.md).  
  
## Requirements  
 **Header:** afx.h
