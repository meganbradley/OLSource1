# MoveSettings::IsVertical
Check if the move data specifies a nonzero vertical movement.  
  
## Syntax  
  
```vb  
BOOL IsVertical() const  
```  
  
## Return Value  
 TRUE if the `MoveSettings` object specifies a nonzero vertical movement.
