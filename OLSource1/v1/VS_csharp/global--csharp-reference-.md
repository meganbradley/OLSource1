# global (C# Reference)
The `global` contextual keyword, when it comes before the [:: operator](../VS_csharp/---operator--csharp-reference-.md), refers to the global namespace, which is the default namespace for any C# program and is otherwise unnamed. For more information, see [How to: Use the Global Namespace Alias](../VS_csharp/how-to--use-the-global-namespace-alias--csharp-programming-guide-.md).
