# .CONTINUE
Generates code to jump to the top of a [.WHILE](../VS_csharp/.while.md) or [.REPEAT](../VS_csharp/.repeat.md) block if `condition` is true.  
  
## Syntax  
  
```  
  
.CONTINUE [[.IF condition]]  
```
