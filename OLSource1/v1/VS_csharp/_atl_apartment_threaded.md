# _ATL_APARTMENT_THREADED
Define if one or more of your objects use apartment threading.  
  
## Syntax  
  
```  
  
_ATL_APARTMENT_THREADED  
  
```  
  
## Remarks  
 Specifies apartment threading. See [Specifying the Project's Threading Model](../VS_csharp/specifying-the-threading-model-for-a-project--atl-.md) for other threading options, and [Options, ATL Simple Object Wizard](../VS_csharp/options--atl-simple-object-wizard.md) for a description of the threading models available for an ATL object.
