# -Y (Precompiled Headers)
The following compiler options affect the generation and use of precompiled headers:  
  
-   [/Y- (Ignore Precompiled Header Options)](../VS_csharp/-y---ignore-precompiled-header-options-.md)  
  
-   [/Yc (Create Precompiled Header File)](../VS_csharp/-yc--create-precompiled-header-file-.md)  
  
-   [/Yd (Place Debug Information in Object File)](../VS_csharp/-yd--place-debug-information-in-object-file-.md)  
  
-   [/Yl (Inject PCH Reference for Debug Library)](../VS_csharp/-yl--inject-pch-reference-for-debug-library-.md)  
  
-   [/Yu (Use Precompiled Header File)](../VS_csharp/-yu--use-precompiled-header-file-.md)  
  
 For details on working with precompiled headers, see [Creating Precompiled Header Files](../VS_csharp/creating-precompiled-header-files.md).
