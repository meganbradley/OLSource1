# ML Nonfatal Error A2034
**must be in segment block**  
  
 One of the following was found outside of a segment block:  
  
-   An instruction  
  
-   A label definition  
  
-   A [THIS](../VS_csharp/operator-this.md) operator  
  
-   A [$](../VS_csharp/$.md) operator  
  
-   A procedure definition  
  
-   An [ALIGN](../VS_csharp/align--masm-.md) directive  
  
-   An [ORG](../VS_csharp/org.md) directive
