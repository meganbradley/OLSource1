# _ATL_FREE_THREADED
Define if one or more of your objects use free or neutral threading.  
  
## Syntax  
  
```  
  
_ATL_FREE_THREADED  
  
```  
  
## Remarks  
 Specifies free threading. Free threading is equivalent to a multithread apartment model. See [Specifying the Project's Threading Model](../VS_csharp/specifying-the-threading-model-for-a-project--atl-.md) for other threading options, and [Options, ATL Simple Object Wizard](../VS_csharp/options--atl-simple-object-wizard.md) for a description of the threading models available for an ATL object.
