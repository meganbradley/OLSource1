# C# Preprocessor Directives
This section contains information about the following C# preprocessor directives.  
  
 [#if](../VS_csharp/sharpif--csharp-reference-.md)  
  
 [#else](../VS_csharp/sharpelse--csharp-reference-.md)  
  
 [#elif](../VS_csharp/sharpelif--csharp-reference-.md)  
  
 [#endif](../VS_csharp/sharpendif--csharp-reference-.md)  
  
 [#define](../VS_csharp/sharpdefine--csharp-reference-.md)  
  
 [#undef](../VS_csharp/sharpundef--csharp-reference-.md)  
  
 [#warning](../VS_csharp/sharpwarning--csharp-reference-.md)  
  
 [#error](../VS_csharp/sharperror--csharp-reference-.md)  
  
 [#line](../VS_csharp/sharpline--csharp-reference-.md)  
  
 [#region](../VS_csharp/sharpregion--csharp-reference-.md)  
  
 [#endregion](../VS_csharp/sharpendregion--csharp-reference-.md)  
  
 [#pragma](../VS_csharp/sharppragma--csharp-reference-.md)  
  
 [#pragma warning](../VS_csharp/sharppragma-warning--csharp-reference-.md)  
  
 [#pragma checksum](../VS_csharp/sharppragma-checksum--csharp-reference-.md)  
  
 See the individual topics for more information and examples.  
  
 Although the compiler does not have a separate preprocessor, the directives described in this section are processed as if there were one. They are used to help in conditional compilation. Unlike C and C++ directives, you cannot use these directives to create macros.  
  
 A preprocessor directive must be the only instruction on a line.
