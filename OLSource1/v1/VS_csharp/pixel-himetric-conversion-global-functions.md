# Pixel-HIMETRIC Conversion Global Functions
These functions provide support for converting to and from pixel and HIMETRIC units.  
  
> [!IMPORTANT]
>  The functions listed in the following table cannot be used in applications that execute in the [!INCLUDE[wrt](../VS_csharp/includes/wrt_md.md)].  
  
|||  
|-|-|  
|[AtlHiMetricToPixel](../VS_csharp/atlhimetrictopixel.md)|Converts HIMETRIC units (each unit is 0.01 millimeter) to pixels.|  
|[AtlPixelToHiMetric](../VS_csharp/atlpixeltohimetric.md)|Converts pixels to HIMETRIC units (each unit is 0.01 millimeter).|
