# Linker Tools Warning LNK4220
invalid 'linker option' value 'value'; assumed default  
  
 An out-of-range value was specified with the [/TLBID](../VS_csharp/-tlbid--specify-resource-id-for-typelib-.md) option. The default value for **/TLBID** is 1.
