# Contents of a Makefile
A makefile contains:  
  
-   [Description blocks](../VS_csharp/description-blocks.md)  
  
-   [Commands](../VS_csharp/commands-in-a-makefile.md)  
  
-   [Macros](../VS_csharp/macros-and-nmake.md)  
  
-   [Inference rules](../VS_csharp/inference-rules.md)  
  
-   [Dot directives](../VS_csharp/dot-directives.md)  
  
-   [Preprocessing directives](../VS_csharp/makefile-preprocessing.md)  
  
## Remarks  
 Other features you can use in a makefile are [wildcards](../VS_csharp/wildcards-and-nmake.md), [long filenames](../VS_csharp/long-filenames-in-a-makefile.md), [comments](../VS_csharp/comments-in-a-makefile.md), and [special characters](../VS_csharp/special-characters-in-a-makefile.md).  
  
 For a sample, see [Sample Makefile](../VS_csharp/sample-makefile.md).
