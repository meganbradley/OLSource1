# using (C# Reference)
The `using` keyword has two major uses:  
  
-   As a directive, when it is used to create an alias for a namespace or to import types defined in other namespaces. See [using Directive](../VS_csharp/using-directive--csharp-reference-.md).  
  
-   As a statement, when it defines a scope at the end of which an object will be disposed. See [using Statement](../VS_csharp/using-statement--csharp-reference-.md).
