# BEGIN_PROPERTY_SET
Marks the beginning of a property set in a property set map.  
  
## Syntax  
  
```  
  
BEGIN_PROPERTY_SET(  
guid   
)  
  
```  
  
#### Parameters  
 `guid`  
 [in] The property GUID.  
  
## Example  
 See [BEGIN_PROPSET_MAP](../VS_csharp/begin_propset_map.md).  
  
## Requirements  
 **Header:** atldb.h
