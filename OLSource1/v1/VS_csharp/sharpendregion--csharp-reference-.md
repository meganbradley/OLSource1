# #endregion (C# Reference)
`#endregion` marks the end of a [#region](../VS_csharp/sharpregion--csharp-reference-.md) block. For example:  
  
```  
  
      #region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
