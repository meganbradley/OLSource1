# Debugger Settings and Preparation
This section describes the debugger-related property settings, and the preparation needed in order to debug your program with the [!INCLUDE[vsprvs](../VS_csharp/includes/vsprvs_md.md)] debugger. If you create your program in [!INCLUDE[vsprvs](../VS_csharp/includes/vsprvs_md.md)] using the project templates, these settings are set correctly for you in the Debug configuration.  
  
 For more information, see [How to: Set Debug and Release Configurations](../VS_csharp/how-to--set-debug-and-release-configurations.md).
