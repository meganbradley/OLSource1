# Attributes by Usage
This topic lists attributes according to the C++ language elements to which they apply.  
  
 If an attribute precedes a Visual C++ element that is not in the attribute's scope, the attribute block is treated as a comment.  
  
|Attribute|Description|  
|---------------|-----------------|  
|[Module Attributes](../VS_csharp/module-attributes.md)|Applies to the [module](../VS_csharp/module--c---.md) attribute.|  
|[Interface Attributes](../VS_csharp/interface-attributes.md)|Applies to the [__interface](../VS_csharp/__interface.md) C++ keyword.|  
|[Class Attributes](../VS_csharp/class-attributes.md)|Applies to the C++ keyword.|  
|[Method Attributes](../VS_csharp/method-attributes.md)|Applies to the methods in a class, coclass, or interface.|  
|[Parameter Attributes](../VS_csharp/parameter-attributes.md)|Applies to parameters of a method in a class or interface.|  
|[Data Member Attributes](../VS_csharp/data-member-attributes.md)|Applies to the data members in a class, coclass, or interface.|  
|[Typedef, Enum, Union, and Struct Attributes](../VS_csharp/typedef--enum--union--and-struct-attributes.md)|Applies to the C++ keywords.|  
|[Array Attributes](../VS_csharp/array-attributes.md)|Applies to arrays or **SAFEARRAY**s.|  
|[Stand-Alone Attributes](../VS_csharp/stand-alone-attributes.md)|Operates more like a line of code but does not operate on a C++ keyword. Stand-alone attribute statements require a semicolon at the end of the line.|  
|[Custom Attributes](../VS_csharp/custom-attributes--c---.md)|Allows the user to extend metadata.|
