# Data (Debug Interface Access SDK)
All variables, such as parameters, local variables, global variables, and class members, are identified by `SymTagData` symbols. Constant values (`LocIsConstant`) are also identified with this type.  
  
## Properties  
 The following table shows the properties that are valid for this symbol type.  
  
|Property|Data type|Description|  
|--------------|---------------|-----------------|  
|[IDiaSymbol::get_access](../VS_csharp/idiasymbol--get_access.md)|`DWORD`|If a field, then one of the values of the [CV_access_e Enumeration](../VS_csharp/cv_access_e.md).|  
|[IDiaSymbol::get_addressOffset](../VS_csharp/idiasymbol--get_addressoffset.md)|`DWORD`|Offset part of location; for details, see the [LocationType Enumeration](../VS_csharp/locationtype.md).|  
|[IDiaSymbol::get_addressSection](../VS_csharp/idiasymbol--get_addresssection.md)|`DWORD`|Section part of location; for details, see the [LocationType Enumeration](../VS_csharp/locationtype.md).|  
|[addressTaken](../VS_csharp/idiasymbol--get_addresstaken.md)|`BOOL`|`TRUE` if this data's address is referenced by another symbol.|  
|[IDiaSymbol::get_bitPosition](../VS_csharp/idiasymbol--get_bitposition.md)|`DWORD`|Bit position of location; for details, see the [LocationType Enumeration](../VS_csharp/locationtype.md) (not supported in DIA SDK v8.0).|  
|[IDiaSymbol::get_classParent](../VS_csharp/idiasymbol--get_classparent.md)|`IDiaSymbol*`|Symbol for the class, if this is a structure, union, or class field.|  
|[classParentId](../VS_csharp/idiasymbol--get_classparentid.md)|`DWORD`|ID of the class parent symbol.|  
|[compilerGenerated](../VS_csharp/idiasymbol--get_compilergenerated.md)|`BOOL`|`TRUE` if the data was generated by the compiler.|  
|[constType](../VS_csharp/idiasymbol--get_consttype.md)|`BOOL`|`TRUE` if the data is marked as being constant.|  
|[IDiaSymbol::get_dataKind](../VS_csharp/idiasymbol--get_datakind.md)|`DWORD`|One of the [DataKind Enumeration](../VS_csharp/datakind.md) values.|  
|[IDiaSymbol::get_isAggregated Method](../VS_csharp/idiasymbol--get_isaggregated.md)|`BOOL`|`TRUE` if the data is part of an aggregated data type (only in DIA SDK v8.0 and later).|  
|[IDiaSymbol::get_isSplitted Method](../VS_csharp/idiasymbol--get_issplitted.md)|`BOOL`|`TRUE` if data is has been split into an aggregate of multiple symbols (only in DIA SDK v8.0 and later).|  
|[IDiaSymbol::get_length](../VS_csharp/idiasymbol--get_length.md)|`ULONGLONG`|Length of bitfield; for details, see the [LocationType Enumeration](../VS_csharp/locationtype.md).|  
|[IDiaSymbol::get_lexicalParent](../VS_csharp/idiasymbol--get_lexicalparent.md)|`IDiaSymbol*`|Symbol for the enclosing compiland, function, or block.|  
|[lexicalParentId](../VS_csharp/idiasymbol--get_lexicalparentid.md)|`DWORD`|ID of the lexical parent symbol.|  
|[IDiaSymbol::get_locationType](../VS_csharp/idiasymbol--get_locationtype.md)|`DWORD`|Any of the allowable location types; for details, see [Symbol Locations](../VS_csharp/symbol-locations.md)|  
|[IDiaSymbol::get_name](../VS_csharp/idiasymbol--get_name.md)|`BSTR`|Name of the variable.|  
|[IDiaSymbol::get_offset](../VS_csharp/idiasymbol--get_offset.md)|`LONG`|Offset from register contents; for details, see the [LocationType Enumeration](../VS_csharp/locationtype.md).|  
|[IDiaSymbol::get_registerId](../VS_csharp/idiasymbol--get_registerid.md)|`DWORD`|Register designator of location; for details, see the [LocationType Enumeration](../VS_csharp/locationtype.md).|  
|[relativeVirtualAddress](../VS_csharp/idiasymbol--get_relativevirtualaddress.md)|`DWORD`|Relative position of the data within its block.|  
|[IDiaSymbol::get_slot Method](../VS_csharp/idiasymbol--get_slot.md)|`DWORD`|Gets slot number of the data.|  
|[IDiaSymbol::get_symIndexId Method](../VS_csharp/idiasymbol--get_symindexid.md)|`DWORD`|Index ID of symbol.|  
|[symTag](../VS_csharp/idiasymbol--get_symtag.md)|`DWORD`|Returns `SymTagData` (one of the [SymTagEnum Enumeration](../VS_csharp/symtagenum.md) values).|  
|[IDiaSymbol::get_token Method](../VS_csharp/idiasymbol--get_token.md)|`DWORD`|The metadata token representing the data.|  
|[IDiaSymbol::get_type](../VS_csharp/idiasymbol--get_type.md)|`IDiaSymbol*`|Symbol for the variable type.|  
|[typeId](../VS_csharp/idiasymbol--get_typeid.md)|`DWORD`|ID of the variable type symbol.|  
|[unalignedType](../VS_csharp/idiasymbol--get_unalignedtype.md)|`BOOL`|`TRUE` if the data is unaligned.|  
|[IDiaSymbol::get_value](../VS_csharp/idiasymbol--get_value.md)|`VARIANT`|The value of constant data.|  
|[virtualAddress](../VS_csharp/idiasymbol--get_virtualaddress.md)|`ULONGLONG`|Position of the data within the executable.|  
|[volatileType](../VS_csharp/idiasymbol--get_volatiletype.md)|`BOOL`|`TRUE` if the data is marked as volatile.|
