# Internet URL Parsing Globals
When a client sends a query to the Internet server, you can use one of the URL parsing globals to extract information about the client.  
  
### Internet URL Parsing Globals  
  
|||  
|-|-|  
|[AfxParseURL](../VS_csharp/afxparseurl.md)|Parses a URL string and returns the type of service and its components.|  
|[AfxParseURLEx](../VS_csharp/afxparseurlex.md)|Parses a URL string and returns the type of service and its components, as well as providing the user name and password.|
