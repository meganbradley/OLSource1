# ENDS (MASM)
Marks the end of segment, structure, or union *name* previously begun with [SEGMENT](../VS_csharp/segment.md), [STRUCT](../VS_csharp/struct--masm-.md), [UNION](../VS_csharp/union.md), or a simplified segment directive.  
  
## Syntax  
  
```  
  
name ENDS  
```
