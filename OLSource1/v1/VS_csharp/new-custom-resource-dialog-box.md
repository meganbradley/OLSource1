# New Custom Resource Dialog Box
Enables you to create a new custom resource.  
  
 **Resource Type**  
 Provides a text box for you to enter the name of a custom resource type. Visual C++ automatically capitalizes the name when you exit the **New Custom Resource** dialog box.  
  
## Remarks  
 Do not specify a resource name or [!INCLUDE[TLA2#tla_id](../VS_csharp/includes/tla2sharptla_id_md.md)] that is reserved by [!INCLUDE[vsprvs](../VS_csharp/includes/vsprvs_md.md)]. The reserved names are DESIGNINFO, HWB, and TEXTINCLUDE, and the reserved [!INCLUDE[TLA2#tla_id](../VS_csharp/includes/tla2sharptla_id_md.md)] is 255.
