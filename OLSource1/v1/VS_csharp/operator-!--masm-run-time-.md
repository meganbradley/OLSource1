# operator ! (MASM Run Time)
Logical negation. Used only within [.IF](../VS_csharp/.if.md), [.WHILE](../VS_csharp/.while.md), or [.REPEAT](../VS_csharp/.repeat.md) blocks and evaluated at run time, not at assembly time.  
  
## Syntax  
  
```  
  
!  
expression  
  
```
