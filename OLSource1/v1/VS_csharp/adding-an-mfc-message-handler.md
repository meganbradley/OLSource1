# Adding an MFC Message Handler
You can [use the Properties window to add a message handler](../VS_csharp/mapping-messages-to-functions.md) (a member function that handles Windows messages) to a class and map Windows messages to the message handler. You can also add [an event handler for any dialog box control](../VS_csharp/adding-event-handlers-for-dialog-box-controls.md).  
  
 By using the Properties window to define message- and event-handling functions, you can automatically update the message-dispatch table (or message map) and your class header file.  
  
> [!NOTE]
>  You can add a message handler to an ATL class using the Properties window; however, some results may vary. For more information, see the ATL topic [Adding an ATL Message Handler](../VS_csharp/adding-an-atl-message-handler.md).
