# END_PROP_MAP
Marks the end of the object's property map.  
  
## Syntax  
  
```  
  
END_PROP_MAP( )  
  
```  
  
## Remarks  
 When you create an object with the ATL Project Wizard, the wizard will create an empty property map by specifying [BEGIN_PROP_MAP](../VS_csharp/begin_prop_map.md) followed by `END_PROP_MAP`.  
  
## Example  
 See the example for [BEGIN_PROP_MAP](../VS_csharp/begin_prop_map.md).  
  
## Requirements  
 **Header:** atlcom.h
