# Making Owner-Drawn Tabs
You can define individual items of a tab control ([CTabCtrl](../VS_csharp/ctabctrl-class.md)) to be owner-drawn items. For more information, see [Owner-Drawn Tabs](http://msdn.microsoft.com/library/windows/desktop/bb760550) in the [!INCLUDE[winSDK](../VS_csharp/includes/winsdk_md.md)].
