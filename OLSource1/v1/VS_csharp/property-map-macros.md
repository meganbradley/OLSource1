# Property Map Macros
These macros define property maps and entries.  
  
|||  
|-|-|  
|[BEGIN_PROP_MAP](../VS_csharp/begin_prop_map.md)|Marks the beginning of the ATL property map.|  
|[PROP_DATA_ENTRY](../VS_csharp/prop_data_entry.md)|Indicates the extent, or dimensions, of an ActiveX control.|  
|[PROP_ENTRY_TYPE](../VS_csharp/prop_entry_type.md)|Enters a property description, property DISPID, and property page CLSID into the property map.|  
|[PROP_ENTRY_TYPE_EX](../VS_csharp/prop_entry_type_ex.md)|Enters a property description, property DISPID, property page CLSID, and `IDispatch` IID into the property map.|  
|[PROP_PAGE](../VS_csharp/prop_page.md)|Enters a property page CLSID into the property map.|  
|[END_PROP_MAP](../VS_csharp/end_prop_map.md)|Marks the end of the ATL property map.|
