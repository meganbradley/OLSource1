# &lt;code&gt; (C# Programming Guide)
## Syntax  
  
```  
<code>content</code>  
```  
  
#### Parameters  
 `content`  
 The text you want marked as code.  
  
## Remarks  
 The <code\> tag gives you a way to indicate multiple lines as code. Use [<c\>](../VS_csharp/-c---csharp-programming-guide-.md) to indicate that text within a description should be marked as code.  
  
 Compile with [/doc](../VS_csharp/-doc--csharp-compiler-options-.md) to process documentation comments to a file.
