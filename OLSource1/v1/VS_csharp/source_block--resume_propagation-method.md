# source_block::resume_propagation Method
When overridden in a derived class, resumes propagation after a reservation has been released.  
  
## Syntax  
  
```  
virtual void resume_propagation() = 0;  
```  
  
## Requirements  
 **Header:** agents.h  
  
 **Namespace:** concurrency
