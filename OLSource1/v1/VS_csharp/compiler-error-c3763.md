# Compiler Error C3763
'type': 'retval' and 'out' can only appear on a data-pointer type  
  
 The [out](../VS_csharp/out--c---.md) or [retval](../VS_csharp/retval.md) attributes can only appear on parameters of type pointer. Either remove the attribute or make the parameter of type pointer.  
  
 The following sample generates C3763:  
  
```  
// C3763.cpp  
#define _ATL_ATTRIBUTES 1  
#include <atlbase.h>  
#include <atlplus.h>  
#include <atlcom.h>  
  
[ module(name=test) ];  
  
[ dispinterface, uuid("00000000-0000-0000-0000-000000000001") ]  
__interface IF84 : IDispatch  
{  
   [id(0x00000002)]HRESULT m2([out]unsigned char);  
};  
  
[ coclass, uuid("00000000-0000-0000-0000-000000000002") ]  
class CF84 : public IF84  
{   // C3763  
public:  
   HRESULT __stdcall m2(unsigned char i)  
   {  
      return S_OK;  
   }  
};  
  
int main()  
{  
}  
```
