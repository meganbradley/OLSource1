# Enumerators and Collections Classes
The following classes provide support for COM collections and enumerations:  
  
-   [CComEnum](../VS_csharp/ccomenum-class.md) Defines a COM enumerator object based on an array.  
  
-   [CComEnumImpl](../VS_csharp/ccomenumimpl-class.md) Provides the implementation for a COM enumerator interface where the items being enumerated are stored in an array.  
  
-   [CComEnumOnSTL](../VS_csharp/ccomenumonstl-class.md) Defines a COM enumerator object based on an STL collection.  
  
-   [IEnumOnSTLImpl](../VS_csharp/ienumonstlimpl-class.md) Provides the implementation for a COM enumerator interface where the items being enumerated are stored in an STL-compatible container.  
  
-   [ICollectionOnSTLImpl](../VS_csharp/icollectiononstlimpl-class.md) Provides the implementation for the **Count**, **Item**, and `_NewEnum` properties of a collection interface.
