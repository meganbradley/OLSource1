# Operator Keywords (C# Reference)
Used to perform miscellaneous actions such as creating objects, checking the run-time type of an object, obtaining the size of a type, and other actions. This section introduces the following keywords:  
  
-   [as](../VS_csharp/as--csharp-reference-.md) Converts an object to a compatible type.  
  
-   [await](../VS_csharp/await--csharp-reference-.md) Suspends an async method until an awaited task is completed.  
  
-   [is](../VS_csharp/is--csharp-reference-.md) Checks the run-time type of an object.  
  
-   [new](../VS_csharp/new--csharp-reference-.md)  
  
    -   [new Operator](../VS_csharp/new-operator--csharp-reference-.md) Creates objects.  
  
    -   [new Modifier](../VS_csharp/new-modifier--csharp-reference-.md) Hides an inherited member.  
  
    -   [new Constraint](../VS_csharp/new-constraint--csharp-reference-.md) Qualifies a type parameter.  
  
-   [sizeof](../VS_csharp/sizeof--csharp-reference-.md) Obtains the size of a type.  
  
-   [typeof](../VS_csharp/typeof--csharp-reference-.md) Obtains the **System.Type** object for a type.  
  
-   [true](../VS_csharp/true--csharp-reference-.md)  
  
    -   [true Operator](../VS_csharp/true-operator--csharp-reference-.md) Returns the boolean value true to indicate true and returns false otherwise.  
  
    -   [true Literal](../VS_csharp/true-literal--csharp-reference-.md) Represents the boolean value true.  
  
-   [false](../VS_csharp/false--csharp-reference-.md)  
  
    -   [false Operator](../VS_csharp/false-operator--csharp-reference-.md) Returns the Boolean value true to indicate false and returns false otherwise.  
  
    -   [false Literal](../VS_csharp/false-literal--csharp-reference-.md) Represents the boolean value false.  
  
-   [stackalloc](../VS_csharp/stackalloc--csharp-reference-.md) Allocates a block of memory on the stack.  
  
 The following keywords, which can be used as operators and as statements, are covered in the [Statements](../VS_csharp/statement-keywords--csharp-reference-.md) section:  
  
-   [checked](../VS_csharp/checked--csharp-reference-.md) Specifies checked context.  
  
-   [unchecked](../VS_csharp/unchecked--csharp-reference-.md) Specifies unchecked context.
