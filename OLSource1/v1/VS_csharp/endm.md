# ENDM
Terminates a macro or repeat block.  
  
## Syntax  
  
```  
  
ENDM  
  
```  
  
## Remarks  
 See [MACRO](../VS_csharp/macro.md), [FOR](../VS_csharp/for--masm-.md), [FORC](../VS_csharp/forc.md), [REPEAT](../VS_csharp/repeat.md), or **WHILE**.
