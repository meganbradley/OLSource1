# Generic-Text Mappings
To simplify writing code for international markets, generic-text mappings are defined in TCHAR.H for:  
  
-   [Data types](../VS_csharp/data-type-mappings.md)  
  
-   [Constants and global variables](../VS_csharp/constant-and-global-variable-mappings.md)  
  
-   [Routine mappings](../VS_csharp/routine-mappings.md)  
  
 For more information, see [Using Generic-Text Mappings](../VS_csharp/using-generic-text-mappings.md). Generic-text mappings are Microsoft extensions that are not ANSI compatible.
