# Create a Transact-SQL Script for Running a Trace (SQL Server Profiler)
  This topic describes how to create a Transact\-SQL script from an existing trace file or table by using [!INCLUDE[ssSqlProfiler](../../Token\Other/ssSqlProfiler_md.md)].  
  
### To create a Transact\-SQL script to run a trace  
  
1.  Open a trace file or table. For more information, see [Open a Trace File &#40;SQL Server Profiler&#41;](../Topic/Open%20a%20Trace%20File%20\(SQL%20Server%20Profiler\).md) or [Open a Trace Table &#40;SQL Server Profiler&#41;](../Topic/Open%20a%20Trace%20Table%20\(SQL%20Server%20Profiler\).md).  
  
2.  On the**File**menu, point to **Export**, point to **Script Trace Definition**, and then click the version that corresponds to the server you want to trace.  
  
3.  In the **Save As** dialog box, enter a name for the script file, and then click **Save**.
