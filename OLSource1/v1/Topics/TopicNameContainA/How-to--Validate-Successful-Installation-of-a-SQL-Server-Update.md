# How to: Validate Successful Installation of a SQL Server Update
  To validate successful installation of a [!INCLUDE[ssNoVersion](../../Token\Other/ssNoVersion_md.md)] update:  
  
-   Verify that [!INCLUDE[ssNoVersion](../../Token\Other/ssNoVersion_md.md)] services are online.  
  
-   Verify version information returned by the @@version query.
