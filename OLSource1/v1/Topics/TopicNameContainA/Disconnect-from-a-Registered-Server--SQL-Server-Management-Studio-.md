# Disconnect from a Registered Server (SQL Server Management Studio)
  This topic describes how to  disconnect Object Explorer from a server at any time in [!INCLUDE[ssCurrent](../../Token\Other/ssCurrent_md.md)]. Disconnecting Object Explorer does not disconnect other [!INCLUDE[ssManStudioFull](../../Token\Other/ssManStudioFull_md.md)] components such as SQL Editor.  
  
##  <a name="SSMSProcedure"></a>  
  
#### To disconnect from a registered server  
  
-   In Object Explorer, right\-click a server and click **Disconnect** or, on the Object Explorer toolbar, click the **Disconnect** icon.
