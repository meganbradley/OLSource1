# View Registered Servers in SQL Server Management Studio
This topic describes how to view registered servers in [!INCLUDE[ssCurrent](../../Topics/TopicNameContainA/includes/ssCurrent_md.md)].  
  
##  <a name="SSMSProcedure"></a>  
  
#### To view Registered Servers in SQL Server Management Studio  
  
1.  In [!INCLUDE[ssManStudioFull](../../Topics/TopicNameContainA/includes/ssManStudioFull_md.md)], on the **View** menu, click **Registered Servers**.  
  
     By default, Registered Servers appears above Object Explorer in the main window.
