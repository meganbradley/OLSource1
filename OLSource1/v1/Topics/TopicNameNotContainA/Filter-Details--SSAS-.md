# Filter Details (SSAS)
  This page of the **Table Import Wizard** enables you to view the filters that are specified for the data import. To access the wizard from the [!INCLUDE[ssBIDevStudio](../../Token\Other/ssBIDevStudio_md.md)], on the **Model** menu, click **Import from Data Source**.  
  
 The filter is based on the options that you selected in the **Preview Selected Table** dialog box.
