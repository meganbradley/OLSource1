# Run SQL Server Profiler
To run [!INCLUDE[ssSqlProfiler](../../Topics/TopicNameContainA/includes/ssSqlProfiler_md.md)], on the **Start** menu, point to **All Programs**, **Microsoft SQL Server 2014**, **Performance Tools**, and then click **SQL Server Profiler**.
