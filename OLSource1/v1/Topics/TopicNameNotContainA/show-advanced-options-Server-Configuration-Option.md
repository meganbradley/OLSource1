# show advanced options Server Configuration Option
  Use the **show advanced options** option to display the **sp\_configure** system stored procedure advanced options. When you set **show advanced options** to 1, you can list the advanced options by using **sp\_configure**. The default is 0.  
  
 The setting takes effect immediately without a server restart.
