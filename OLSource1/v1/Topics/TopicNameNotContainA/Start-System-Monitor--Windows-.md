# Start System Monitor (Windows)
  Use System Monitor to monitor the utilization of system resources. Collect and view real\-time performance data in the form of counters, for server resources such as processor and memory use, and for many Microsoft [!INCLUDE[ssNoVersion](../../Token\Other/ssNoVersion_md.md)] resources such as locks and transactions.  
  
### To start System Monitor in Windows  
  
1.  On the **Start** menu, point to **Run**, type **perfmon** in the Run dialog box, and then click **OK**.
