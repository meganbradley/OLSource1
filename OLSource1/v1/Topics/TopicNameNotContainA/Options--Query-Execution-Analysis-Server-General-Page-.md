# Options (Query Execution/Analysis Server/General Page)
  Use this page to specify the options for running [!INCLUDE[ssASnoversion](../../Topics/TopicNameContainA/includes/ssASnoversion_md.md)] queries. To display these settings, on the **Tools** menu, click **Options**. Under **Query Execution,** expand the **Analysis Server** folder, and click **General**.  
  
## UIElement List  
 **Execution time-out**  
 Sets the default value in the **New Connection** dialog box. This box indicates the number of seconds to wait before canceling the query. A value of 0 indicates an infinite wait, or no time-out. This value is 0 on a new installation.  
  
 **Reset to Default**  
 Resets all values on this page to the original default values.
