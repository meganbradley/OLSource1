# Analysis Services Query Results (General Page)
Use this page to specify the options for running [!INCLUDE[ssASnoversion](../../Topics/TopicNameContainA/includes/ssASnoversion_md.md)] queries.  
  
 **Execution time-out**  
 Indicates the number of seconds to wait before canceling the query.  
  
 A value of 0 indicates an infinite wait, or no time-out.  
  
 **Reset to Default**  
 Resets all values on this page to the original default values.
