# Client Protocols - Shared Memory Properties (Protocol Tab)
In [!INCLUDE[msCoName](../../Topics/TopicNameContainA/includes/msCoName_md.md)] [!INCLUDE[ssNoVersion](../../Topics/TopicNameContainA/includes/ssNoVersion_md.md)] Configuration Manager use the **Protocol** tab on the **Shared Memory Properties** dialog box to view or modify shared memory. For more information about connection strings, see [Creating a Valid Connection String Using Shared Memory Protocol](../../Topics/TopicNameContainA/Creating-a-Valid-Connection-String-Using-Shared-Memory-Protocol.md).  
  
## Options  
 **Enabled**  
 Possible values are **Yes** and **No**.
