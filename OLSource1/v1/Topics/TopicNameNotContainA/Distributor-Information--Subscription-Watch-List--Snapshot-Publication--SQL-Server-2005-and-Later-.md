# Distributor Information, Subscription Watch List (Snapshot Publication, SQL Server 2005 and Later)
Information for snapshot publication subscriptions includes the name of the Publisher. Otherwise, the functionality and the information that is provided in this dialog box is the same as in the Publisher view. For more information about how to use this dialog box, see [Publisher Information, Subscription Watch List (Snapshot Publication, SQL Server 2005 and Later)](../../Topics/TopicNameNotContainA/Publisher-Information--Subscription-Watch-List--Snapshot-Publication--SQL-Server-2005-and-Later-.md).
