# Report Properties Dialog Box, Code (Report Builder)
Select **Code** on the **Report Properties** dialog box to define custom code to be used by expressions within the report.  
  
## Options  
 **Custom Code**  
 Type the methods to be used within the report. Code must be written in [!INCLUDE[msCoName](../../Topics/TopicNameContainA/includes/msCoName_md.md)] [!INCLUDE[vbprvb](../../Topics/TopicNameContainA/includes/vbprvb_md.md)].
