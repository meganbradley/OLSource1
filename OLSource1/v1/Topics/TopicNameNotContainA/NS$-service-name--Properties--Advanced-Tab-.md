# NS$&lt;service name&gt; Properties (Advanced Tab)
[!INCLUDE[ssNS](../../Topics/TopicNameNotContainA/includes/ssNS_md.md)] is implemented as a service named in the format **NS$***<user_configured name>*. If custom properties are defined, they appear on this tab, with their values. This tab may have no properties listed.
