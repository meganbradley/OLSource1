# Prepare for upgrade by running Upgrade Advisor
Prepare to upgrade your OLTP databases and to adopt new features in [!INCLUDE[ssSQL15](../../Topics/TopicNameContainA/includes/ssSQL15_md.md)] by downloading and running SQL Server 2016 Upgrade Advisor.  
  
## Download and install Upgrade Advisor  
 Download and install Upgrade Advisor from [here](http://go.microsoft.com/fwlink/?LinkID=613421). This tool is not included on the [!INCLUDE[ssSQL15](../../Topics/TopicNameContainA/includes/ssSQL15_md.md)] installation media.  
  
## What you can do with Upgrade Advisor  
  
-   Identify upgrade issues in your relational databases by running Database Upgrade Analyzer.  
  
-   Identify databases and tables that are eligible for Stretch Database by running Stretch Database Advisor. For more info, see [Identify databases and tables for Stretch Database by running Stretch Database Advisor](../../Topics/TopicNameNotContainA/Identify-databases-and-tables-for-Stretch-Database-by-running-Stretch-Database-Advisor.md).
