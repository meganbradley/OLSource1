# Keep the Affinity Mask Default Value
  This rule checks whether the [!INCLUDE[ssNoVersion](../../Token\Other/ssNoVersion_md.md)] setting for the affinity mask is set to 0, which is the default value. The affinity mask option dynamically controls CPU affinity.
