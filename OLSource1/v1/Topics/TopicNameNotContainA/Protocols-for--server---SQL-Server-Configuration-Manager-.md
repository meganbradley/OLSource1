# Protocols for &lt;server&gt; (SQL Server Configuration Manager)
  To configure the protocols used by the [!INCLUDE[ssDE](../../Token\Other/ssDE_md.md)], select **Protocols** for the desired server instance in the console pane, right\-click the appropriate protocol in the details pane, and then click **Properties**.
