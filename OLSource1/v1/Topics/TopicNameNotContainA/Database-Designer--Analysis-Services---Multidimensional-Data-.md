# Database Designer (Analysis Services - Multidimensional Data)
  Use the Database Designer in [!INCLUDE[ssBIDevStudioFull](../../Token\Other/ssBIDevStudioFull_md.md)] to change the properties of and review warnings generated by an [!INCLUDE[msCoName](../../Token\Other/msCoName_md.md)] [!INCLUDE[ssNoVersion](../../Token\Other/ssNoVersion_md.md)] [!INCLUDE[ssASnoversion](../../Token\Other/ssASnoversion_md.md)] database.  
  
## Tabs  
 The Database Designer contains the following tabs:  
  
 [General](../Topic/General%20\(Database%20Designer\)%20\(Analysis%20Services%20-%20Multidimensional%20Data\).md)  
 Use the **General** tab to change the properties of an [!INCLUDE[ssASnoversion](../../Token\Other/ssASnoversion_md.md)] database.  
  
 [Warnings](../Topic/Warnings%20\(Database%20Designer\)%20\(Analysis%20Services%20-%20Multidimensional%20Data\).md)  
 Use the **Warnings** tab to view and dismiss rules globally, and to view and re\-enable specific instances of dismissed warnings.
