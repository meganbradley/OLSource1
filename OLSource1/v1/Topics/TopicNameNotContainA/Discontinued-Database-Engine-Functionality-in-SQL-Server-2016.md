# Discontinued Database Engine Functionality in SQL Server 2016
  This topic describes the [!INCLUDE[ssDE](../../Token\Other/ssDE_md.md)] features that are no longer available in [!INCLUDE[ssCurrent](../../Token\Other/ssCurrent_md.md)].  
  
## Discontinued Features in [!INCLUDE[ssSQL15](../../Token\Other/ssSQL15_md.md)]  
  
-   [!INCLUDE[ssSQL15](../../Token\Other/ssSQL15_md.md)] is a 64\-bit application. 32\-bit installation is discontinued, though some elements run as 32\-bit components.  
  
-   Compatibility level 90 is discontinued. For more information, see [ALTER DATABASE Compatibility Level &#40;Transact-SQL&#41;](../Topic/ALTER%20DATABASE%20Compatibility%20Level%20\(Transact-SQL\).md).
