# Discontinued Database Engine Functionality in SQL Server 2016
This topic describes the [!INCLUDE[ssDE](../../Topics/TopicNameContainA/includes/ssDE_md.md)] features that are no longer available in [!INCLUDE[ssCurrent](../../Topics/TopicNameContainA/includes/ssCurrent_md.md)].  
  
## Discontinued Features in [!INCLUDE[ssSQL15](../../Topics/TopicNameContainA/includes/ssSQL15_md.md)]  
  
-   [!INCLUDE[ssSQL15](../../Topics/TopicNameContainA/includes/ssSQL15_md.md)] is a 64-bit application. 32-bit installation is discontinued, though some elements run as 32-bit components.  
  
-   Compatibility level 90 is discontinued. For more information, see [ALTER DATABASE Compatibility Level (Transact-SQL)](assetId:///ca5fd220-d5ea-4182-8950-55d4101a86f6).
