# Source database (Import Analysis Services Database Wizard)
  Use the **Source database** page to select the [!INCLUDE[ssASnoversion](../../Token\Other/ssASnoversion_md.md)] instance and database from which to create an [!INCLUDE[ssASnoversion](../../Token\Other/ssASnoversion_md.md)] project.  
  
## Options  
 **Server**  
 Type the [!INCLUDE[ssASnoversion](../../Token\Other/ssASnoversion_md.md)] instance that contains the database from which to create the project.  
  
 **Database**  
 Type or select the database from which to create the project.
