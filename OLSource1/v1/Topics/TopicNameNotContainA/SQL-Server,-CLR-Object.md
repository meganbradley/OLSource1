# SQL Server, CLR Object
  The **SQLServer:CLR** object provides counters to monitor common language runtime \(CLR\) execution in [!INCLUDE[msCoName](../../Token\Other/msCoName_md.md)] [!INCLUDE[ssNoVersion](../../Token\Other/ssNoVersion_md.md)].  
  
 The following table describes the [!INCLUDE[ssNoVersion](../../Token\Other/ssNoVersion_md.md)] **CLR** counters.  
  
|CLR counters|Description|  
|------------------|-----------------|  
|**CLR Execution**|Total execution time in CLR \(microseconds\)|
