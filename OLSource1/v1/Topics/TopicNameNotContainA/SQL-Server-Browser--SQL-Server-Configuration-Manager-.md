# SQL Server Browser (SQL Server Configuration Manager)
This is the service that supports [!INCLUDE[ssNoVersion](../../Topics/TopicNameContainA/includes/ssNoVersion_md.md)] Browser. To view the properties of the service, right-click the service in the details pane, and then click **Properties**.  
  
 For a description of the [!INCLUDE[ssNoVersion](../../Topics/TopicNameContainA/includes/ssNoVersion_md.md)] Browser service and how it works, see [SQL Server Browser Service](../../Topics/TopicNameNotContainA/SQL-Server-Browser-Service.md).
