# Dimension Structure (Dimension Designer) (Analysis Services - Multidimensional Data)
  Use the **Dimension Structure** page in Dimension Designer to modify the attributes, hierarchies, and levels of the selected dimension. The **Dimension Structure** page contains a toolbar and three panes.  
  
 **To display the Dimension Structure page**  
  
-   In [!INCLUDE[ssBIDevStudioFull](../../Token\Other/ssBIDevStudioFull_md.md)], open anÂ [!INCLUDE[ssASnoversion](../../Token\Other/ssASnoversion_md.md)] project, and then open the dimension that you want to view.  
  
     By default, the **Dimension Structure** page appears when you first open a dimension in Dimension Designer.  
  
## Toolbar  
 Use the toolbar to perform common actions in the **Dimension Structure** page of Dimension Designer. For more information about this pane, see [Toolbar &#40;Dimension Structure Tab, Dimension Designer&#41; &#40;Analysis Services - Multidimensional Data&#41;](../Topic/Toolbar%20\(Dimension%20Structure%20Tab,%20Dimension%20Designer\)%20\(Analysis%20Services%20-%20Multidimensional%20Data\).md).  
  
## Panes  
 The **Dimension Structure** page contains the following panes:  
  
 [Data Source View](../Topic/Data%20Source%20View%20\(Dimension%20Structure%20Tab,%20Dimension%20Designer\)%20\(Analysis%20Services%20-%20Multidimensional%20Data\).md)  
 Use the **Data Source View** pane to explore and manage the data source view associated with a dimension.  
  
 [Hierarchies](../Topic/Hierarchies%20\(Dimension%20Structure%20Tab,%20Dimension%20Designer\)%20\(Analysis%20Services%20-%20Multidimensional%20Data\).md)  
 Use the **Hierarchies** pane to manage the hierarchies and levels for a dimension.  
  
 [Attributes](../Topic/Attributes%20\(Dimension%20Structure%20Tab,%20Dimension%20Designer\)%20\(Analysis%20Services%20-%20Multidimensional%20Data\).md)  
 Use the **Attributes** pane to explore and manage attributes for a dimension.
