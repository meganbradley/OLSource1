# Monitor and Enforce Best Practices by Using Policy-Based Management
  Policy\-Based Management allows you to monitor best practices for the [!INCLUDE[ssDEnoversion](../../Token\Other/ssDEnoversion_md.md)].Â [!INCLUDE[ssNoVersion](../../Token\Other/ssNoVersion_md.md)] provides a set of policy files you can import as best practice policies, and then evaluate the policies against a target set that includes instances, instance objects, databases, or database objects. Evaluate policies manually, set policies to evaluate a target set according to a schedule, or set policies to evaluate a target set according to an event. For more information about Policy\-Based Management, see [Administer Servers by Using Policy-Based Management](../../Topics\TopicNameNotContainA/Administer-Servers-by-Using-Policy-Based-Management.md).
