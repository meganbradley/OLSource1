To learn about roles in tabular models, see [Roles &#40;SSAS Tabular&#41;](../Topic/Roles%20\(SSAS%20Tabular\).md). To learn about creating and managing roles in a tabular model project by using Role Manager in [!INCLUDE[ssBIDevStudioFull](../../Token\Other/ssBIDevStudioFull_md.md)], see [Create and Manage Roles &#40;SSAS Tabular&#41;](../Topic/Create%20and%20Manage%20Roles%20\(SSAS%20Tabular\).md).
