# Specify Availability Group Name Page (New Availability Group Wizard/Add Database Wizard)
  This topic describes the options of the **Specify Availability Group Name** page. This topic is used by both the [!INCLUDE[ssAoNewAgWiz](../../Topics/TopicNameNotContainA/includes/ssAoNewAgWiz_md.md)] and [!INCLUDE[ssAoAddDbWiz](../../Topics/TopicNameNotContainA/includes/ssAoAddDbWiz_md.md)] of [!INCLUDE[ssCurrent](../../Topics/TopicNameContainA/includes/ssCurrent_md.md)].  
  
##  <a name="PageOptions"></a> Specify Availability Group Name Options  
 **Specify an availability group name**  
 Specify the name of the availability group. For a new availability group, specify a valid [!INCLUDE[ssNoVersion](../../Topics/TopicNameContainA/includes/ssNoVersion_md.md)] identifier that is unique across all availability groups in the WSFC cluster. The maximum length for an availability group name is 128 characters.
