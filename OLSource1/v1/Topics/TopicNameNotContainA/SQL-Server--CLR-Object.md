# SQL Server, CLR Object
The **SQLServer:CLR** object provides counters to monitor common language runtime (CLR) execution in [!INCLUDE[msCoName](../../Topics/TopicNameContainA/includes/msCoName_md.md)] [!INCLUDE[ssNoVersion](../../Topics/TopicNameContainA/includes/ssNoVersion_md.md)].  
  
 The following table describes the [!INCLUDE[ssNoVersion](../../Topics/TopicNameContainA/includes/ssNoVersion_md.md)] **CLR** counters.  
  
|CLR counters|Description|  
|------------------|-----------------|  
|**CLR Execution**|Total execution time in CLR (microseconds)|
