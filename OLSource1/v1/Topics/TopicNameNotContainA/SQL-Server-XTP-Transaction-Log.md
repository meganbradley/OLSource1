# SQL Server XTP Transaction Log
The SQL Server XTP Transaction Log performance object contains counters related to In-Memory OLTP transaction log activity in [!INCLUDE[ssNoVersion](../../Topics/TopicNameContainA/includes/ssNoVersion_md.md)].  
  
 This table describes the **SQL Server XTP Transaction Log** counters.  
  
|Counter|Description|  
|-------------|-----------------|  
|**Log bytes written/sec**|The number of bytes written to the [!INCLUDE[ssNoVersion](../../Topics/TopicNameContainA/includes/ssNoVersion_md.md)] transaction log by the In-Memory OLTP engine (on average), per second.|  
|**Log records written/sec**|The number of records written to the [!INCLUDE[ssNoVersion](../../Topics/TopicNameContainA/includes/ssNoVersion_md.md)] transaction log by the In-Memory OLTP engine (on average), per second.|
