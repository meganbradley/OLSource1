# Export As Policy Dialog Box
Use this dialog box to save a set of Policy-Based Management facet properties as a policy. Policies can be saved in an XML file, or saved directly to the current instance of the [!INCLUDE[ssDE](../../Topics/TopicNameContainA/includes/ssDE_md.md)].  
  
## Options  
 **Policy definition name**  
 Type a name for the policy.  
  
 **Condition name**  
 Type a name for the condition.  
  
 **To local server**  
 Select to save the policy on the local server instance.  
  
 **To file**  
 To save the policy as an XML file, type the path and name of the file or use the Browse (**...**) button.
