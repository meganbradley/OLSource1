# Select SMO Enumeration Dialog Box
Use the **Select SMO Enumeration** dialog box to specify the [!INCLUDE[ssNoVersion](../../Topics/TopicNameContainA/includes/ssNoVersion_md.md)] Management Objects (SMO) object on the specified instance of [!INCLUDE[ssNoVersion](../../Topics/TopicNameContainA/includes/ssNoVersion_md.md)] to enumerate, and to select the enumeration type.  
  
## Options  
 **Enumerate**  
 Expand the server and select the SMO object.  
  
 **Objects**  
 Use the Objects enumeration type.  
  
 **Prepopulate**  
 Use the **Prepopulate** option with the Objects enumeration type.  
  
 **Names**  
 Use the Names enumeration type.  
  
 **URNs**  
 Use the URNs enumeration type.  
  
 **Locations**  
 Use the Locations enumeration type. This option is available only for files.
