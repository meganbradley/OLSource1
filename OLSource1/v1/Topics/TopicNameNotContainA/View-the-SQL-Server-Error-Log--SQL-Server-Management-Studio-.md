# View the SQL Server Error Log (SQL Server Management Studio)
The [!INCLUDE[ssNoVersion](../../Topics/TopicNameContainA/includes/ssNoVersion_md.md)] error log contains user-defined events and certain system events. You can use this error log to troubleshoot problems related to [!INCLUDE[ssNoVersion](../../Topics/TopicNameContainA/includes/ssNoVersion_md.md)].  
  
### To view the SQL Server error log  
  
1.  In **Object Explorer**, expand a server, expand **Management**, and then expand **SQL Server Logs**.  
  
2.  Right-click a log and click **View SQL Server Log**.
