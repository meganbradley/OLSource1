# Results Page (Always On Availability Group Wizards)
This help topic describes the options of the **Results** page. This topic applies to the [!INCLUDE[ssAoNewAgWiz](../../Topics/TopicNameNotContainA/includes/ssAoNewAgWiz_md.md)], [!INCLUDE[ssAoAddRepWiz](../../Topics/TopicNameNotContainA/includes/ssAoAddRepWiz_md.md)], [!INCLUDE[ssAoAddDbWiz](../../Topics/TopicNameNotContainA/includes/ssAoAddDbWiz_md.md)], and [!INCLUDE[ssAoFoAgWiz](../../Topics/TopicNameNotContainA/includes/ssAoFoAgWiz_md.md)] of [!INCLUDE[ssCurrent](../../Topics/TopicNameContainA/includes/ssCurrent_md.md)]. Use this page to view the results of the wizard.  
  
##  <a name="PageOptions"></a> Results Page Options  
 The **Summary** grid contains the following columns:  
  
 **Name**  
 Displays a phrase that describes a specific operation.  
  
 **Result**  
 For each completed step, displays one of the following hyperlink texts.  
  
|Result|Description|  
|------------|-----------------|  
|**Error**|Indicates that the validation step failed. Click the link to view the error message.|  
|**Success**|Indicates that the validation step completed successfully.|
