# SQL Server 2016 Resources
  This article provides links to videos, samples and community resources for [!INCLUDE[ssCurrent](../../Token\Other/ssCurrent_md.md)].  
  
 Learn about  new[!INCLUDE[ssCurrent](../../Token\Other/ssCurrent_md.md)] features in [What's New in SQL Server 2016 Release Candidate &#40;RC3&#41;](../Topic/What's%20New%20in%20SQL%20Server%202016%20Release%20Candidate%20\(RC3\).md).
