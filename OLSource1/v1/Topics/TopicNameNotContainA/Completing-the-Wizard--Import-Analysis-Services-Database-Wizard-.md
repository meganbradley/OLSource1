# Completing the Wizard (Import Analysis Services Database Wizard)
  Use the **Completing the Wizard** page to display the progress of the wizard as the data and metadata from the selected [!INCLUDE[ssASnoversion](../../Token\Other/ssASnoversion_md.md)] database is imported into the [!INCLUDE[ssASnoversion](../../Token\Other/ssASnoversion_md.md)] project.  
  
## Options  
 **\(Progress\)**  
 Displays the progress of the wizard as each object in the database is imported into the [!INCLUDE[ssASnoversion](../../Token\Other/ssASnoversion_md.md)] project.
