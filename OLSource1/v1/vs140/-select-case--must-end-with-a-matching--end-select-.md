# &#39;Select Case&#39; must end with a matching &#39;End Select&#39;
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement occurs without a corresponding <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement. An <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement must be used to end the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> block.  
  
 **Error ID:** BC30095  
  
### To correct this error  
  
1.  If this <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> block is part of a set of nested <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> blocks, make sure each block is properly terminated.  
  
2.  Add an <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> statement to the end of the <CodeContentPlaceHolder>8\</CodeContentPlaceHolder> block.
