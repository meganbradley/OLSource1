# CMFCBaseTabCtrl::SetTabsOrder
Arranges the tabs in the specified order.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 An array of zero-based indexes that defines the new tab order.  
  
## Return Value  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> if successful; <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> otherwise.  
  
## Remarks  
 The size of the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> array must be equal to the number of tabs in the tab control.

 **Header:** afxbasetabctrl.h
