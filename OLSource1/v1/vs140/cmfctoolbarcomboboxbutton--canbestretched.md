# CMFCToolBarComboBoxButton::CanBeStretched
Indicates whether the combo box button size can change.  
  
## Syntax  
  
```  
virtual BOOL CanBeStretched() const;  
```  
  
## Return Value  
 Returns `TRUE`.  
  
## Requirements  
 **Header:** afxtoolbarcomboboxbutton.h
