# numpunct::do_decimal_point
A protected virtual member function that is called to return a locale-specific element to use as a decimal point.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 A locale-specific element to use as a decimal point.  
  
## Example  
 See the example for [decimal_point](../vs140/numpunct--decimal_point.md), where the virtual member function is called by <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
## Requirements  
 **Header:** \<locale>  
  
 **Namespace:** std
