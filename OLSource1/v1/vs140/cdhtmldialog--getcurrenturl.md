# CDHtmlDialog::GetCurrentUrl
Retrieves the Uniform Resource Locator (URL) associated with the current document.  
  
## Syntax  
  
```  
  
      void GetCurrentUrl(  
   CString& szUrl   
);  
```  
  
#### Parameters  
 `szUrl`  
 A [CString](../vs140/cstringt-class.md) object containing the URL to retrieve.  
  
## Requirements  
 **Header:** afxdhtml.h
