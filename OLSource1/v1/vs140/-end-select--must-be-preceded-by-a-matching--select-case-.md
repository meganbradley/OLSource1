# &#39;End Select&#39; must be preceded by a matching &#39;Select Case&#39;

1.  If this <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> block is part of a set of nested <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> blocks, make sure each block is properly terminated.
