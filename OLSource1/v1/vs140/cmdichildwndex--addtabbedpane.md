# CMDIChildWndEx::AddTabbedPane
Adds a tabbed pane.  
  
## Syntax  
  
```  
void AddTabbedPane(  
   CDockablePane* pControlBar  
);  
```  
  
#### Parameters  
 [in] `pControlBar`  
 A pointer to the pane.
