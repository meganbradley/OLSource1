# CBasePane::GetMFCStyle
Returns the pane styles that are specific to MFC.  
  
## Syntax  
  
```  
virtual DWORD GetMFCStyle() const;  
```  
  
## Return Value  
 A combination of library-specific (AFX_CBRS_) pane styles.  
  
## Requirements  
 **Header:** afxbasepane.h
