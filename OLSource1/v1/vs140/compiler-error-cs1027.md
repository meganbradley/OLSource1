# Compiler Error CS1027
\#endif directive expected  
  
 A matching <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> [preprocessor directive](../vs140/csharp-preprocessor-directives.md) was not found for a specified <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> directive. Or, the compiler may have found a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> directive when there was no matching <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> directive inside a <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> block.  
  
 The following sample generates CS1027:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
