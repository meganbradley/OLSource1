# CMFCVisualManagerOffice2003::IsDefaultWinXPColorsEnabled
Indicates whether the visual manager uses theme colors that are native to Windows XP.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the visual manager uses native colors; otherwise, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Remarks  
 For more information about native colors, see [CMFCVisualManagerOffice2003::SetDefaultWinXPColors](../vs140/cmfcvisualmanageroffice2003--setdefaultwinxpcolors.md).

 **Header:** afxvisualmanageroffice2003.h
