# CHandle::operator HANDLE
Returns the value of the stored handle.  
  
## Syntax  
  
```  
  
operator HANDLE( ) const throw( );  
  
```  
  
## Remarks  
 Returns the value stored in [CHandle::m_h](../vs140/chandle--m_h.md).  
  
## Requirements  
 **Header:** atlbase.h
