# basic_string::value_type
A type that represents the type of characters stored in a string.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 It is equivalent to **traits_type::char_type** and is equivalent to <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> for objects of type **string**.  
  
## Example  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 **The character ch1 is: G.**  
**The character ch2 is: H.**   
## Requirements  
 **Header:** \<string>  
  
 **Namespace:** std
