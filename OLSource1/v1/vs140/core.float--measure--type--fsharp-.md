# Core.float&lt;&#39;Measure&gt; Type (F#)
The type of floating point numbers, annotated with a unit of measure. The unit of measure is erased in compiled code and when values of this type are analyzed using reflection. The type is representationally equivalent to <xref:System.Double*>.  
  
 **Namespace/Module Path:** Microsoft.FSharp.Core  
  
 **Assembly:** FSharp.Core (in FSharp.Core.dll)  
  
## Syntax  
  
```  
[<MeasureAnnotatedAbbreviationAttribute>]  
type float<'Measure> = float  
```
