# CFileTimeSpan::GetTimeSpan
Call this method to retrieve the time span from the `CFileTimeSpan` object.  
  
## Syntax  
  
```  
  
LONGLONG GetTimeSpan( ) const throw( );  
  
```  
  
## Return Value  
 Returns the time span in milliseconds.  
  
## Requirements  
 **Header:** atltime.h
