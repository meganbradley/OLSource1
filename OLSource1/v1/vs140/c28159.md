# C28159
warning C28159: Consider using another function instead.  
  
 This warning is reported for Drivers is suggesting that you use a preferred function call that is semantically equivalent to the function that the driver is calling. This is a general warning message; the annotation <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> was used (possibly with a conditional a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>() annotation) to flag a bad coding practice.  
  
## Example  
 The following code example generates this warning:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 The following code example avoids this warning:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
