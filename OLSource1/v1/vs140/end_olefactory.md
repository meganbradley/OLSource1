# END_OLEFACTORY
Ends the declaration of your control's class factory.  
  
## Syntax  
  
```  
  
END_OLEFACTORY(  
class_name )  
```  
  
#### Parameters  
 *class_name*  
 The name of the control class whose class factory this is.  
  
## Requirements  
 **Header:** afxctl.h
