# sinpi Function
Calculates the sine value of pi * _X  
  
## Syntax  
  
```  
inline float sinpi(  
   float _X  
) restrict(amp);  
inline double sinpi(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the sine value of pi * _X  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
