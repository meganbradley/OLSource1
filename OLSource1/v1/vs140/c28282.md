# C28282
warning C28282: Format Strings must be in preconditions  
  
 This warning indicates that a <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> annotation is found, which is not in a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> (<CodeContentPlaceHolder>2\</CodeContentPlaceHolder>) annotation (function parameters are by default <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>). Check whether the annotation used in an explicit block with a <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> (<CodeContentPlaceHolder>5\</CodeContentPlaceHolder>) annotation. If so, remove the annotation from any enclosing block that has put it in a <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> context.
