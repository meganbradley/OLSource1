# Compiler Warning (level 1) CS1589
Unable to include XML fragment 'fragment' of file 'file' -- reason  
  
 The syntax (*fragment*) of a [\<include>](../vs140/-include---csharp-programming-guide-.md) tag, which referenced a file (<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>), was incorrect for the specified ***reason***.  
  
 A malformed line will be placed in the generated XML file.  
  
 The following sample generates CS1589:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
