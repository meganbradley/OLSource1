# Compiler Error CS0535
'class' does not implement interface member 'member'  
  
 A [class](../vs140/class--csharp-reference-.md) derived from an [interface](../vs140/interface--csharp-reference-.md), but the class did not implement one or more of the interface's members. A class must implement all members of interfaces from which it derives or else be declared <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.
