# CMFCColorPickerCtrl::SetLuminance
Changes the luminance of the currently selected color.  
  
## Syntax  
  
```  
void SetLuminance(  
   double Luminance   
);  
```  
  
#### Parameters  
 [in] `Luminance`  
 A luminance value.
