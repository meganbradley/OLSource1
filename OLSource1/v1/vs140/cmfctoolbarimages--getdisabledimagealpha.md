# CMFCToolBarImages::GetDisabledImageAlpha
Returns the alpha channel (opacity) value that is used for disabled images.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 The current alpha channel value.  
  
## Remarks  
 You can call [CMFCToolBarImages::SetDisabledImageAlpha](../vs140/cmfctoolbarimages--setdisabledimagealpha.md) to change the alpha channel value.  
  
## Requirements  
 **Header:** afxtoolbarimages.h
