# &#39;RemoveHandler&#39; is already declared
More than one <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> declaration appeared in a custom event declaration. An <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> declaration declares procedure used to remove an event handler.  
  
 **Error ID:** BC31128  
  
### To correct this error  
  
-   Remove the redundant <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement.
