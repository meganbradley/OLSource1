# Compiler Error C2174
'function' : actual parameter has type 'void' : parameter number1, parameter list number2  
  
 Parameter <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> passed to parameter list <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> is a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> parameter. Parameters cannot have type <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>. Use <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> instead.
