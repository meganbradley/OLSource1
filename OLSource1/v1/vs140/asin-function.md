# asin Function
Calculates the arcsine of the argument  
  
## Syntax  
  
```  
inline float asin(  
   float _X  
) restrict(amp);  
inline double asin(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the arcsine value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
