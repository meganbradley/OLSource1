# Compiler Warning (level 1) CS2023
Ignoring /noconfig option because it was specified in a response file  
  
 The [/noconfig](../vs140/-noconfig--csharp-compiler-options-.md) compiler option was specified in a response file, which is not allowed.  
  
 This warning cannot be suppressed by the [/nowarn](../vs140/-nowarn--csharp-compiler-options-.md) option.
