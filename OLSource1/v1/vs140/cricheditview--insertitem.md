# CRichEditView::InsertItem
Call this function to insert a [CRichEditCntrItem](../vs140/cricheditcntritem-class.md) object into a rich edit view.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 Pointer to the item to be inserted.  
  
## Return Value  
 An <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> value indicating the success of the insertion.  
  
## Remarks  
 For more information on <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, see [Structure of COM Error Codes](http://msdn.microsoft.com/library/windows/desktop/ms690088) in the [!INCLUDE[winSDK](../vs140/includes/winsdk_md.md)].

 **Header:** afxrich.h
