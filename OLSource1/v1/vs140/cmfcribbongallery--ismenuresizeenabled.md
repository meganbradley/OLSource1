# CMFCRibbonGallery::IsMenuResizeEnabled
Specifies whether menu resizing is enabled.  
  
## Syntax  
  
```  
BOOL IsMenuResizeEnabled() const;  
```  
  
## Return Value  
 `TRUE` if menu resizing has been enabled; otherwise, `FALSE`.
