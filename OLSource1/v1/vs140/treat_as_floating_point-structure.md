# treat_as_floating_point Structure
Specifies whether <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> can be treated as a floating-point type.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> can be treated as a floating-point type only when the specialization <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> is derived from [true_type](../vs140/-type_traits--typedefs.md#true_type_typedef). The template class can be specialized for a user-defined type.  
  
## Requirements  
 **Header:** chrono  
  
 **Namespace:** std::chrono
