# CMFCToolBarImages::SmoothResize
Smoothly resizes underlined images.  
  
## Syntax  
  
```  
BOOL SmoothResize(  
   double dblImageScale  
);  
```  
  
#### Parameters  
 `dblImageScale`  
 Scale ratio.  
  
## Return Value  
 `TRUE` if resize succeeds; otherwise `FALSE`.
