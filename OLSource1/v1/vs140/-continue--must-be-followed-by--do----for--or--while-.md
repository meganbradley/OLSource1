# &#39;Continue&#39; must be followed by &#39;Do&#39;, &#39;For&#39; or &#39;While&#39;

 **Error ID:** BC30781  
  
### To correct this error  
  
1.  If the <CodeContentPlaceHolder>8\</CodeContentPlaceHolder> statement is in a <CodeContentPlaceHolder>9\</CodeContentPlaceHolder> loop, change the statement to <CodeContentPlaceHolder>10\</CodeContentPlaceHolder>.  
  
2.  If the <CodeContentPlaceHolder>11\</CodeContentPlaceHolder> statement is in a <CodeContentPlaceHolder>12\</CodeContentPlaceHolder> loop, change the statement to <CodeContentPlaceHolder>13\</CodeContentPlaceHolder>.  
  
3.  If the <CodeContentPlaceHolder>14\</CodeContentPlaceHolder> statement is in a <CodeContentPlaceHolder>15\</CodeContentPlaceHolder> loop, change the statement to <CodeContentPlaceHolder>16\</CodeContentPlaceHolder>.  
  
4.  Otherwise, remove the <CodeContentPlaceHolder>17\</CodeContentPlaceHolder> statement.
