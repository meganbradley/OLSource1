# CMFCRibbonStatusBar::AddSeparator
Adds a separator to the ribbon status bar.  
  
## Syntax  
  
```  
void AddSeparator();  
```  
  
## Remarks  
 The framework adds a separator after the method [CMFCRibbonStatusBar::AddElement](../vs140/cmfcribbonstatusbar--addelement.md). inserts the last element.  
  
## Requirements  
 **Header:** afxribbonstatusbar.h
