# &#39;Group&#39; or an identifier expected
The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> portion of a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> clause does not include the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> keyword. You must include the <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> keyword in the <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> clause of a <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>8\</CodeContentPlaceHolder> clause to identify the variable name to use for the grouped results. This can be either a name you specify or the keyword <CodeContentPlaceHolder>9\</CodeContentPlaceHolder>.  
  
 **Error ID:** BC36707
