# CMFCRibbonPanel::GetPreferedMenuLocation
Retrieves the preferred display rectangle for the pop-up menu of the ribbon panel.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [out] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 This parameter is not used.  
  
## Return Value  
 Always returns <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Remarks  
 This method always returns <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>. Override this method to retrieve the preferred display rectangle for the pop-up menu of the ribbon panel.

 **Header:** afxribbonpanel.h
