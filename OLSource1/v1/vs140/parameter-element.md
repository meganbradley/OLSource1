# Parameter Element
Contains information about a specific parameter for a task that is generated by a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  The name of the element is the name of the parameter.  For more informations, see [UsingTask](../vs140/usingtask-element--msbuild-.md).

 \<Parameter>
