# Compiler Error CS1565
Conflicting options specified: Win32 resource file; Win32 icon  
  
 It is not valid to specify both the [/win32res](../vs140/-win32res--csharp-compiler-options-.md) compiler option and the [/win32ico](../vs140/-win32icon--csharp-compiler-options-.md) compiler option in the same compilation.
