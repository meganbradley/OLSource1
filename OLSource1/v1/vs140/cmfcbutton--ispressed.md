# CMFCButton::IsPressed
Indicates whether a button is pushed and highlighted.  
  
## Syntax  
  
```  
BOOL IsPressed() const;  
```  
  
## Return Value  
 TRUE if the button is pressed; otherwise, FALSE.

## Requirements
