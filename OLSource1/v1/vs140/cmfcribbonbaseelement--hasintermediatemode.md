# CMFCRibbonBaseElement::HasIntermediateMode
Specifies whether the ribbon element has an intermediate mode.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the ribbon element has an intermediate mode, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> otherwise. In the intermediate mode, an element displays a small image and text on the right of the image.
