# CBasePane::DockPaneUsingRTTI
Docks the pane by using run-time type information.  
  
## Syntax  
  
```  
void DockPaneUsingRTTI(  
   BOOL bUseDockSite  
);  
```  
  
#### Parameters  
 [in] `bUseDockSite`  
 If `TRUE`, dock to the docking site. If `FALSE`, dock to the parent frame.  
  
## Requirements  
 **Header:** afxbasepane.h
