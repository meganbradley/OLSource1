# IDebugProcess2::CanDetach
Determines if the session debug manager (SDM) can detach the process.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>

 If successful, returns <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> Returns <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> if the debugger cannot detach from the process. Otherwise, returns an error code.
