# basic_streambuf::pos_type
Associates a type name within basic_streambuf scope with one of the types in a template parameter.  
  
## Syntax  
  
```  
  
typedef typename traits  
_  
type::pos  
_  
type pos  
_  
type;  
  
```  
  
## Requirements  
 **Header:** <streambuf\>  
  
 **Namespace:** std
