# Methods cannot be declared &#39;Static&#39;
The <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> modifier cannot be applied to entire procedures.  
  
 **Error ID:** BC30810  
  
### To correct this error  
  
-   Use the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> modifier to declare individual items within a procedure as static.
