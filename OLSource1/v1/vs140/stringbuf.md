# stringbuf
Creates a type `basic_stringbuf` specialized on a `char` template parameter.  
  
## Syntax  
  
```  
  
typedef basic  
_  
stringbuf<char> stringbuf;  
  
```  
  
## Remarks  
 The type is a synonym for template class [basic_stringbuf](../vs140/basic_stringbuf-class.md), specialized for elements of type `char`*.*  
  
## Requirements  
 **Header:** \<sstream>  
  
 **Namespace:** std
