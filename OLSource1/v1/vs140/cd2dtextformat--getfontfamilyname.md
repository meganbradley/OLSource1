# CD2DTextFormat::GetFontFamilyName
Gets a copy of the font family name.  
  
## Syntax  
  
```  
CString GetFontFamilyName() const;  
```  
  
## Return Value  
 CString object that contains the current font family name.  
  
## Requirements  
 **Header:** afxrendertarget.h
