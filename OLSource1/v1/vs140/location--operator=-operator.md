# location::operator= Operator
Assigns the contents of a different `location` object to this one.  
  
## Syntax  
  
```  
location& operator=(  
   const location& _Rhs  
);  
```  
  
#### Parameters  
 `_Rhs`  
 The source `location` object.

## Requirements  
 **Header:** concrt.h  
  
 **Namespace:** concurrency
