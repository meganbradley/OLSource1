# CAutoVectorPtr::~CAutoVectorPtr
The destructor.  
  
## Syntax  
  
```  
  
~CAutoVectorPtr( ) throw( );  
  
```  
  
## Remarks  
 Frees any allocated resources. Calls [CAutoVectorPtr::Free](../vs140/cautovectorptr--free.md).
