# map::size_type
An unsigned integer type that can represent the number of elements in a map.  
  
## Syntax  
  
```  
typedef typename allocator_type::size_type size_type;  
```  
  
## Example  
 See the example for [size](../vs140/map--size.md) for an example of how to declare and use `size_type`.  
  
## Requirements  
 **Header:** \<map>  
  
 **Namespace:** std
