# &#39;End Sub&#39; expected
An <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement must match the block it terminates. A <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> procedure definition has not been properly terminated.  
  
 **Error ID:** BC30026  
  
### To correct this error  
  
-   Add an <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement to the end of the procedure.
