# time_put::do_put
A virtual function that outputs time and date information as a sequence of **CharType**s.

#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 An output iterator where the sequence of characters representing time and date are to be inserted.  
  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 Unused.  
  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 The time and date information being output.  
  
 <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>  
 The format of the output. See [strftime, wcsftime, _strftime_l, _wcsftime_l](../vs140/strftime--wcsftime--_strftime_l--_wcsftime_l.md) for valid values.  
  
 <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>

## Return Value  
 An iterator to the first position after the last element inserted.

 The virtual protected member function generates sequential elements beginning at <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> from time values stored in the object \*<CodeContentPlaceHolder>7\</CodeContentPlaceHolder>, of type **tm**. The function returns an iterator designating the next place to insert an element beyond the generated output.  
  
 The output is generated by the same rules used by <CodeContentPlaceHolder>8\</CodeContentPlaceHolder>, with a last argument of <CodeContentPlaceHolder>9\</CodeContentPlaceHolder>, for generating a series of <CodeContentPlaceHolder>10\</CodeContentPlaceHolder> elements into an array. Each such <CodeContentPlaceHolder>11\</CodeContentPlaceHolder> element is assumed to map to an equivalent element of type **CharType** by a simple, one-to-one mapping. If <CodeContentPlaceHolder>12\</CodeContentPlaceHolder> equals zero, the effective format is "%F", where F is replaced by <CodeContentPlaceHolder>13\</CodeContentPlaceHolder>. Otherwise, the effective format is "%MF", where M is replaced by <CodeContentPlaceHolder>14\</CodeContentPlaceHolder>.
