# LanguagePrimitives.IntrinsicOperators Module (F#)
The F# compiler emits calls to some of the functions in this module as part of the compiled form of some language constructs  
  
 **Namespace/Module Path:** Microsoft.FSharp.Core.LanguagePrimitives  
  
 **Assembly:** FSharp.Core (in FSharp.Core.dll)

## Remarks  
  
## Values  
  
|Value|Description|  
|-----------|-----------------|  
|[( & )](../vs140/intrinsicoperators.------function--fsharp-.md)|Computed the Boolean AND operation. When used as a binary operator the right hand value is evaluated only on demand.|  
|[( && )](../vs140/intrinsicoperators.-------function--fsharp-.md)|Computes the Boolean AND operation. When used as a binary operator the right hand value is evaluated only on demand|  
|[( &#124;&#124; )](../vs140/intrinsicoperators.-------function--fsharp-.md)|Computes the Boolean OR operation. When used as a binary operator the right hand value is evaluated only on demand|  
|[( ~& )](../vs140/intrinsicoperators.--~-----t--function--fsharp-.md)|Returns the address of the argument as a managed pointer. Uses of this value may result in the generation of unverifiable code.|  
|[( ~&& )](../vs140/intrinsicoperators.--~------t--function--fsharp-.md)|Returns the address of the argument as a native pointer. Uses of this value may result in the generation of unverifiable code.|  
|[or](../vs140/intrinsicoperators.or-function--fsharp-.md)|Computes the Boolean OR operation. When used as a binary operator the right hand value is evaluated only on demand.|
