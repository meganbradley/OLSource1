# -verbose
Causes the compiler to produce verbose status and error messages.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Arguments  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> &#124; <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 Optional. Specifying <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> is the same as specifying <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>, which causes the compiler to emit verbose messages. The default for this option is <CodeContentPlaceHolder>6\</CodeContentPlaceHolder>.  
  
## Remarks  
 The <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> option displays information about the total number of errors issued by the compiler, reports which assemblies are being loaded by a module, and displays which files are currently being compiled.  
  
> [!NOTE]
>  The <CodeContentPlaceHolder>8\</CodeContentPlaceHolder> option is not available from within the Visual Studio development environment; it is available only when compiling from the command line.
