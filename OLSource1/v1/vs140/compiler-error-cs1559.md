# Compiler Error CS1559
Cannot use 'object' for Main method because it is imported  
  
 An invalid class was specified to the [/main](../vs140/-main--csharp-compiler-options-.md) compiler option; the class cannot be used as a location for the [Main](../vs140/main---and-command-line-arguments--csharp-programming-guide-.md) method.
