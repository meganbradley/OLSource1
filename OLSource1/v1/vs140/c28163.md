# C28163
warning C28163: The function should never be called from within a try/except block  
  
 This warning is reported when a function is of a type that should never be enclosed in a <CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  block is found in a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> block. The code analysis tool found at least one path in which the function called was within a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> block.
