# CMFCColorButton::m_bEnabledInCustomizeMode
Sets a color button to customization mode.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 If you need to add a color button to a customization dialog's page (or allow the user to make another color selection during customization), enable the button by setting the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> member to <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>. By default, this member is set to <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.
