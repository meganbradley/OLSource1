# wstringbuf
Creates a type `basic_stringbuf` specialized on a `wchar_t` template parameter.  
  
## Syntax  
  
```  
  
typedef basic  
_  
stringbuf<wchar  
_  
t> wstringbuf;  
  
```  
  
## Remarks  
 The type is a synonym for template class [basic_stringbuf](../vs140/basic_stringbuf-class.md), specialized for elements of type `wchar_t`.  
  
## Requirements  
 **Header:** \<sstream>  
  
 **Namespace:** std
