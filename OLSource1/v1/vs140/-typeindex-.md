# &lt;typeindex&gt;
Include the standard header \<typeindex> to define a class and function that support the indexing of objects of class [type_info](../vs140/type_info-class.md).  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 The [hash Structure](../vs140/hash-structure.md) defines a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> that's suitable for mapping values of type [type_index](../vs140/type_index-class.md) to a distribution of index values.  
  
 The <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> class wraps a pointer to a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> object to assist in indexing.
