# CMFCRibbonBaseElement::OnAutoRepeat
Updates the ribbon element in response to sustained user input.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Always returns <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
## Remarks  
 By default this method always return <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>. Override this method to process sustained user input.
