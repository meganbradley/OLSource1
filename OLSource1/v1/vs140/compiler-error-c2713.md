# Compiler Error C2713
only one form of exception handling permitted per function  
  
 You cannot use structured exception handling (<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>/<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>) and C++ exception handling (<CodeContentPlaceHolder>2\</CodeContentPlaceHolder>/<CodeContentPlaceHolder>3\</CodeContentPlaceHolder>) in the same function.
