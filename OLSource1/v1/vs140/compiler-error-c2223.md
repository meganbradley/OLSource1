# Compiler Error C2223
left of '->identifier' must point to struct/union  
  
 The operand to the left of <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> is not a pointer to a class, structure, or union.  
  
 This error can be caused by a left operand that is an undefined variable (therefore type <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>).
