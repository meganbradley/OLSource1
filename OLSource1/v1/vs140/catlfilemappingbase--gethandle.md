# CAtlFileMappingBase::GetHandle
Call this method to return a handle to the file-mapping object.  
  
## Syntax  
  
```  
  
HANDLE GetHandle( ) throw ( );  
  
```  
  
## Return Value  
 Returns a handle to the file-mapping object.  
  
## Requirements  
 **Header:** atlfile.h
