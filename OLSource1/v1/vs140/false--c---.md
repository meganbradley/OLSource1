# false (C++)
The keyword is one of the two values for a variable of type [bool](../vs140/bool--c---.md) or a conditional expression (a conditional expression is now a **true** Boolean expression). For example, if <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> is a variable of type <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement assigns **false** to <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
## Example  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 **1**  
**0**
