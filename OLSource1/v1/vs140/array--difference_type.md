# array::difference_type
The type of a signed distance between two elements.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 The signed integer type describes an object that can represent the difference between the addresses of any two elements in the controlled sequence. It is a synonym for the type <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.
