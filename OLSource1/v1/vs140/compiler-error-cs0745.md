# Compiler Error CS0745
Expected contextual keyword 'by'  
  
 The pattern for the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> clause is <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> followed by an optional <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>, as shown in the following example:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 or  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
### To correct this error  
  
1.  Add the <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> keyword to the <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> clause.  
  
## Example  
 The following code generates CS0745:  
  
<CodeContentPlaceHolder>2\</CodeContentPlaceHolder>
