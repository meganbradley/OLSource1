# &#39;Continue For&#39; can only appear inside a &#39;For&#39; statement
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement can only appear within a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> loop.  
  
 **Error ID:** BC30783  
  
### To correct this error  
  
1.  If the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement is in a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, change the statement to <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
     —or—  
  
     If the <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> statement is in a <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> loop, change the statement to <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>.  
  
2.  Otherwise, remove the <CodeContentPlaceHolder>8\</CodeContentPlaceHolder> statement.
