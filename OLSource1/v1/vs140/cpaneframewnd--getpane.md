# CPaneFrameWnd::GetPane
Returns a pane that is contained in the mini-frame window.  
  
## Syntax  
  
```  
virtual CWnd* GetPane() const;  
```  
  
## Return Value  
 The pane that is contained in the mini-frame, or `NULL` if the mini-frame window contains no panes.
