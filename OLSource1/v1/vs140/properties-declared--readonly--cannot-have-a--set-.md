# Properties declared &#39;ReadOnly&#39; cannot have a &#39;Set&#39;
The <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> procedure writes the value of a property. <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> properties cannot be written.  
  
 **Error ID:** BC30022  
  
### To correct this error  
  
-   Remove the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> keyword from the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement, or remove the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> procedure from the property body.
