# CBaseTransition::m_pRelatedVariable
A pointer to an animation variable, which is animated with the transition stored in m_transition.  
  
## Syntax  
  
```  
CAnimationVariable* m_pRelatedVariable;  
```  
  
## Requirements  
 **Header:** afxanimationcontroller.h
