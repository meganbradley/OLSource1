# Xml (XElement Dynamic Property)
Gets the unformatted XML content of the element.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Property Value/Return Value  
 A \<xref:System.String*> that represents the unformatted XML content of the element.  
  
## Remarks  
 This property is equivalent to the \<xref:System.Xml.Linq.XNode.ToString(System.Xml.Linq.SaveOptions)> method of the \<xref:System.Xml.Linq.XNode*?displayProperty=fullName> class, with the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> parameter set to \<xref:System.Xml.Linq.SaveOptions.DisableFormatting*>.
