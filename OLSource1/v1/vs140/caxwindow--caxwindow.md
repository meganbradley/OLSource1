# CAxWindow::CAxWindow
Constructs a `CAxWindow` object using an existing window object handle.  
  
## Syntax  
  
```  
  
      CAxWindow(  
   HWND hWnd = NULL   
);  
```  
  
#### Parameters  
 `hWnd`  
 A handle to an existing window object.  
  
## Requirements  
 **Header:** atlwin.h
