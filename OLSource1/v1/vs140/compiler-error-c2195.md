# Compiler Error C2195
'identifier' : is a data segment  
  
 The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> pragma uses a segment name used with the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> pragma.  
  
 The following sample generates C2195:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
