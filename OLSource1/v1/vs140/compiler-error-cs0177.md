# Compiler Error CS0177
The out parameter 'parameter' must be assigned to before control leaves the current method  
  
 A parameter marked with the [out](../vs140/out--csharp-reference-.md) keyword was not assigned a value in the method body. For more information, see [Passing Parameters (C# Programmer's Reference)](../vs140/passing-parameters--csharp-programming-guide-.md)
