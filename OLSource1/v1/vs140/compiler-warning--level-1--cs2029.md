# Compiler Warning (level 1) CS2029
Invalid value for '/define'; 'identifier' is not a valid identifier  
  
 This warning occurs if the value that is used in the [/define](../vs140/-define--csharp-compiler-options-.md) option has some invalid characters.  
  
 This warning cannot be suppressed by the [/nowarn](../vs140/-nowarn--csharp-compiler-options-.md) option.
