# C6214
warning C6214: cast between semantically different integer types: HRESULT to a Boolean type  
  
 This warning indicates that an <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> is being cast to a Boolean type. The success value (<CodeContentPlaceHolder>3\</CodeContentPlaceHolder>) of an <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> equals 0. However, 0 indicates failure for a Boolean type. Casting an <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> to a Boolean type and then using it in a test expression will yield an incorrect result. Sometimes, this warning occurs if an <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> is being stored in a Boolean variable. Any comparison that uses the Boolean variable to test for <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> success or failure could lead to incorrect results.
