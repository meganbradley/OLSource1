# CRgn::RectInRegion
Determines whether any part of the rectangle specified by <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> is within the boundaries of the region stored in the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> object.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 Points to a <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> structure or <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> object. The <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> structure has the following form:  
  
 <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>  
  
 <CodeContentPlaceHolder>8\</CodeContentPlaceHolder>  
  
 <CodeContentPlaceHolder>9\</CodeContentPlaceHolder>  
  
 <CodeContentPlaceHolder>10\</CodeContentPlaceHolder>  
  
 <CodeContentPlaceHolder>11\</CodeContentPlaceHolder>  
  
 <CodeContentPlaceHolder>12\</CodeContentPlaceHolder>  
  
## Return Value  
 Nonzero if any part of the specified rectangle lies within the boundaries of the region; otherwise 0.
