# Compiler Error C3044
'section' : only allowed directly nested under an OpenMP 'sections' directive  
  
 The compiler found a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> directive was used incorrectly. For more information, see [sections](../vs140/sections--openmp-.md).  
  
 The following sample generates C3044:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
