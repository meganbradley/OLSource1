# Compiler Warning (level 1) C4630
'symbol' : 'extern' storage class specifier illegal on member definition  
  
 A data member or member function is defined as <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. Members cannot be external, although entire objects can. The compiler ignores the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> keyword. The following sample generates C4630:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
