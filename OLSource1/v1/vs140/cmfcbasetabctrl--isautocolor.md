# CMFCBaseTabCtrl::IsAutoColor
Determines whether the tab control is in autocolor mode.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the tab control is in autocolor mode; <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> otherwise.  
  
## Remarks  
 You can enable or disable the autocolor mode by using the [CMFCBaseTabCtrl::EnableAutoColor](../vs140/cmfcbasetabctrl--enableautocolor.md) method.

 **Header:** afxbasetabctrl.h
