# CHtmlView::SetWidth
Sets the width of the Internet Explorer main window.  
  
## Syntax  
  
```  
  
      void SetWidth(  
   long nNewValue   
);  
```  
  
#### Parameters  
 `nNewValue`  
 The width, in pixels, of the Internet Explorer main window.  
  
## Requirements  
 **Header:** afxhtml.h
