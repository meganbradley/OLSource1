# Expected closing &#39;%&gt;&#39; for embedded expression
An embedded expression in an XML literal is missing the required closing <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> characters.  
  
 **Error ID:** BC31159  
  
### To correct this error  
  
-   Add the required closing <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> characters to the end of the embedded expression.
