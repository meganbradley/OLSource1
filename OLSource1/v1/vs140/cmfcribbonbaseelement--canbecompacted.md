# CMFCRibbonBaseElement::CanBeCompacted
Indicates whether the size of the ribbon element can be compact.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the size of the ribbon element can be compact; otherwise, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Remarks  
 The size of a ribbon element can be compact, intermediate, or large.

 **Header:** afxbaseribbonelement.h
