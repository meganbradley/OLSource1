# &#39;End Event&#39; must be preceded by a matching &#39;Custom Event&#39;
An <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement occurs without a corresponding <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement. <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> must be preceded by a corresponding <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement.  
  
 **Error ID:** BC31123  
  
### To correct this error  
  
-   Ensure that a preceding <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> statement is valid and correctly spelled.
