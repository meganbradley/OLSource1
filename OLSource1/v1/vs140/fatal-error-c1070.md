# Fatal Error C1070
mismatched #if/#endif pair in file 'filename'  
  
 An <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, or <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> directive has no corresponding <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>.  
  
 The following sample generates C1070:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 Possible resolution:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
