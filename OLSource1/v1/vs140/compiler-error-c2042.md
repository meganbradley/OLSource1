# Compiler Error C2042
signed/unsigned keywords mutually exclusive  
  
 The keywords <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> are used in a single declaration.  
  
 The following sample generates C2042:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 Possible resolution:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
