# Compiler Error C3869
gcnew constraint is missing empty parameter list '()'  
  
 The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> special constraint was specified without the empty parameter list. See [Constraints](../vs140/constraints-on-generic-type-parameters--c---cli-.md) for more information.  
  
## Example  
 The following sample generates C3869.  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
