# is_nothrow_default_constructible Class
Tests whether type has a non-throwing default constructor.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 The type to query.  
  
## Remarks  
 An instance of the type predicate holds true if the type <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> has a nothrow default constructor, otherwise it holds false. An instance of the type predicate is equivalent to <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.
