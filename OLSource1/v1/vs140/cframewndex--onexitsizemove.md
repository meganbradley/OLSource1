# CFrameWndEx::OnExitSizeMove
Called by the framework when the frame stops moving or resizing.  
  
## Syntax  
  
```  
LRESULT OnExitSizeMove(  
   WPARAM wp,  
   LPARAM lp  
);  
```  
  
#### Parameters  
 [in] `wp`  
 This parameter is not used.  
  
 [in] `lp`  
 This parameter is not used.  
  
## Return Value  
 Always returns 0.

## Requirements  
 **Header:** afxframewndex.h
