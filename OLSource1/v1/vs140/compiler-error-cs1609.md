# Compiler Error CS1609
Modifiers cannot be placed on event accessor declarations  
  
 Modifiers can only be placed on event declarations and not on the event accessor declarations. For more information, see [Using Properties (C# Programming Guide)](../vs140/using-properties--csharp-programming-guide-.md).
