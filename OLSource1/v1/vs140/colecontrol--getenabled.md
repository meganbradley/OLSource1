# COleControl::GetEnabled
Implements the Get function of your control's stock Enabled property.  
  
## Syntax  
  
```  
  
BOOL GetEnabled( );  
```  
  
## Return Value  
 Nonzero if the control is enabled; otherwise 0.  
  
## Requirements  
 **Header:** afxctl.h
