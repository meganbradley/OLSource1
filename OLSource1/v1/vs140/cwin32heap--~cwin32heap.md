# CWin32Heap::~CWin32Heap
The destructor.  
  
## Syntax  
  
```  
  
~CWin32Heap( ) throw( );  
  
```  
  
## Remarks  
 Destroys the heap handle if the `CWin32Heap` object has ownership of the heap.  
  
## Requirements  
 **Header:** atlmem.h
