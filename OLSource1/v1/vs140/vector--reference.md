# vector::reference
A type that provides a reference to an element stored in a vector.  
  
## Syntax  
  
```  
typedef typename Allocator::reference reference;  
```  
  
## Example  
 See [at](../vs140/vector--at.md) for an example of how to use **reference** in the vector class.  
  
## Requirements  
 **Header:** \<vector>  
  
 **Namespace:** std
