# Compiler Error CS0637
The FieldOffset attribute is not allowed on static or const fields  
  
 The [FieldOffset](frlrfsystemruntimeinteropservicesfieldoffsetattributeclasstopic) attribute cannot be used on fields marked with [static](../vs140/static--csharp-reference-.md) or [const](../vs140/const--csharp-reference-.md).
