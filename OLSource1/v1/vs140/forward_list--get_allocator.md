# forward_list::get_allocator
Returns a copy of the allocator object used to construct a forward list.  
  
## Syntax  
  
```  
allocator_type get_allocator() const;  
```  
  
## Return Value  
 The stored [allocator](../vs140/allocator-class.md) object.  
  
## Requirements  
 **Header:** <forward_list>  
  
 **Namespace:** std
