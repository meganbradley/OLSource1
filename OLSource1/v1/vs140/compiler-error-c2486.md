# Compiler Error C2486
'__LOCAL_SIZE' only allowed in function with the 'naked' attribute  
  
 In inline assembly functions, the name <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> is reserved for functions declared with the [naked](../vs140/naked--c---.md) attribute.  
  
 The following sample generates C2486:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
