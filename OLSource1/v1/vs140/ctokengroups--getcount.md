# CTokenGroups::GetCount
Returns the number of <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> objects contained in <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.

<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
