# CMFCColorBar::GetVertMargin
Retrieves the vertical margin, which is the space between the top or bottom color cell and the client area boundary.  
  
## Syntax  
  
```  
int GetVertMargin() const;  
```  
  
## Return Value  
 The vertical margin.  
  
## Requirements  
 **Header:** afxcolorbar.h
