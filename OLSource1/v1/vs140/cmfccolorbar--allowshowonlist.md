# CMFCColorBar::AllowShowOnList
Indicates whether the color bar control object can appear in a toolbar list during the customization process.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Always <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
## Remarks  
 By default, this method always returns <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, which means the framework can display the color bar control during the customization process. Override this method to implement a different behavior.  
  
## Requirements  
 **Header:** afxcolorbar.h
