# Operators.sqrt&lt;^T,^U&gt; Function (F#)

 **Namespace/Module Path:** Microsoft.FSharp.Core.Operators  
  
 **Assembly:** FSharp.Core (in FSharp.Core.dll)  
  
## Syntax  
  
```  
// Signature:  
sqrt : ^T -> ^U (requires ^T with static member Sqrt)  
  
// Usage:  
sqrt value  
```  
  
#### Parameters  
 `value`  
 Type: `^T`

## Return Value  
 The square root of the input.
