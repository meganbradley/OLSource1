# CTokenPrivileges::GetCount
Returns the number of privilege entries in the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Returns the number of privileges contained in the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> object.
