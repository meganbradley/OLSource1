# Handles clause requires a WithEvents variable defined in the containing type or one of its base types
You did not supply a <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> variable in your <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> clause. The <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> keyword at the end of a procedure declaration causes it to handle events raised by an object variable declared using the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> keyword.  
  
 **Error ID:** BC30506
