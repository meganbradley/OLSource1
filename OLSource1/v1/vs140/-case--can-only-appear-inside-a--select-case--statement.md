# &#39;Case&#39; can only appear inside a &#39;Select Case&#39; statement
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement occurs outside a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> block. A <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement can be used only between a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> statement and its corresponding <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> statement.  
  
 **Error ID:** BC30072  
  
### To correct this error  
  
-   Remove the <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> statement or move it to within a <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> block.
