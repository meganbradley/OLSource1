# CCommandLineInfo::m_bRunAutomated
Indicates that the **/Automation** flag was found on the command line.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 If **TRUE**, this means start up as an OLE automation server.  
  
## Requirements  
 **Header:** afxwin.h
