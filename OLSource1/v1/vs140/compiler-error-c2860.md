# Compiler Error C2860
'void' cannot be an argument type, except for '(void)'  
  
 Type <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> cannot be used as an argument type with other arguments.  
  
 The following sample generates C2860:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
