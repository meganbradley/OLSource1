# trunc Function (fast_math)
Truncates the argument to the integer component  
  
## Syntax  
  
```  
inline float trunc(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the integer component of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
