# Compiler Error C2391
'identifier' : 'friend' cannot be used during type definition  
  
 The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> declaration includes a complete class declaration. A <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> declaration can specify a member function or an elaborated type specifier, but not a complete class declaration.
