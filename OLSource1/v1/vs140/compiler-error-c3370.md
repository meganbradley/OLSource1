# Compiler Error C3370
'idl_module name': idl_module not yet defined  
  
 Before you can use [idl_module](../vs140/idl_module.md) to specify an entry point in a DLL, you must first use <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> to specify the DLL name.  
  
 The following sample generates C3370:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
