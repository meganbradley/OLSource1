# Compiler Error CS0073
An add or remove accessor must have a body  
  
 An **add** or **remove** keyword in an [event](../vs140/event--csharp-reference-.md) definition must have a body. For more information, see [Events (C# Programmers Reference)](../vs140/events--csharp-programming-guide-.md).
