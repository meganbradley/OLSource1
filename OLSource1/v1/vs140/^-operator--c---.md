# ^ operator (C++)
The <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> operator has two meanings in Visual C++, depending on its context. For more information see:  
  
-   [Bitwise Exclusive OR Operator: ^](../vs140/bitwise-exclusive-or-operator--^.md), for use in all flavors of C++.  
  
-   [Handle to Object Operator (^)](../vs140/handle-to-object-operator--^----c---component-extensions-.md), for use in [!INCLUDE[cppwrt_short](../vs140/includes/cppwrt_short_md.md)] and [!INCLUDE[cppcli](../vs140/includes/cppcli_md.md)]
