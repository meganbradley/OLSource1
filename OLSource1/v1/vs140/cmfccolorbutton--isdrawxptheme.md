# CMFCColorButton::IsDrawXPTheme
Indicates whether the current color button is displayed in the visual style of Windows XP.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if visual styles are supported and the current color button is displayed in the visual style of Windows XP; otherwise, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.

 **Header:** afxcolorbutton.h
