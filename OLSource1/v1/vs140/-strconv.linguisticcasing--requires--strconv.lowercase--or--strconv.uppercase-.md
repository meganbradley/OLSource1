# &#39;StrConv.LinguisticCasing&#39; requires &#39;StrConv.LowerCase&#39; or &#39;StrConv.UpperCase&#39;
You have attempted to use <CodeContentPlaceHolder>0\</CodeContentPlaceHolder>, which is only valid in combination with <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
### To correct this error  
  
1.  Use <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> in conjunction with <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>.
