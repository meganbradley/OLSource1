# -UseEnv (devenv.exe)
Starts [!INCLUDE[vsprvs](../vs140/includes/vsprvs_md.md)] and loads environmental variables into the **VC++ Directories** dialog box.  
  
## Syntax  
  
```  
Devenv /useenv  
```  
  
## Example  
 The following example starts [!INCLUDE[vsprvs](../vs140/includes/vsprvs_md.md)] and loads environment variables into the **VC++ Directories** dialog box.  
  
```  
Devenv.exe /useenv  
```
