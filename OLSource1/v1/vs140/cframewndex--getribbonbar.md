# CFrameWndEx::GetRibbonBar
Retrieves the ribbon bar control for the frame.  
  
## Syntax  
  
```  
CMFCRibbonBar* GetRibbonBar();  
```  
  
## Return Value  
 Pointer to the [CMFCRibbonBar](../vs140/cmfcribbonbar-class.md) for the frame.

## Requirements  
 **Header:** afxframewndex.h
