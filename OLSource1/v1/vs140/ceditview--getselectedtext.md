# CEditView::GetSelectedText
Call <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> to copy the selected text into a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> object, up to the end of the selection or the character preceding the first carriage-return character in the selection.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 A reference to the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> object that is to receive the selected text.  
  
## Requirements  
 **Header:** afxext.h
