# COMM
Creates a communal variable with the attributes specified in <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 Each <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> has the following form:  
  
 [[*langtype*]] [[**NEAR** &#124; **FAR**]] *label***:**<CodeContentPlaceHolder>3\</CodeContentPlaceHolder>[[**:***count*]]  
  
 The *label* is the name of the variable. The <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> can be any type specifier ([BYTE](../vs140/byte--masm-.md), [WORD](../vs140/word.md), and so on) or an integer specifying the number of bytes. The *count* specifies the number of data objects (one is the default).
