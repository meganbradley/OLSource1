# Compiler Error C2150
'identifier' : bit field must have type 'int', 'signed int', or 'unsigned int'  
  
 bit fields are required to be <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, or <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.  
  
 The following sample generates C2150:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
