# XML comment tag &#39;include&#39; must have a &#39;&lt;attribute&gt;&#39; attribute
XML comment tag 'include' must have a \<attribute> attribute. XML comment will be ignored.  
  
 A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> attribute, passed to the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>include<CodeContentPlaceHolder>3\</CodeContentPlaceHolder> tag, was missing or incomplete.  
  
 **Error ID:** BC42310  
  
### To correct this error  
  
-   Add or complete the required attribute to the \<include> tag.
