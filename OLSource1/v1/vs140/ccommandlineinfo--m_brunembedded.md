# CCommandLineInfo::m_bRunEmbedded
Indicates that the **/Embedding** flag was found on the command line.  
  
## Syntax  
  
```  
  
BOOL m_bRunEmbedded;  
  
```  
  
## Remarks  
 If **TRUE**, this means start up for editing an embedded OLE item.  
  
## Requirements  
 **Header:** afxwin.h
