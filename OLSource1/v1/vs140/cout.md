# cout
Specifies the `cout` global stream.  
  
## Syntax  
  
```  
extern ostream cout;  
```  
  
## Return Value  
 An [ostream](../vs140/ostream.md) object.  
  
## Remarks  
 The object controls insertions to the standard output as a byte stream.  
  
## Example  
 See [cerr](../vs140/cerr.md) for an example of using `cout`.  
  
## Requirements  
 **Header:** \<iostream>  
  
 **Namespace:** std
