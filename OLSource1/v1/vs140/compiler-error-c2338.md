# Compiler Error C2338
'identifier' Atl Attribute Provider  
  
 The attribute references an undeclared identified. You may see this error associated with [ATL Provider Error ATL2010](assetId:///27a25924-b52b-4971-80d3-c3b3e9ffbe41).
