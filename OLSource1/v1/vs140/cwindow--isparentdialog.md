# CWindow::IsParentDialog
Determines if the parent window of the control is a dialog window.  
  
## Syntax  
  
```  
  
BOOL IsParentDialog( ) throw( );  
  
```  
  
## Return Value  
 Returns TRUE if the parent window is a dialog, FALSE otherwise.  
  
## Requirements  
 **Header:** atlwin.h
