# Compiler Error CS0594
Floating-point constant is outside the range of type 'type'  
  
 A value was assigned to a floating-point variable that is too large for the variables of this data type. See [Integral Types Table](../vs140/integral-types-table--csharp-reference-.md) for information about the range of values allowed in data types.
