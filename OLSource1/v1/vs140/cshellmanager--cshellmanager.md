# CShellManager::CShellManager
Constructs a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 In most cases, you do not have to create a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> directly. By default, the framework creates one for you. To get a pointer to the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, call [CWinAppEx::GetShellManager](../vs140/cwinappex--getshellmanager.md). If you do create a <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> manually, you must initialize it with the method [CWinAppEx::InitShellManager](../vs140/cwinappex--initshellmanager.md).  
  
## Requirements  
 **Header:** afxshellmanager.h
