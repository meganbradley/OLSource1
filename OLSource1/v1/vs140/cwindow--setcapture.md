# CWindow::SetCapture
Sends all subsequent mouse input to the window.  
  
## Syntax  
  
```  
  
HWND SetCapture( ) throw();  
```  
  
## Remarks  
 See [SetCapture](http://msdn.microsoft.com/library/windows/desktop/ms646262) in the [!INCLUDE[winSDK](../vs140/includes/winsdk_md.md)].
