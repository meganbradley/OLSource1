# Compiler Error C2173
'function' : actual parameter is not a pointer : parameter number1, parameter list number2  
  
 Parameter <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> passed to parameter list <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> is not a pointer. The function expects a pointer.
