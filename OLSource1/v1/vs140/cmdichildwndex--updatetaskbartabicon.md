# CMDIChildWndEx::UpdateTaskbarTabIcon
Updates the Windows 7 taskbar tab icon.  
  
## Syntax  
  
```  
virtual void UpdateTaskbarTabIcon(  
   HICON hIcon  
);  
```  
  
#### Parameters  
 `hIcon`  
 A handle to an icon to display on the Windows 7 taskbar tab.
