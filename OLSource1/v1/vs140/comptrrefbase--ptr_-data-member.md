# ComPtrRefBase::ptr_ Data Member
Supports the WRL infrastructure and is not intended to be used directly from your code.  
  
## Syntax  
  
```  
T* ptr_;  
```  
  
## Remarks  
 Pointer to the type specified by the current template parameter.  
  
## Remarks
