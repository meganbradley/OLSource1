# Compiler Error CS0128
A local variable named 'variable' is already defined in this scope  
  
 The compiler detected declarations of two local variables with the same name. For more information, see [Class and Struct Methods (C# Programmer's Reference)](../vs140/methods--csharp-programming-guide-.md).
