# Compiler Error C2050
switch expression not integral  
  
 The <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> expression evaluates to a noninteger value. To resolve the error, use only integral values in switch statements.  
  
 The following sample generates C2050:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 Possible resolution:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
