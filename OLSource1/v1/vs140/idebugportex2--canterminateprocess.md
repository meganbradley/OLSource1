# IDebugPortEx2::CanTerminateProcess
Determines whether a process can be terminated.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 [in] An [IDebugProcess2](../vs140/idebugprocess2.md) object representing the process to be terminated.  
  
## Return Value  
 Returns <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> if the process can be terminated; otherwise, returns <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.
