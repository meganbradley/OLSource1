# partial_sort_copy (STL-CLR)
Copies elements from a source range into a destination range where the source elements are ordered by either less than or another specified binary predicate.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This function behaves the same as the STL function <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. For more information, see [partial_sort_copy](../vs140/partial_sort_copy.md).  
  
## Requirements  
 **Header:** \<cliext/algorithm>  
  
 **Namespace:** cliext
