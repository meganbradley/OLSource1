# Compiler Warning (level 3) C4995
'function': name was marked as #pragma deprecated  
  
 The compiler encountered a function that was marked with pragma [deprecated](../vs140/deprecated--c-c---.md). The function may no longer be supported in a future release. You can turn this warning off with the [warning](../vs140/warning.md) pragma (example below).
