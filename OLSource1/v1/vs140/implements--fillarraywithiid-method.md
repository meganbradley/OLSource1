# Implements::FillArrayWithIid Method
Inserts the interface ID specified by the current zeroth template parameter into the specified array element.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 A zero-based index that indicates the starting array element for this operation. When this operation completes, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> is incremented by 1.  
  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 An array of type IID.
