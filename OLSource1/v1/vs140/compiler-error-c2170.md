# Compiler Error C2170
'identifier' : not declared as a function, cannot be intrinsic  
  
### To fix by checking the following possible causes  
  
1.  Pragma <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> is used for an item other than a function.  
  
2.  Pragma <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> is used for a function with no intrinsic form.
