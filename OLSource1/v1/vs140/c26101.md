# C26101
warning C26101: Failing to use interlocked operation properly for variable \<var>.  
  
 Windows APIs offer a variety of interlocked operations. Annotation <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> specifies that a variable should only be accessed through an interlocked operation. Warning C26101 is issued when an access is not consistent with the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> annotation.  
  
## Example  
 The following example generates warning C26101 because there is a violation of the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> contract.  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
