# CD2DBitmapBrush::Get
Returns ID2D1BitmapBrush interface  
  
## Syntax  
  
```  
ID2D1BitmapBrush* Get();  
```  
  
## Return Value  
 Pointer to an ID2D1BitmapBrush interface or NULL if object is not initialized yet.
