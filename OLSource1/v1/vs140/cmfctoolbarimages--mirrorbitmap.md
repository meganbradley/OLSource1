# CMFCToolBarImages::MirrorBitmap
Replaces a bitmap with its horizontal mirror image.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in, out] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 A handle to bitmap to mirror.  
  
 [in] <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 Width of the image in pixels.  
  
## Return Value  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> if the image was successfully mirrored; otherwise <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
## Remarks  
 This function is used to support right-to-left writing systems.

 **Header:** afxtoolbarimages.h
