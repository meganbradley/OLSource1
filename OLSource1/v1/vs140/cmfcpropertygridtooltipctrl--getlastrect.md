# CMFCPropertyGridToolTipCtrl::GetLastRect
Returns the coordinates of the last position of the tooltip control.  
  
## Syntax  
  
```  
void GetLastRect(  
   CRect& rect  
) const;  
```  
  
#### Parameters  
 [out] `rect`  
 Contains the last position of the tooltip control.  
  
## Requirements  
 **Header:** afxpropertygridtooltipctrl.h
