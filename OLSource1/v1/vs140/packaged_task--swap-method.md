# packaged_task::swap Method
Exchanges the associated asynchronous state with that of a specified object.  
  
## Syntax  
  
```cpp  
void swap(packaged_task& Right) _NOEXCEPT;  
```  
  
#### Parameters  
 `Right`  
 A `packaged_task` object.
