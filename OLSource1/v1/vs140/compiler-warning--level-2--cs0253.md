# Compiler Warning (level 2) CS0253
Possible unintended reference comparison; to get a value comparison, cast the right hand side to type 'type'  
  
 The compiler is doing a reference comparison. If you want to compare the value of strings, cast the right side of the expression to <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
 The following sample generates CS0253:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
