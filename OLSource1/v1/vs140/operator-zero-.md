# operator ZERO?
Status of zero flag. Used only within [.IF](../vs140/.if.md), [.WHILE](../vs140/.while.md), or [.REPEAT](../vs140/.repeat.md) blocks and evaluated at run time, not at assembly time.  
  
## Syntax  
  
```  
  
ZERO?  
  
```
