# Compiler Error CS1518
Expected class, delegate, enum, interface, or struct  
  
 A declaration was found that is not supported in a [namespace](../vs140/namespace--csharp-reference-.md). Inside a namespace, the compiler accepts only classes, structs, enums, interfaces, namespaces, and delegates.
