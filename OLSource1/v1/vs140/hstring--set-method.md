# HString::Set Method
Sets the value of the current HString object to the specified wide-character string or HString parameter.

<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>

 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 The maximum length of the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> parameter that is assigned to the current HString object.  
  
 <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>
