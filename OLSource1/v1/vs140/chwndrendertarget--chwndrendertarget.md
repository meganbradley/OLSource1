# CHwndRenderTarget::CHwndRenderTarget
Constructs a CHwndRenderTarget object from HWND.  
  
## Syntax  
  
```  
CHwndRenderTarget(  
   HWND hwnd = NULL  
);  
```  
  
#### Parameters  
 `hwnd`  
 The HWND associated with this render target  
  
## Requirements  
 **Header:** afxrendertarget.h
