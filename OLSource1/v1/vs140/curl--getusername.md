# CUrl::GetUserName
Call this method to get the user name from the URL.  
  
## Syntax  
  
```  
  
inline LPCTSTR GetUserName( ) const throw( );  
  
```  
  
## Return Value  
 Returns the user name.  
  
## Requirements  
 **Header:** atlutil.h
