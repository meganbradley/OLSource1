# &#39;End Using&#39; must be preceded by a matching &#39;Using&#39;
An <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement appears with no matching <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> declaration preceding it.  
  
 **Error ID:** BC36007  
  
### To correct this error  
  
-   Remove the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement if it is redundant.  
  
-   Supply the missing [Using Statement (Visual Basic)](../vs140/using-statement--visual-basic-.md) if one is missing.  
  
-   Move the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement to the appropriate place in the code.
