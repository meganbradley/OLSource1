# CDHtmlDialog::EnableModeless
Enables modeless dialog boxes.

<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 See <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> in [IDocHostUIHandler::EnableModeless](https://msdn.microsoft.com/en-us/library/aa753253.aspx) in the [!INCLUDE[winSDK](../vs140/includes/winsdk_md.md)].
