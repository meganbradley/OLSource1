# CToolBarCtrl::DeleteButton
Deletes a button from the toolbar control.  
  
## Syntax  
  
```  
  
      BOOL DeleteButton(  
   int nIndex   
);  
```  
  
#### Parameters  
 `nIndex`  
 Zero-based index of the button to delete.  
  
## Return Value  
 Nonzero if successful; otherwise zero.  
  
## Remarks  
  
## Requirements  
 **Header:** afxcmn.h
