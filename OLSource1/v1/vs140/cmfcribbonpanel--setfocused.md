# CMFCRibbonPanel::SetFocused
Sets focus to the specified Ribbon element.  
  
## Syntax  
  
```  
void SetFocused(  
   CMFCRibbonBaseElement* pNewFocus  
);  
```  
  
#### Parameters  
 `pNewFocus`  
 A pointer to a Ribbon element that receives focus.
