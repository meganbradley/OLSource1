# map::empty
Tests if a map is empty.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 **true** if the map is empty; **false** if the map is nonempty.  
  
## Example  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 **The map m1 is not empty.**  
**The map m2 is empty.**   
## Requirements  
 **Header:** \<map>  
  
 **Namespace:** std
