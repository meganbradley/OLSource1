# Compiler Error C2271
'operator' : new/delete cannot have formal list modifiers  
  
 The operator (<CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>) is declared with a memory-model specifier.  
  
 The following sample generates C2271:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
