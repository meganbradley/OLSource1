# Compiler Error C2894
templates cannot be declared to have 'C' linkage  
  
 This error can be caused by a template defined inside an <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> "C" block.  
  
 The following sample generates C2894:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 The following sample generates C2894:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
