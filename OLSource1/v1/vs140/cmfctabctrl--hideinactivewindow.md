# CMFCTabCtrl::HideInactiveWindow
Specifies whether the framework displays inactive tab control windows.

 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> not to display an inactive window; <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> to display an inactive window. The default value is <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.
