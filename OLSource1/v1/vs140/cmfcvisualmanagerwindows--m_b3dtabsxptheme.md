# CMFCVisualManagerWindows::m_b3DTabsXPTheme
A Boolean parameter that determines whether the visual manager displays 3D tabs.  
  
## Syntax  
  
```  
AFX_IMPORT_DATA static BOOL m_b3DTabsXPTheme;  
```  
  
## Requirements  
 **Header:** afxvisualmanagerwindows.h
