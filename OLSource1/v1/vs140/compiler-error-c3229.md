# Compiler Error C3229
'type' : indirections on a generic type parameter are not allowed  
  
 You cannot use generic parameters with <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, or <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
## Example  
 The following sample generates C3229.  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Example  
 The following sample generates C3229.  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
