# Compiler Warning (level 1) CS1694
Invalid filename specified for preprocessor directive. Filename is too long or not a valid filename.  
  
 This warning occurs when using the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> preprocessor directive. The file name specified is longer than 256 characters. To resolve this warning, use a shorter file name.  
  
## Example  
 The following sample generates CS1694.  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
