# CPathT::IsDirectory
Call this method to check whether the path is a valid directory.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Returns a non-zero value (16) if the path is a directory, <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> otherwise.  
  
## Remarks  
 For more information, see [PathIsDirectory](http://msdn.microsoft.com/library/windows/desktop/bb773621).  
  
## Requirements  
 **Header:** atlpath.h
