# _com_error::HRESULTToWCode
**Microsoft Specific**  
  
 Maps 32-bit <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> to 16-bit <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 The 32-bit <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> to be mapped to 16-bit <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>.  
  
## Return Value  
 16-bit <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> mapped from the 32-bit <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>.  
  
## Remarks  
 See [_com_error::WCode](../vs140/_com_error--wcode.md) for more information.  
  
 **END Microsoft Specific**
