# CMFCToolBar::EnableLargeIcons
Enables or disables large icons on toolbar buttons.  
  
## Syntax  
  
```  
void EnableLargeIcons(  
   BOOL bEnable   
);  
```  
  
#### Parameters  
 [in] `bEnable`  
 `TRUE` to enable large icons, `FALSE` to disable large icons.  
  
## Remarks  
 By default, large icons are enabled.  
  
## Requirements  
 **Header:** afxtoolbar.h
