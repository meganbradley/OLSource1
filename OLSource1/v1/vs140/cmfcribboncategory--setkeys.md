# CMFCRibbonCategory::SetKeys
Assigns a keytip to the ribbon category.  
  
## Syntax  
  
```  
void SetKeys(  
   LPCTSTR lpszKeys   
);  
```  
  
#### Parameters  
 [in] `lpszKeys`  
 The keytip text.  
  
## Remarks  
 Keytips are displayed when the user presses the Alt key or the F10 key.  
  
## Requirements  
 **Header:** afxribboncategory.h
