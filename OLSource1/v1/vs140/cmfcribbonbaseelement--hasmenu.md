# CMFCRibbonBaseElement::HasMenu
Indicates whether the ribbon element has a menu.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Always returns <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
## Remarks  
 By default this method always returns <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>. Override this method in a derived class to indicate whether the ribbon element has a menu.

 **Header:** afxbaseribbonelement.h
