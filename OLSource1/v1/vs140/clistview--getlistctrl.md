# CListView::GetListCtrl
Call this member function to get a reference to the list control associated with the view.  
  
## Syntax  
  
```  
  
CListCtrl& GetListCtrl( ) const;  
  
```  
  
## Return Value  
 A reference to the list control associated with the view.
