# unordered_set::iterator
A type that provides a constant [forward iterator](../vs140/forward_iterator_tag-struct.md) that can read elements in an unordered_set.  
  
## Syntax  
  
```  
typedef implementation-defined iterator;  
```  
  
## Example  
 See the example for [begin](../vs140/set--begin.md) for an example of how to declare and use an **iterator**.  
  
## Requirements  
 **Header:** <unordered_set>  
  
 **Namespace:** std
