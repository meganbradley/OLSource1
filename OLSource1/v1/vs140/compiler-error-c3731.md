# Compiler Error C3731
incompatible event 'function1' and handler 'function2'; event source and event handler must be the same type  
  
 The event source and event receiver must have the same type (for example <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> vs. <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> types). To fix this error, make the types of the event source and the event handler match.
