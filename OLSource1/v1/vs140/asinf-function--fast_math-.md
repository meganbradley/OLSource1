# asinf Function (fast_math)
Calculates the arcsine of the argument  
  
## Syntax  
  
```  
inline float asinf(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the arcsine value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
