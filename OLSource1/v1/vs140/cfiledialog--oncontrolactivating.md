# CFileDialog::OnControlActivating
Called when the control is activated.  
  
## Syntax  
  
```  
virtual void OnControlActivating(  
   DWORD dwIDCtl  
);  
```  
  
#### Parameters  
 `dwIDCtl`  
 The ID of the control.
