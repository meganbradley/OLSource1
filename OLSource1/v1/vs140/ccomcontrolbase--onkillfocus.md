# CComControlBase::OnKillFocus
Checks that the control is in-place active and has a valid control site, then informs the container that the control has lost focus.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>

 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 Reserved.  
  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 Reserved.  
  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 Reserved.  
  
 <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>  
 Flag that indicates whether the window message was successfully handled. The default is <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>.  
  
## Return Value  
 Always returns 1.
