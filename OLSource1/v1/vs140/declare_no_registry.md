# DECLARE_NO_REGISTRY
Use `DECLARE_NO_REGISTRY` if you want to avoid any default ATL registration for the class in which this macro appears.  
  
## Syntax  
  
```  
  
DECLARE_NO_REGISTRY( )  
  
```  
  
## Requirements  
 **Header:** atlcom.h
