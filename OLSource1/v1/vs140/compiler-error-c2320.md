# Compiler Error C2320
expected ':' to follow access specifier 'specifier'  
  
 The keyword <CodeContentPlaceHolder>0\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> must be followed by a colon.
