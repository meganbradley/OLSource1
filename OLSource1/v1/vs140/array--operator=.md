# array::operator=
Replaces the controlled sequence.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 right  
 Container to copy.  
  
## Remarks  
 The member operator assigns each element of <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> to the corresponding element of the controlled sequence, then returns <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>. You use it to replace the controlled sequence with a copy of the controlled sequence in <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.
