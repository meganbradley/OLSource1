# CSimpleArrayEqualHelper::IsEqual
Tests two `CSimpleArray` object elements for equality.  
  
## Syntax  
  
```  
  
      static bool IsEqual(  
   const T& t1,  
   const T& t2   
);  
```  
  
#### Parameters  
 *t1*  
 An object of type T.  
  
 *t2*  
 An object of type T.  
  
## Return Value  
 Returns true if the elements are equal, false otherwise.  
  
## Requirements  
 **Header:** atlsimpcoll.h
