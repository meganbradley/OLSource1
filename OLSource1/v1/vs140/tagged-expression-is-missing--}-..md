# Tagged expression is missing &#39;}&#39;.
This error generally occurs when an opening brace (<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>) has been specified but the closing brace (<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>) has been omitted in a find string.  
  
### To correct this error  
  
1.  To search for the literal character <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, use <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.  
  
2.  To tag an expression, enter the missing <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.
