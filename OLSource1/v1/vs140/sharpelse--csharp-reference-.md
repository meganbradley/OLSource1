# #else (C# Reference)
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder> lets you create a compound conditional directive, so that, if none of the expressions in the preceding [#if](../vs140/sharpif--csharp-reference-.md) or (optional) [#elif](../vs140/sharpelif--csharp-reference-.md) directives to <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, the compiler will evaluate all code between <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> and the subsequent <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.  
  
## Remarks  
 [#endif](../vs140/sharpendif--csharp-reference-.md) must be the next preprocessor directive after <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>. See [#if](../vs140/sharpif--csharp-reference-.md) for an example of how to use <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>.
