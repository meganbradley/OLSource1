# &#39;For&#39; must end with a matching &#39;Next&#39;
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement occurs without a corresponding <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement. A <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement must be used to end the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> loop.  
  
 **Error ID:** BC30084  
  
### To correct this error  
  
-   If this <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> loop is part of a set of nested loops, make sure each loop is properly terminated.  
  
-   Add a <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> statement to the end of the <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> loop.
