# CDaoDatabase::m_pDAODatabase
Contains a pointer to the OLE interface for the DAO database object underlying the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> object.  
  
## Remarks  
 Use this pointer if you need to access the DAO interface directly.  
  
 For information about calling DAO directly, see [Technical Note 54](../vs140/tn054--calling-dao-directly-while-using-mfc-dao-classes.md).  
  
## Requirements  
 **Header:** afxdao.h
