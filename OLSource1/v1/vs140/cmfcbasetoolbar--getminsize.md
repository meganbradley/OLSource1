# CMFCBaseToolBar::GetMinSize
Returns the minimum size of a toolbar.  
  
## Syntax  
  
```  
virtual void GetMinSize(  
   CSize& size  
) const;  
```  
  
#### Parameters  
 [out] `size`  
 The minimum size of a toolbar.  
  
## Requirements  
 **Header:** afxbasetoolbar.h
