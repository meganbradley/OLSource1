# &#39;&lt;typename&gt;&#39; must be declared &#39;MustInherit&#39; because it contains methods declared &#39;MustOverride&#39;
Types that contain <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> members must be declared <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
 **Error ID:** BC31411  
  
### To correct this error  
  
-   Add the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> modifier to the type.
