# .NOLISTIF
Suppresses listing of conditional blocks whose condition evaluates to false (0).  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This is the default. Same as [.SFCOND](../vs140/.sfcond.md).
