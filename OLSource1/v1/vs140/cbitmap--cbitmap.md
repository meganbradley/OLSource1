# CBitmap::CBitmap
Constructs a `CBitmap` object.  
  
## Syntax  
  
```  
  
CBitmap( );  
```  
  
## Remarks  
 The resulting object must be initialized with one of the initialization member functions.  
  
## Requirements  
 **Header:** afxwin.h
