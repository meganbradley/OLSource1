# wstring_convert::wide_string
A type that represents a wide string.  
  
## Syntax  
  
```  
typedef std::basic_string<_Elem> wide_string;  
```  
  
## Remarks  
 The type is a synonym for `std::basic_string<_Elem>`.  
  
## Requirements  
 **Header:** \<locale>  
  
 **Namespace:** std
