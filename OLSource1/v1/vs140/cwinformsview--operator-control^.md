# CWinFormsView::operator Control^
Casts a type as a pointer to a Windows Forms control.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This operator allows you to pass a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> view to functions that accept a pointer to a Windows Forms control of type \<xref:System.Windows.Forms.Control*?displayProperty=fullName>.

 See [CWinFormsView::GetControl](../vs140/cwinformsview--getcontrol.md).
