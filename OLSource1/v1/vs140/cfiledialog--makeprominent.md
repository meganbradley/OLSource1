# CFileDialog::MakeProminent
Places a control in the dialog so that it stands out compared to other controls.  
  
## Syntax  
  
```  
HRESULT MakeProminent(  
   DWORD dwIDCtl  
);  
```  
  
#### Parameters  
 `dwIDCtl`  
 The ID of the control.
