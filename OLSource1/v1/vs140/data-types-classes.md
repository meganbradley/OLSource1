# Data Types Classes
The following classes wrap C++ data types:  
  
-   [CComBSTR](../vs140/ccombstr-class.md) Wraps the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> data type.  
  
-   [CComVariant](../vs140/ccomvariant-class.md) Wraps the **VARIANT** data type.  
  
-   [CComCurrency](../vs140/ccomcurrency-class.md) Includes methods and operators for creating and managing a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object.
