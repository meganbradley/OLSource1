# MMWORD
Used for 64-bit multimedia operands with MMX and SSE (XMM) instructions.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> is a type.  Prior to MMWORD being added to MASM, equivalent functionality could have been achieved with:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 While both instructions work on 64-bit operands, <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> is the type for 64-bit unsigned integers and <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> is the type for a 64-bit multimedia value.  
  
 <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> is intended to represent the same type as [__m64](../vs140/__m64.md).
