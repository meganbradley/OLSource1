# COleControl::GetForeColor
Implements the Get function of the stock ForeColor property.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 The return value specifies the current foreground color as a **OLE_COLOR** value, if successful. This value can be translated to a [COLORREF](http://msdn.microsoft.com/library/windows/desktop/dd183449) value with a call to <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
## Requirements  
 **Header:** afxctl.h
