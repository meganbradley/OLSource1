# &#39;Option Strict&#39; can be followed only by &#39;On&#39; or &#39;Off&#39;
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statements must specify either <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>. All other values are invalid.  
  
 **Error ID:** BC30620  
  
### To correct this error  
  
-   Specify either <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> for the <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> statement.
