# CD2DBitmapBrush::GetBitmap
Gets the bitmap source that this brush uses to paint  
  
## Syntax  
  
```  
CD2DBitmap* GetBitmap();  
```  
  
## Return Value  
 Pointer to an CD2DBitmap object or NULL if object is not initialized yet.  
  
## Requirements  
 **Header:** afxrendertarget.h
