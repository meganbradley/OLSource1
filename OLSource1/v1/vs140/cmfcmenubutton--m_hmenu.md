# CMFCMenuButton::m_hMenu
The handle to the attached menu.  
  
## Syntax  
  
```  
HMENU m_hMenu;  
```  
  
## Remarks  
 The framework displays the menu indicated by this member variable when the user clicks the menu button.  
  
## Requirements  
 **Header:** afxmenubutton.h
