# expm1 Function
Calculates the base-e exponential of the argument, minus 1  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 The exponential term *n* of the mathematical expression <CodeContentPlaceHolder>2\</CodeContentPlaceHolder><sup>n\</sup>, where <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> is the base of the natural logarithm.  
  
## Return Value  
 Returns the base-e exponential of the argument, minus 1  
  
## Remarks  
 The result of the function call <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> is the same as the result of the expression <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> if *n* is a single-precision floating-point number, or the expression <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> if *n* is a double-precision floating-point number.
