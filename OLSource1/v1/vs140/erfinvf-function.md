# erfinvf Function
Computes the inverse error function of _X  
  
## Syntax  
  
```  
inline float erfinvf(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the inverse error function of _X  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
