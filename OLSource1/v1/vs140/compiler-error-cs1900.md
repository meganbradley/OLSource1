# Compiler Error CS1900
Warning level must be in the range 0-4  
  
 The [/warn](../vs140/-warn--csharp-compiler-options-.md) compiler option can only take one of five possible values (0, 1, 2, 3, or 4). Any other value passed to **/warn** will result in CS1900.
