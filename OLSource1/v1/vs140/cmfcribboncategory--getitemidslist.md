# CMFCRibbonCategory::GetItemIDsList
Retrieves the command IDs for the ribbon elements that are contained in the ribbon category.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [out] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 The list of command IDs for the ribbon elements in the ribbon category.  
  
 [in] <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> to exclude ribbon elements displayed on the ribbon panels in the ribbon category; <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> to include all ribbon elements in the ribbon category.
