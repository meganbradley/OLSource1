# CMFCTasksPane::IsWrapLabelsEnabled
Indicates whether the task pane wraps words in labels.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if words in labels are wrapped; otherwise, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.
