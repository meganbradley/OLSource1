# Resource Compiler Warning RC4006
\#undef expected an identifier  
  
 The name of the identifier whose definition was to be removed was not given with the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> directive. The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> directive was ignored.
