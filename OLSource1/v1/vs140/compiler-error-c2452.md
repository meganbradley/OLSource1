# Compiler Error C2452
'type' : invalid source type for safe_cast  
  
 The source type for [safe_cast](../vs140/safe_cast--c---component-extensions-.md) was not valid.  For example, all types in a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> operation must be CLR types.  
  
 The following sample generates C2452:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
