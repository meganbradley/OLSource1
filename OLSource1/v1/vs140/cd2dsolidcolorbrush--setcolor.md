# CD2DSolidColorBrush::SetColor
Specifies the color of this solid color brush  
  
## Syntax  
  
```  
void SetColor(  
   D2D1_COLOR_F color  
);  
```  
  
#### Parameters  
 `color`  
 The color of this solid color brush  
  
## Requirements  
 **Header:** afxrendertarget.h
