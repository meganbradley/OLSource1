# &#39;Compare&#39; expected
An <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement specifies <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, but not <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.  
  
 **Error ID:** BC30208  
  
### To correct this error  
  
-   Add the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> keyword to the <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> statement; for example, <CodeContentPlaceHolder>6\</CodeContentPlaceHolder>.
