# CPropExchange::IsAsynchronous
Determines if property exchanges are done asynchronously.  
  
## Syntax  
  
```  
  
BOOL IsAsynchronous( );  
  
```  
  
## Return Value  
 Returns TRUE if properties are exchanged asynchronously, otherwise FALSE.  
  
## Requirements  
 **Header:** afxctl.h
