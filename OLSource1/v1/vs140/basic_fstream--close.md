# basic_fstream::close
Closes a file.  
  
## Syntax  
  
```  
  
void close( );  
  
```  
  
## Remarks  
 The member function calls [rdbuf](../vs140/basic_fstream--rdbuf.md) **->** [close](../vs140/basic_filebuf--close.md).  
  
## Example  
 See [basic_filebuf::close](../vs140/basic_filebuf--close.md) for an example of how to use **close**.  
  
## Requirements  
 **Header:** <fstream\>  
  
 **Namespace:** std
