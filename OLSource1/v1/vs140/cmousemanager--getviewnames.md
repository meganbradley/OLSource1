# CMouseManager::GetViewNames
Retrieves a list of all the registered view names.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [out] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>

## Remarks  
 This method fills the parameter <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> with the names of all the views registered by using [CMouseManager::AddView](../vs140/cmousemanager--addview.md).
