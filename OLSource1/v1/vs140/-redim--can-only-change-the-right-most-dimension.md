# &#39;ReDim&#39; can only change the right-most dimension
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement attempted to use the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> keyword to change a dimension of an array that is not the last dimension. When using <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, you can resize only the last dimension of an array. For all other dimensions, you must specify the same size as for the existing array.  
  
### To correct this error  
  
-   Remove the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> keyword.
