# HStringReference::Operator= Operator
Moves the value of another HStringReference object to the current HStringReference object.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 An existing HStringReference object.  
  
## Remarks  
 The value of the existing <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> object is copied to the current HStringReference object, and then the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> object is destroyed.  
  
## Requirements  
 **Header:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
