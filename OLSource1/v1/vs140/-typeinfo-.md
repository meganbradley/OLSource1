# &lt;typeinfo&gt;
Include the standard header `<typeinfo>` to define several types associated with the type-identification operator [typeid](../vs140/typeid-operator.md), which yields information about both static and dynamic types.  
  
## Syntax  
  
```  
#include <typeinfo>  
  
```  
  
## Remarks  
 For information on classes defined in <typeinfo\>, see the following topics:  
  
-   [bad_cast Exception](../vs140/bad_cast-exception.md)  
  
-   [bad_typeid Exception](../vs140/bad_typeid-exception.md)  
  
-   [type_info Class](../vs140/type_info-class.md)
