# Compiler Error CS0631
ref and out are not valid in this context  
  
 The declaration for an [indexer](../vs140/indexers--csharp-programming-guide-.md) cannot include the use of [ref](../vs140/ref--csharp-reference-.md) or [out](../vs140/out--csharp-reference-.md) parameters.
