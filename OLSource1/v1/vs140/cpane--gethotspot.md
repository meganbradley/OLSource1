# CPane::GetHotSpot
Returns the hot spot that is stored in an underlying <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
  
## Remarks  
 The <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> class contains a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> object, <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>, that is responsible for drawing the rectangle that appears when the user moves a pane in the standard docking mode. The hot spot is used to draw the rectangle relative to the current mouse position as the user moves the pane.
