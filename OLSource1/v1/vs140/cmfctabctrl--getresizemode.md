# CMFCTabCtrl::GetResizeMode
Retrieves a value that specifies how the current tab control can be resized.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 One of the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> enumeration values that specifies how the tab control can be resized. For a list of possible values, see the Remarks section of the [CMFCTabCtrl::SetResizeMode](../vs140/cmfctabctrl--setresizemode.md) method.  
  
## Requirements  
 **Header:** afxtabctrl.h
