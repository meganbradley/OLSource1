# CAtlModuleT::RegisterAppId
Adds the EXE to the registry.  
  
## Syntax  
  
```  
  
HRESULT RegisterAppId( ) throw( );  
  
```  
  
## Return Value  
 Returns S_OK on success, or an error HRESULT on failure.  
  
## Requirements  
 **Header:** atlbase.h
