# CFileDialog::EnableOpenDropDown
Enables a drop-down list on the Open or Save button in the dialog.  
  
## Syntax  
  
```  
HRESULT EnableOpenDropDown(  
   DWORD dwIDCtl  
);  
```  
  
#### Parameters  
 `dwIDCtl`  
 The ID of the drop-down list.
