# CToolBarCtrl::SetToolTips
Associates a tool tip control with a toolbar control.  
  
## Syntax  
  
```  
  
      void SetToolTips(  
   CToolTipCtrl* pTip   
);  
```  
  
#### Parameters  
 *pTip*  
 Pointer to the [CToolTipCtrl](../vs140/ctooltipctrl-class.md) object.  
  
## Requirements  
 **Header:** afxcmn.h
