# CMDIFrameWndEx::GetTearOffBars
Returns a list of tear-off menus.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 A reference to a [CObList Class](../vs140/coblist-class.md) object that contains a collection of pointers to <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>-derived objects that are in a tear-off state.  
  
## Remarks  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> maintains a collection of tear-off menus. Use this method to retrieve a reference to this list.
