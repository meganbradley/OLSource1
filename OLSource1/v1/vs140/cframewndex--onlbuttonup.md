# CFrameWndEx::OnLButtonUp
The framework calls this method when the user releases the left mouse button.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 Indicates whether the user pressed modifier keys. For possible values see the parameter <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> in [WM_LBUTTONUP Notification](http://msdn.microsoft.com/library/windows/desktop/ms645608).  
  
 [in] <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 Specifies the x and y coordinates of the pointer, relative to the upper-left corner of the window.
