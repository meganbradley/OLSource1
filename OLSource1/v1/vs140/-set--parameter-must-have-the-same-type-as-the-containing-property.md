# &#39;Set&#39; parameter must have the same type as the containing property
The parameter for the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> property procedure has a different type from the property that it belongs to.  
  
 **Error ID:** BC31064  
  
### To correct this error  
  
-   Change the data type of the parameter to <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> so that it matches the data type for the property. For example:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
