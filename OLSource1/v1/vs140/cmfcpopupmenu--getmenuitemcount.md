# CMFCPopupMenu::GetMenuItemCount
Returns the number of items in a pop-up menu.  
  
## Syntax  
  
```  
int GetMenuItemCount() const;  
```  
  
## Return Value  
 The number of items in the menu.  
  
## Requirements  
 **Header:** afxpopupmenu.h
