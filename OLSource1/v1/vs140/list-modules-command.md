# List Modules Command
Lists the modules for the current process.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 /Address:<CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 Optional. Specifies whether to show the memory addresses of the modules. Default value is <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.  
  
 /Name:<CodeContentPlaceHolder>4\</CodeContentPlaceHolder>  
 Optional. Specifies whether to show the names of the modules. Default value is <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>.  
  
 /Order:<CodeContentPlaceHolder>6\</CodeContentPlaceHolder>  
 Optional. Specifies whether to show the order of the modules. Default value is <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>.  
  
 /Path:<CodeContentPlaceHolder>8\</CodeContentPlaceHolder>  
 Optional. Specifies whether to show the paths of the modules. Default value is <CodeContentPlaceHolder>9\</CodeContentPlaceHolder>.  
  
 /Process:<CodeContentPlaceHolder>10\</CodeContentPlaceHolder>  
 Optional. Specifies whether to show the processes of the modules. Default value is <CodeContentPlaceHolder>11\</CodeContentPlaceHolder>.  
  
 /SymbolFile:<CodeContentPlaceHolder>12\</CodeContentPlaceHolder>  
 Optional. Specifies whether to show the symbol files of the modules. Default value is <CodeContentPlaceHolder>13\</CodeContentPlaceHolder>.  
  
 /SymbolStatus:<CodeContentPlaceHolder>14\</CodeContentPlaceHolder>  
 Optional. Specifies whether to show the symbol statuses of the modules. Default value is <CodeContentPlaceHolder>15\</CodeContentPlaceHolder>.  
  
 /Timestamp:<CodeContentPlaceHolder>16\</CodeContentPlaceHolder>  
 Optional. Specifies whether to show the timestamps of the modules. Default value is <CodeContentPlaceHolder>17\</CodeContentPlaceHolder>.  
  
 /Version:<CodeContentPlaceHolder>18\</CodeContentPlaceHolder>  
 Optional. Specifies whether to show the versions of the modules. Default value is <CodeContentPlaceHolder>19\</CodeContentPlaceHolder>.
