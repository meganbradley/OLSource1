# .CONST
When used with [.MODEL](../vs140/.model.md), starts a constant data segment (with segment name CONST).  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This segment has the read-only attribute.
