# Properties cannot be declared &#39;&lt;modifiername&gt;&#39;
There is an error in the syntax used in the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> declaration. This error generally indicates that a specified keyword, such as <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, cannot be used in a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> declaration.  
  
 **Error ID:** BC30639  
  
### To correct this error  
  
-   Check the syntax for the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> declaration.
