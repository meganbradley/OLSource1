# DHTML_EVENT_ONFOCUS
Handles (at the element level) the **onfocus** event. This is a nonbubbling event.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 An <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> holding the ID of the HTML element sourcing the event.  
  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 The handler function for the event.  
  
## Remarks  
 Use this macro to add an entry to the [DHTML event map](../vs140/begin_dhtml_event_map_inline.md) in your class.  
  
## Requirements  
 **Header:** afxdhtml.h
