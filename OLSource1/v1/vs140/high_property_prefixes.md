# high_property_prefixes
**C++ Specific**  
  
 Specifies alternate prefixes for three property methods.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 Prefix to be used for the **propget** methods.  
  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 Prefix to be used for the **propput** methods.  
  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 Prefix to be used for the **propputref** methods.  
  
## Remarks  
 By default, high-level error-handling **propget**, **propput**, and **propputref** methods are exposed by member functions named with prefixes **Get**, <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>, and **PutRef**, respectively.  
  
 **END C++ Specific**
