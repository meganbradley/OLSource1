# CPaneFrameWnd::GetVisiblePaneCount
Returns the number of visible panes that are contained in a mini-frame window.  
  
## Syntax  
  
```  
virtual int GetVisiblePaneCount() const;  
```  
  
## Return Value  
 The number of visible panes.
