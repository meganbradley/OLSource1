# event::timeout_infinite Constant
Value indicating that a wait should never time out.  
  
## Syntax  
  
```  
static const unsigned int timeout_infinite = COOPERATIVE_TIMEOUT_INFINITE;  
```  
  
## Requirements  
 **Header:** concrt.h  
  
 **Namespace:** concurrency
