# &#39;Inherits&#39; not valid in Modules
An <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement occurs inside a module. Modules cannot inherit from classes.  
  
 **Error ID:** BC30230  
  
### To correct this error  
  
-   Remove the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement, or retype the module as a class.
