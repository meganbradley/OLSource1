# CAcl::SetNull
Marks the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object as NULL.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 The <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> can be set to empty or to NULL: the two states are distinct.  
  
## Requirements  
 **Header:** atlsecurity.h
