# CHtmlView::OnDownloadComplete
This member function is called by the framework to indicate that a navigation operation finished, was halted, or failed.  
  
## Syntax  
  
```  
  
virtual void OnDownloadComplete( );  
  
```  
  
## Requirements  
 **Header:** afxhtml.h
