# Compiler Error C3540
'type': sizeof cannot be applied to a type that contains 'auto'  
  
 The [sizeof](../vs140/sizeof-operator.md) operator cannot be applied to the indicated type because it contains the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> specifier.  
  
## Example  
 The following example yields C3540.  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
