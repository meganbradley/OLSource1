# Fatal Error C1022
expected #endif  
  
 An <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, or <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> directive has no matching <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> directive. Be sure each <CodeContentPlaceHolder>6\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>, or <CodeContentPlaceHolder>8\</CodeContentPlaceHolder> has a matching <CodeContentPlaceHolder>9\</CodeContentPlaceHolder>.  
  
 The following sample generates C1022:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 Possible resolution:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
