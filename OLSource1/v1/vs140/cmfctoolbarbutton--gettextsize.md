# CMFCToolBarButton::GetTextSize
Retrieves the size of the button text.  
  
## Syntax  
  
```  
SIZE GetTextSize() const;  
```  
  
## Return Value  
 A `SIZE` object that contains the size, in pixels, of the button text.  
  
## Requirements  
 **Header:** afxtoolbarbutton.h
