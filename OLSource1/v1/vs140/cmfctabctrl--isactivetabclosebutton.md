# CMFCTabCtrl::IsActiveTabCloseButton
Indicates whether the Close button (**X**) is displayed on an active tab or on the upper-right corner of the tab area.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the Close button is displayed on the active tab; <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> if the Close button is displayed on the upper-right corner of the tab area.
