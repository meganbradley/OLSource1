# &#39;#Region&#39; statement must end with a matching &#39;#End Region&#39;
Use the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> directive to specify a block of code that you can expand or collapse when using the outlining feature of the [!INCLUDE[vsprvs](../vs140/includes/vsprvs_md.md)] Code Editor. The start and end of <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statements must be in the same code block.  
  
 **Error ID:** BC30681  
  
### To correct this error  
  
1.  Insert <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> at the appropriate place in the code after the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement.
