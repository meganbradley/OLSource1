# CMFCToolBarComboBoxButton::IsRibbonButton
Indicates whether the combo box button resides on a ribbon panel.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Always <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
## Remarks  
 By default, this method always returns <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, which means the combo box button is never displayed on a ribbon panel.  
  
## Requirements  
 **Header:** afxtoolbarcomboboxbutton.h
