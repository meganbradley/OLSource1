# -SUMMARY
```  
/SUMMARY  
```  
  
## Remarks  
 This option displays minimal information about sections, including total size. This option is the default if no other option is specified.  
  
 Only the [/HEADERS](../vs140/-headers.md) DUMPBIN option is available for use on files produced with the [/GL](../vs140/-gl--whole-program-optimization-.md) compiler option.
