# OperatorIntrinsics.PowInt16 Function (F#)
This is a library intrinsic. Calls to this function may be generated by uses of the generic [pown](../vs140/operators.pown-^t--function--fsharp-.md) operator on values of type `int16`.  
  
 **Namespace/Module Path:** Microsoft.FSharp.Core.Operators.OperatorIntrinsics  
  
 **Assembly:** FSharp.Core (in FSharp.Core.dll)  
  
## Syntax  
  
```  
// Signature:  
PowInt16 : int16 -> int -> int16  
  
// Usage:  
PowInt16 x n  
```  
  
#### Parameters  
 `x`  
 Type: [int16](../vs140/core.int16-type-abbreviation--fsharp-.md)  
  
 The value to raise to a power.  
  
 `n`  
 Type: [int](../vs140/core.int-type-abbreviation--fsharp-.md)  
  
 The power to raise the value to.  
  
## Return Value  
 The result of the exponentiation operation.  
  
## Remarks  
 This function is for use by compiled F# code and should not be used directly.
