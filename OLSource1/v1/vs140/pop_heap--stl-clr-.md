# pop_heap (STL-CLR)
Removes the largest element from the front of a heap to the next-to-last position in the range and then forms a new heap from the remaining elements.

<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This function behaves the same as the STL function <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. For more information, see [pop_heap](../vs140/pop_heap.md).
