# CAtlFileMappingBase::~CAtlFileMappingBase
The destructor.  
  
## Syntax  
  
```  
  
~CAtlFileMappingBase( ) throw( );  
  
```  
  
## Remarks  
 Frees any resources allocated by the class and calls the [CAtlFileMappingBase::Unmap](../vs140/catlfilemappingbase--unmap.md) method.

 **Header:** atlfile.h
