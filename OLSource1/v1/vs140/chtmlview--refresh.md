# CHtmlView::Refresh
Reloads the URL or file that the web browser is currently displaying.  
  
## Syntax  
  
```  
  
void Refresh( );  
  
```  
  
## Remarks  
 **Refresh** contains no parameters for setting the refresh level.  
  
 Applies to Internet Explorer and WebBrowser.  
  
## Requirements  
 **Header:** afxhtml.h
