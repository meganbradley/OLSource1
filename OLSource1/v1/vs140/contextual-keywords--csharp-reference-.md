# Contextual Keywords (C# Reference)
A contextual keyword is used to provide a specific meaning in the code, but it is not a reserved word in C#. The following contextual keywords are introduced in this section:  
  
|Keyword|Description|  
|-------------|-----------------|  
|[add](../vs140/add--csharp-reference-.md)|Defines a custom event accessor that is invoked when client code subscribes to the event.|  
|[async](../vs140/async--csharp-reference-.md)|Indicates that the modified method, lambda expression, or anonymous method is asynchronous.|  
|[await](../vs140/await--csharp-reference-.md)|Suspends an async method until an awaited task is completed.|  
|[dynamic](../vs140/dynamic--csharp-reference-.md)|Defines a reference type that enables operations in which it occurs to bypass compile-time type checking.|  
|[get](../vs140/get--csharp-reference-.md)|Defines an accessor method for a property or an indexer.|  
|[global](../vs140/global--csharp-reference-.md)|Specifies the default global namespace, which is otherwise unnamed.|  
|[partial](../vs140/partial--type---csharp-reference-.md)|Defines partial classes, structs, and interfaces throughout the same compilation unit.|  
|[remove](../vs140/remove--csharp-reference-.md)|Defines a custom event accessor that is invoked when client code unsubscribes from the event.|  
|[set](../vs140/set--csharp-reference-.md)|Defines an accessor method for a property or an indexer.|  
|[value](../vs140/value--csharp-reference-.md)|Used to set accessors and to add or remove event handlers.|  
|[var](../vs140/var--csharp-reference-.md)|Enables the type of a variable declared at method scope to be determined by the compiler.|  
|[where](../vs140/where--generic-type-constraint---csharp-reference-.md)|Adds constraints to a generic declaration. (See also [where](../vs140/where-clause--csharp-reference-.md)).|  
|[yield](../vs140/yield--csharp-reference-.md)|Used in an iterator block to return a value to the enumerator object or to signal the end of iteration.|  
  
 All query keywords introduced in C# 3.0 are also contextual. For more information, see [Query Keywords (C# Reference)](../vs140/query-keywords--csharp-reference-.md).
