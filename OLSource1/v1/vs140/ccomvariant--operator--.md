# CComVariant::operator &lt;
Indicates whether the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object is less than the specified **VARIANT**.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 Returns **true** if the value of the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> object is less than the value of *varSrc*. Otherwise, **false**. The operator uses the user's default locale to perform the comparison.
