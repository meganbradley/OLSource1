# CSid::EqualPrefix
Tests <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> (security identifier) prefixes for equality.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 The <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> (security identifier) structure or <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> object to compare.  
  
## Return Value  
 Returns **true** on success, **false** on failure.  
  
## Remarks  
 See [EqualPrefixSid](http://msdn.microsoft.com/library/windows/desktop/aa446621) in the [!INCLUDE[winSDK](../vs140/includes/winsdk_md.md)] for more details.
