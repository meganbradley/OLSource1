# CMFCPopupMenu::CMFCPopupMenu
Constructs a [CMFCPopupMenu](../vs140/cmfcpopupmenu-class.md) object.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 A pointer to a customization page.  
  
 [in] <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 A string that contains the menu caption.  
  
## Remarks  
 This method allocates the resources for a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>. To create the pop-up menu item, call [CMFCPopupMenu::Create](../vs140/cmfcpopupmenu--create.md).

 **Header:** afxpopupmenu.h
