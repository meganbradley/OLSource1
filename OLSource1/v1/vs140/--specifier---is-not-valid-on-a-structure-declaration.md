# &#39;&lt;specifier&gt;&#39; is not valid on a Structure declaration
You have used a specifier, such as <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, that is not valid on a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> declaration.  
  
 **Error ID:** BC30395  
  
### To correct this error  
  
-   Remove the specifier.
