# asinhf Function
Calculates the inverse hyperbolic sine of the argument  
  
## Syntax  
  
```  
inline float asinhf(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the inverse hyperbolic sine value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
