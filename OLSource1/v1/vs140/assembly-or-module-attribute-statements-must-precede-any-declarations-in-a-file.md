# Assembly or Module attribute statements must precede any declarations in a file
Global attributes must be declared at the top of a source file, after <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statements, but before any other statements.  
  
 **Error ID:** BC30637  
  
### To correct this error  
  
1.  Place global attributes, such as <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> at the top of your source file.
