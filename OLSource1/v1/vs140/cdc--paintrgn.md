# CDC::PaintRgn
Fills the region specified by <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> using the current brush.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 Identifies the region to be filled. The coordinates for the given region are specified in logical units.  
  
## Return Value  
 Nonzero if the function is successful; otherwise 0.  
  
## Requirements  
 **Header:** afxwin.h
