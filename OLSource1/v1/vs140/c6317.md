# C6317
warning C6317: incorrect operator: logical-not (!) is not interchangeable with ones-complement (~)  
  
 This warning indicates that a logical-not (<CodeContentPlaceHolder>2\</CodeContentPlaceHolder>) is being applied to a constant that is likely to be a bit-flag. The result of logical-not is Boolean; it is incorrect to apply the bitwise-and (<CodeContentPlaceHolder>3\</CodeContentPlaceHolder>) operator to a Boolean value. Use the ones-complement (<CodeContentPlaceHolder>4\</CodeContentPlaceHolder>) operator to flip flags.  
  
## Example  
 The following code generates this warning:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 To correct this warning, use the following code:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
