# &#39;Next&#39; expected
An <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> construct occurs without a concluding <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
 **Error ID:** BC30003  
  
### To correct this error  
  
-   Add the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> keyword to the end of the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement.
