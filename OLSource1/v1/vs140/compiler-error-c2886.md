# Compiler Error C2886
'class::identifier' : symbol cannot be used in a member using-declaration  
  
 A <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> declaration uses a symbol, such as a namespace name. A <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> declaration is for declaring base class members.  
  
 The following sample generates C2886:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
