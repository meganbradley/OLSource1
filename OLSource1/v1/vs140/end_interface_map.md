# END_INTERFACE_MAP
Ends the interface map in the implementation file.  
  
## Syntax  
  
```  
  
END_INTERFACE_MAP( )  
  
```  
  
## Remarks  
 For more information about interface maps, see [Technical Note 38](../vs140/tn038--mfc-ole-iunknown-implementation.md).
