# CDaoTableDef::m_pDAOTableDef
Contains a pointer to the OLE interface for the DAO tabledef object underlying the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> object.  
  
## Remarks  
 Use this pointer if you need to access the DAO interface directly.  
  
## Requirements  
 **Header:** afxdao.h
