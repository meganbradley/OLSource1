# CMFCPropertyGridProperty::IsAllowEdit
Indicates whether a property is editable.  
  
## Syntax  
  
```  
BOOL IsAllowEdit() const;  
```  
  
## Return Value  
 `TRUE` if the property is editable; otherwise `FALSE`.

## Requirements  
 **Header:** afxpropertygridctrl.h
