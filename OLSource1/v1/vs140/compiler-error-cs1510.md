# Compiler Error CS1510
A ref or out argument must be an assignable variable  
  
 Only a variable can be passed as a [ref](../vs140/ref--csharp-reference-.md) parameter in a method call. A <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> value is like passing a pointer.
