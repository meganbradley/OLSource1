# &#39;Set&#39; is already declared
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement appears more than once in a code block. The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement declares a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> property procedure used to assign a value to a property.  
  
 **Error ID:** BC30444  
  
### To correct this error  
  
-   Remove the redundant <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement.
