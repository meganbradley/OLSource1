# Recordset: Declaring a Class for a Table (ODBC)
This topic applies to the MFC ODBC classes.  
  
 The most common recordset class opens a single table. To declare a recordset class for a single table, use the [MFC ODBC Consumer Wizard](../vs140/adding-an-mfc-odbc-consumer.md) from **Add Class** and choose each column you want by naming a corresponding recordset field data member.  
  
 Other uses for recordsets include:  
  
-   Joining two or more tables.  
  
-   Containing the results of a predefined query.
