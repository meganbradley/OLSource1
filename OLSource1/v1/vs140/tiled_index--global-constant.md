# tiled_index::global Constant
Stores an [index](../vs140/index-class.md) object of rank 1, 2, or 3 that represents the global index of an object.  
  
## Syntax  
  
```  
const index<rank> global;  
```  
  
## Requirements  
 **Header:** amp.h  
  
 **Namespace:** Concurrency
