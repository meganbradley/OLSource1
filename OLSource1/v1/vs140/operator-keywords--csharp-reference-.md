# Operator Keywords (C# Reference)
Used to perform miscellaneous actions such as creating objects, checking the run-time type of an object, obtaining the size of a type, and other actions. This section introduces the following keywords:  
  
-   [as](../vs140/as--csharp-reference-.md) Converts an object to a compatible type.  
  
-   [await](../vs140/await--csharp-reference-.md) Suspends an async method until an awaited task is completed.  
  
-   [is](../vs140/is--csharp-reference-.md) Checks the run-time type of an object.  
  
-   [new](../vs140/new--csharp-reference-.md)  
  
    -   [new Operator](../vs140/new-operator--csharp-reference-.md) Creates objects.  
  
    -   [new Modifier](../vs140/new-modifier--csharp-reference-.md) Hides an inherited member.  
  
    -   [new Constraint](../vs140/new-constraint--csharp-reference-.md) Qualifies a type parameter.  
  
-   [sizeof](../vs140/sizeof--csharp-reference-.md) Obtains the size of a type.  
  
-   [typeof](../vs140/typeof--csharp-reference-.md) Obtains the **System.Type** object for a type.  
  
-   [true](../vs140/true--csharp-reference-.md)  
  
    -   [true Operator](../vs140/true-operator--csharp-reference-.md) Returns the boolean value true to indicate true and returns false otherwise.  
  
    -   [true Literal](../vs140/true-literal--csharp-reference-.md) Represents the boolean value true.  
  
-   [false](../vs140/false--csharp-reference-.md)  
  
    -   [false Operator](../vs140/false-operator--csharp-reference-.md) Returns the Boolean value true to indicate false and returns false otherwise.  
  
    -   [false Literal](../vs140/false-literal--csharp-reference-.md) Represents the boolean value false.  
  
-   [stackalloc](../vs140/stackalloc--csharp-reference-.md) Allocates a block of memory on the stack.  
  
 The following keywords, which can be used as operators and as statements, are covered in the [Statements](../vs140/statement-keywords--csharp-reference-.md) section:  
  
-   [checked](../vs140/checked--csharp-reference-.md) Specifies checked context.  
  
-   [unchecked](../vs140/unchecked--csharp-reference-.md) Specifies unchecked context.
