# CObArray::IsEmpty
Determines if the array is empty.  
  
## Syntax  
  
```  
  
BOOL IsEmpty( ) const;  
  
```  
  
## Return Value  
 Nonzero if the array is empty; otherwise 0.  
  
## Requirements  
 **Header:** afxcoll.h
