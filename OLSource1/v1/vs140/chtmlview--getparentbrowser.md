# CHtmlView::GetParentBrowser
Call this member function to retrieve a pointer to the parent object of the WebBrowser control.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 A pointer to the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> interface of the object that is the parent of the WebBrowser control.
