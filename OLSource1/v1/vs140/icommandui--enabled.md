# ICommandUI::Enabled
Enables or disables the user interface item for this command.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This property enables or disables the user interface item for this command. Set <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> to <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> to enable the item, <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> to disable it.
