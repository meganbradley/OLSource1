# -SECTION (DUMPBIN)
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This option restricts the output to information on the specified *section*. Use the [/HEADERS](../vs140/-headers.md) option to get a list of sections in the file.  
  
 Only the [/HEADERS](../vs140/-headers.md) DUMPBIN option is available for use on files produced with the [/GL](../vs140/-gl--whole-program-optimization-.md) compiler option.
