# CMFCButton::SetTextColor
Sets the color of the button text for a button that is not selected.  
  
## Syntax  
  
```  
void SetTextColor(  
   COLORREF clrText   
);  
```  
  
#### Parameters  
 [in] `clrText`  
 An RGB color value.
