# Property &#39;&lt;propertyname&gt;&#39; is &#39;ReadOnly&#39;
You have tried to assign a value to a property that is declared <CodeContentPlaceHolder>0\</CodeContentPlaceHolder>.  
  
 **Error ID:** BC30526  
  
### To correct this error  
  
-   Remove the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> specifier from the property declaration.
