# &#39;Finally&#39; must end with a matching &#39;End Try&#39;
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement appears in your code without a matching <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>statement. <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statements must be concluded with an <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement.  
  
 **Error ID:** BC30442  
  
### To correct this error  
  
1.  Remove the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> statement.  
  
2.  Add an <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> statement to conclude the block.
