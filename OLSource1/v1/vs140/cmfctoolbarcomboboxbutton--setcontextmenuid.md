# CMFCToolBarComboBoxButton::SetContextMenuID
Sets the shortcut menu resource ID for the combo box button.  
  
## Syntax  
  
```  
void SetContextMenuID(  
   UINT uiResID  
);  
```  
  
#### Parameters  
 [in] `uiResID`  
 The shortcut menu resource ID.  
  
## Requirements  
 **Header:** afxtoolbarcomboboxbutton.h
