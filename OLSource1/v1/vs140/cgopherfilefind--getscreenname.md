# CGopherFileFind::GetScreenName
Call this member function to get the name of the gopher screen.  
  
## Syntax  
  
```  
  
CString GetScreenName( ) const;  
  
```  
  
## Return Value  
 The name of the gopher screen.  
  
## Requirements  
 **Header:** afxinet.h
