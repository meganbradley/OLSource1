# Fatal Error C1103
fatal error importing progid: 'message'  
  
 The compiler detected a problem importing a type library.  For example, you cannot specify a type library with progid and also specify <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
 For more information, see [The #import Directive](../vs140/sharpimport-directive--c---.md).  
  
 The following sample will generate C1103:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
