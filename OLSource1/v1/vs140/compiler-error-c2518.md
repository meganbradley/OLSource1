# Compiler Error C2518
keyword 'keyword' illegal in base class list; ignored  
  
 The keywords <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> should not appear in a base class list.  
  
 The following sample generates C2518:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
