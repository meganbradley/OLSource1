# CMDIFrameWndEx::OnUpdateFrameMenu
Called by the framework to update the frame menu.  
  
## Syntax  
  
```  
virtual void OnUpdateFrameMenu(  
   HMENU hMenuAlt  
);  
```  
  
#### Parameters  
 [in] `hMenuAlt`  
 A handle to a menu.  
  
## Requirements  
 **Header:** afxmdiframewndex.h
