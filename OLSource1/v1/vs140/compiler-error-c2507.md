# Compiler Error C2507
'identifier' : too many virtual modifiers on the base class  
  
 A class or structure is declared as <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> more than once. Only one <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> modifier can appear for each class in a list of base classes.  
  
 The following sample generates C2507:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
