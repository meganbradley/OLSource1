# CToolBarCtrl::SetOwner
Sets the owner window for the toolbar control.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 Pointer to the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>-derived object that will be the new owner window for the toolbar control.  
  
## Remarks  
 The owner window is the window that receives notifications from the toolbar.

 **Header:** afxcmn.h
