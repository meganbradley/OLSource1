# .STACK
When used with [.MODEL](../vs140/.model.md), defines a stack segment (with segment name STACK). The optional <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> specifies the number of bytes for the stack (default 1,024). The <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> directive automatically closes the stack statement.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
