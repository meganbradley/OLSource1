# CD2DTextFormat::GetLocaleName
Gets a copy of the locale name.  
  
## Syntax  
  
```  
CString GetLocaleName() const;  
```  
  
## Return Value  
 CString object that contains the current locale name.  
  
## Requirements  
 **Header:** afxrendertarget.h
