# MSBuild Error MSB1024
**Only one schema can be specified for validation of the project.**  
  
 Only one schema can be specified for the **/validate** switch.  
  
### To correct this error  
  
1.  Specify only one schema to validate the project against using the format <CodeContentPlaceHolder>0\</CodeContentPlaceHolder>, for example, <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
