# list::empty
Tests if a list is empty.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 **true** if the list is empty; **false** if the list is not empty.  
  
## Example  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 **The list is not empty.**   
## Requirements  
 **Header:** \<list>  
  
 **Namespace:** std
