# coshf Function
Calculates the hyperbolic cosine value of the argument  
  
## Syntax  
  
```  
inline float coshf(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the hyperbolic cosine value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
