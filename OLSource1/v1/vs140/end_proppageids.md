# END_PROPPAGEIDS
Ends the definition of your property page ID list.  
  
## Syntax  
  
```  
  
END_PROPPAGEIDS(  
class_name )  
```  
  
#### Parameters  
 *class_name*  
 The name of the control class that owns the property page.  
  
## Requirements  
 **Header:** afxctl.h
