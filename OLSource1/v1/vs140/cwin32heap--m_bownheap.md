# CWin32Heap::m_bOwnHeap
A flag used to determine current ownership of the heap handle stored in [m_hHeap](../vs140/cwin32heap--m_hheap.md).  
  
## Syntax  
  
```  
  
bool m_bOwnHeap;  
  
```  
  
## Requirements  
 **Header:** atlmem.h
