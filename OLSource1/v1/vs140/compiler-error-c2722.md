# Compiler Error C2722
'::operator' : illegal following operator command; use 'operator operator'  
  
 An <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement redefines <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>. The <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> operators are global, so the scope resolution operator (<CodeContentPlaceHolder>5\</CodeContentPlaceHolder>) is meaningless. Remove the <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> operator.
