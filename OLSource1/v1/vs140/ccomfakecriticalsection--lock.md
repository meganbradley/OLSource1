# CComFakeCriticalSection::Lock
Does nothing since there is no critical section.  
  
## Syntax  
  
```  
  
HRESULT Lock( ) throw( );  
  
```  
  
## Return Value  
 Returns S_OK.  
  
## Requirements  
 **Header:** atlcore.h
