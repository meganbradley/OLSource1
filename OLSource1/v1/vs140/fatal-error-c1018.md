# Fatal Error C1018
unexpected #elif  
  
 The <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> directive appears outside an <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>, or <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> construct. Use <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> only within one of these constructs.  
  
 The following sample generates C1018:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 Possible resolution:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
