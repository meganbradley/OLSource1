# set::clear
Erases all the elements of a set.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Example  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 **The size of the set is initially 2.**  
**The size of the set after clearing is 0.**   
## Requirements  
 **Header:** \<set>  
  
 **Namespace:** std
