# CDC::EndPath
Closes a path bracket and selects the path defined by the bracket into the device context.  
  
## Syntax  
  
```  
  
BOOL EndPath( );  
```  
  
## Return Value  
 Nonzero if the function is successful; otherwise 0.  
  
## Example  
 See the example for [CDC::BeginPath](../vs140/cdc--beginpath.md).  
  
## Requirements  
 **Header:** afxwin.h
