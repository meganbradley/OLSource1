# CMFCPropertyGridProperty::IsParentExpanded
Indicates whether the parents of the current property are expanded.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if all parents of the current property are expanded; <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> if the parent properties are collapsed.
