# Compiler Error C2194
'identifier' : is a text segment  
  
 The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> pragma uses a segment name used with <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
 The following sample generates C2194:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
