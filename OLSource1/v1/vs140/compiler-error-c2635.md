# Compiler Error C2635
cannot convert an 'identifier1*' to an 'identifier2\*'; conversion from a virtual base class is implied  
  
 The conversion requires a cast from a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> base class to a derived class, which is not allowed.
