# #endif (C# Reference)
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder> specifies the end of a conditional directive, which began with the [#if](../vs140/sharpif--csharp-reference-.md) directive. For example,  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 A conditional directive, beginning with a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> directive, must explicitly be terminated with a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> directive. See [#if](../vs140/sharpif--csharp-reference-.md) for an example of how to use <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.
