# COleSafeArray::operator ==
This operator compares two arrays (**SAFEARRAY**, **VARIANT**, <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> arrays) and returns nonzero if they are equal; otherwise 0.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 Two arrays are equal if they have an equal number of dimensions, equal size in each dimension, and equal element values.
