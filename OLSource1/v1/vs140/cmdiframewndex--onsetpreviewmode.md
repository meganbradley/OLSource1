# CMDIFrameWndEx::OnSetPreviewMode
Sets the application's main frame window print-preview mode.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>

 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 If <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, sets print-preview mode. If <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, cancels preview mode.  
  
 [in] <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>  
 A pointer to a <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> structure.
