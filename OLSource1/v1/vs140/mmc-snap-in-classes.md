# MMC Snap-In Classes
The following classes provide support for developing Microsoft Management Console (MMC) snap-in components:  
  
-   [CSnapInItemImpl](../vs140/csnapinitemimpl-class.md) Implements a snap-in node object, such as adding menu items and toolbars, and forwarding commands for the snap-in node to the appropriate handler function.  
  
-   [CSnapInPropertyPageImpl](../vs140/csnapinpropertypageimpl-class.md) Implements a snap-in property page object.
