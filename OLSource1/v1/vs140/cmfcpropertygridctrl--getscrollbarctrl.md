# CMFCPropertyGridCtrl::GetScrollBarCtrl
Retrieves a pointer to the scroll bar control in the property grid control.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 The orientation of the scroll bar, which must be <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Return Value  
 A pointer to a scroll bar object, or <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> if there is no scroll bar or the scroll bar orientation is <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
## Remarks  
 Use this method to gain direct access to the vertical scroll bar control.

 **Header:** afxpropertygridctrl.h
