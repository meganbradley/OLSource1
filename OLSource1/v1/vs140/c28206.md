# C28206
warning C28206: \<expression> : left operand points to a struct, use <CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
  
 This warning is reported when the struct pointer dereference notation <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> was expected.
