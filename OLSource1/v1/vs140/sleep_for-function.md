# sleep_for Function
Blocks the calling thread.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 A [duration](../vs140/duration-class.md) object that specifies a time interval.  
  
## Remarks  
 The function blocks the calling thread for at least the time that's specified by <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>. This function does not throw any exceptions.  
  
## Requirements  
 **Header:** thread  
  
 **Namespace:** std::this_thread
