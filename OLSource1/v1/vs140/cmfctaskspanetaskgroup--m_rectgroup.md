# CMFCTasksPaneTaskGroup::m_rectGroup
Specifies the bounding rectangle of the group.  
  
## Syntax  
  
```  
CRect m_rectGroup;  
```  
  
## Remarks  
 This value is calculated automatically by the framework.  
  
## Requirements  
 **Header:** afxTasksPane.h
