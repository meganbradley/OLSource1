# no_namespace
**C++ Specific**  
  
 Specifies that the namespace name is not generated by the compiler.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 The type-library contents in the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> header file are normally defined in a namespace. The namespace name is specified in the **library** statement of the original IDL file. If the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> attribute is specified, then this namespace is not generated by the compiler.  
  
 If you want to use a different namespace name, then use the [rename_namespace](../vs140/rename_namespace.md) attribute instead.  
  
 **END C++ Specific**
