# CSettingsStore::Close
Closes the open registry key.  
  
## Syntax  
  
```  
virtual void Close();  
```  
  
## Remarks  
 By default, this method is called from the destructor of the [CSettingsStore Class](../vs140/csettingsstore-class.md).  
  
## Requirements  
 **Header:** afxsettingsstore.h
