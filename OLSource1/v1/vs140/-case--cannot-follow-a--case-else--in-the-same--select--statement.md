# &#39;Case&#39; cannot follow a &#39;Case Else&#39; in the same &#39;Select&#39; statement
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement introduces statements to be executed if no match is found for the initial <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. A <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement has been found after a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> in the same <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> block.  
  
 **Error ID:** BC30321  
  
### To correct this error  
  
-   Move the <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> to the appropriate location after the <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> statement.
