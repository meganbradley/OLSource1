# CMFCRibbonCategory::GetTabRect
Retrieves the display rectangle for the ribbon category tab.  
  
## Syntax  
  
```  
CRect GetTabRect() const;  
```  
  
## Return Value  
 The display rectangle for the ribbon category tab.
