# Compiler Error CS2022
Options '/out' and '/target' must appear before source file names  
  
 The [/out (Set Output Filename)](../vs140/-out--csharp-compiler-options-.md) and [/target (Specify Output File Format)](../vs140/-target--csharp-compiler-options-.md) compiler options, when specified on the command line, must precede the source code files.
