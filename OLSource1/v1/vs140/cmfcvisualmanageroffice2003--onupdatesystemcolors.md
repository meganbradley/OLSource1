# CMFCVisualManagerOffice2003::OnUpdateSystemColors
The framework calls this function when the system colors change.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 The framework calls this method as a part of processing the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> message. Override this method in a derived visual manager if you want to execute custom code when the colors change in your application.

 **Header:** afxvisualmanageroffice2003.h
