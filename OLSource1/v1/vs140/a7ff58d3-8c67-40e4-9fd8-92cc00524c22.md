# Type parameter &#39;&lt;typeparametername1&gt;&#39; must have either a &#39;New&#39; constraint or a &#39;Structure&#39; constraint to satisfy the &#39;New&#39; constraint for type parameter &#39;&lt;typeparametername2&gt;&#39;
A statement constructs a generic type passing a type parameter that is not constrained to satisfy a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> constraint.  
  
 The <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> constraint means that the type argument supplied to that type parameter must expose a parameterless constructor accessible to the code that creates objects from it. All value types have parameterless constructors, but not all reference types do. Therefore the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> constraint satisfies the <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> constraint, but the <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> constraint, or a class or interface name, does not.  
  
 The following statements can generate this error.  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 When class <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> creates an object from <CodeContentPlaceHolder>8\</CodeContentPlaceHolder>, type parameter <CodeContentPlaceHolder>9\</CodeContentPlaceHolder> does not satisfy the <CodeContentPlaceHolder>10\</CodeContentPlaceHolder> constraint on type parameter <CodeContentPlaceHolder>11\</CodeContentPlaceHolder>.  
  
 **Error ID:** BC32084  
  
### To correct this error  
  
-   If the type parameter to be passed to the generic type can satisfy the <CodeContentPlaceHolder>12\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>13\</CodeContentPlaceHolder> constraint, then add the appropriate constraint to its definition.  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
-   If the type parameter cannot satisfy the <CodeContentPlaceHolder>14\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>15\</CodeContentPlaceHolder> constraint, then do not pass it to the generic type. You must pass something else as the type argument.
