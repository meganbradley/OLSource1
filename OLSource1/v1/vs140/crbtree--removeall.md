# CRBTree::RemoveAll
Call this method to remove all elements from the `CRBTree` object.  
  
## Syntax  
  
```  
  
void RemoveAll( ) throw( );  
  
```  
  
## Remarks  
 Clears out the `CRBTree` object, freeing the memory used to store the elements.  
  
## Requirements  
 **Header:** atlcoll.h
