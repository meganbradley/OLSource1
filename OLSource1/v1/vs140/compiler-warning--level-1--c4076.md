# Compiler Warning (level 1) C4076
'typemod' : can not be used with type 'typename'  
  
 A type modifier, whether it is **signed** or <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, cannot be used with a noninteger type. ***typemod*** is ignored.
