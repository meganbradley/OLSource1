# &#39;Join&#39; expected
A <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> clause has been specified without a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> keyword.  
  
 **Error ID:** BC36631  
  
### To correct this error  
  
1.  Add the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> keyword to the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> clause, as shown in the following example:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
