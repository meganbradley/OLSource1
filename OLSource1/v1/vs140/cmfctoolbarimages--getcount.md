# CMFCToolBarImages::GetCount
Returns the number of images in the toolbar images list.  
  
## Syntax  
  
```  
int GetCount() const;  
```  
  
## Return Value  
 The number of images in the [CMFCToolbarImages](../vs140/cmfctoolbarimages-class.md) object.  
  
## Requirements  
 **Header:** afxtoolbarimages.h
