# CAtlServiceModuleT::m_szServiceName
The name of the service being registered.  
  
## Syntax  
  
```  
  
TCHAR [256] m_szServiceName;  
  
```  
  
## Remarks  
 A null-terminated string which stores the name of the service.  
  
## Requirements  
 **Header:** atlbase.h
