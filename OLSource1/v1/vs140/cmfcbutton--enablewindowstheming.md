# CMFCButton::EnableWindowsTheming
Specifies whether the style of the button border corresponds to the current Windows theme.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> to use the current Windows theme to draw button borders; <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> to not use the Windows theme. The default is <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
## Remarks  
 This method affects all buttons in your application that are derived from the <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> class.

 **Header:** afxbutton.h
