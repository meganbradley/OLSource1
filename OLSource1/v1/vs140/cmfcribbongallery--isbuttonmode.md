# CMFCRibbonGallery::IsButtonMode
Specifies whether the palette is contained in a gallery button.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the palette is displayed as a drop-down menu button; <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> if the palette is displayed directly on the ribbon.
