# Sign of Bit Fields
**ANSI 3.5.2.1** Whether a "plain" <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> field is treated as a **signed int** bit field or as an unsigned int bit field  
  
 Bit fields can be signed or unsigned. Plain bit fields are treated as signed.
