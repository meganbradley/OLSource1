# is_placeholder Class
Tests if type is a placeholder.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 The constant value <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> is 0 if the type <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> is not a placeholder; otherwise, its value is the position of the function call argument that it binds to. You use it to determine the value <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> for the Nth placeholder <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>.  
  
## Example  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
  **0**  
**3**

 **Header:** \<functional>  
  
 **Namespace:** std
