# IDebugModule3::IsUserCode
Retrieves information on whether the module represents user code or not.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 [out] Nonzero (<CodeContentPlaceHolder>3\</CodeContentPlaceHolder>) if module represents user code, zero (<CodeContentPlaceHolder>4\</CodeContentPlaceHolder>) if it does not.  
  
## Return Value  
 If successful, returns <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>; otherwise, returns error code.
