# Messages
The message loop in the **Run** member function of class <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> retrieves queued messages generated by various events. For example, when the user clicks the mouse, Windows sends several mouse-related messages, such as <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> when the left mouse button is pressed and <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> when the left mouse button is released. The framework's implementation of the application message loop dispatches the message to the appropriate window.
