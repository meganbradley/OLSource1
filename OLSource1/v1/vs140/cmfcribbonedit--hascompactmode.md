# CMFCRibbonEdit::HasCompactMode
Indicates whether the display size for the [CMFCRibbonEdit Class](../vs140/cmfcribbonedit-class.md) control can be compact.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Always returns <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
## Remarks  
 By default this method always returns <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>. Override this method to indicate whether the display size can be compact.  
  
## Requirements  
 **Header:** afxribbonedit.h
