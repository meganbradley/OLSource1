# CMFCShellListCtrl::IsDesktop
Determines if the folder that is displayed in the [CMFCShellListCtrl](../vs140/cmfcshelllistctrl-class.md) object is the desktop folder.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the displayed folder is the desktop folder; <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> otherwise.

 **Header:** afxshelllistctrl.h
