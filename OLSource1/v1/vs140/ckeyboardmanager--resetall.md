# CKeyboardManager::ResetAll
Reloads the shortcut key tables from the application resource.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This function clears the shortcuts stored in the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> instance. It will then reload the state of the keyboard manager from the application resource.
