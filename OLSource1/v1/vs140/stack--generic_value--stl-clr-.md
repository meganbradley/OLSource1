# stack::generic_value (STL-CLR)
The type of an element for use with the generic interface for the container.

<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>

 The type describes an object of type <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> that describes the stored element value for use with the generic interface for this template container class.
