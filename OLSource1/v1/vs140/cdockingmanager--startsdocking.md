# CDockingManager::StartSDocking
Starts the smart docking of the specified window according to the alignment of the smart docking manager.  
  
## Syntax  
  
```  
void StartSDocking(  
   CWnd* pDockingWnd  
);  
```  
  
#### Parameters  
 [in] `pDockingWnd`  
 A pointer to a window to dock.  
  
## Requirements  
 **Header:** afxdockingmanager.h
