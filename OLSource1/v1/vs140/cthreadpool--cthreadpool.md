# CThreadPool::CThreadPool
The constructor for the thread pool.  
  
## Syntax  
  
```  
  
CThreadPool( ) throw( );  
  
```  
  
## Remarks  
 Initializes the timeout value to [ATLS_DEFAULT_THREADPOOLSHUTDOWNTIMEOUT](../vs140/atls_default_threadpoolshutdowntimeout.md).  
  
## Requirements  
 **Header:** atlutil.h
