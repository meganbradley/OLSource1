# CBasePane::SetControlBarStyle
Sets the control bar style.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 A bitwise-OR combination of the following possible values.  
  
|Style|Description|  
|-----------|-----------------|  
|<CodeContentPlaceHolder>2\</CodeContentPlaceHolder>|Makes the control bar float.|  
|<CodeContentPlaceHolder>3\</CodeContentPlaceHolder>|Enables auto-hide mode.|  
|<CodeContentPlaceHolder>4\</CodeContentPlaceHolder>|Enables resizing of the control bar. When this flag is set, the control bar can be placed in a dockable pane.|  
|<CodeContentPlaceHolder>5\</CodeContentPlaceHolder>|Enables hiding of the control bar.|  
  
## Requirements  
 **Header:** afxbasepane.h
