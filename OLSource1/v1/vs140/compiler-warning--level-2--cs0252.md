# Compiler Warning (level 2) CS0252
Possible unintended reference comparison; to get a value comparison, cast the left hand side to type 'type'  
  
 The compiler is doing a reference comparison. If you want to compare the value of strings, cast the left side of the expression to <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
 The following sample generates CS0252:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
