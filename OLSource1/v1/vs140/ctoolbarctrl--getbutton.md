# CToolBarCtrl::GetButton
Retrieves information about the specified button in a toolbar control.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 Zero-based index of the button for which to retrieve information.  
  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 Address of the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> structure that is to receive a copy of the button information. See [CToolBarCtrl::AddButtons](../vs140/ctoolbarctrl--addbuttons.md) for information about the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> structure.  
  
## Return Value  
 Nonzero if successful; otherwise zero.  
  
## Requirements  
 **Header:** afxcmn.h
