# CD2DBitmapBrush::SetBitmap
Specifies the bitmap source that this brush uses to paint  
  
## Syntax  
  
```  
void SetBitmap(  
   CD2DBitmap* pBitmap  
);  
```  
  
#### Parameters  
 `pBitmap`  
 The bitmap source used by the brush  
  
## Requirements  
 **Header:** afxrendertarget.h
