# #warning (C# Reference)
<CodeContentPlaceHolder>2\</CodeContentPlaceHolder> lets you generate a level one warning from a specific location in your code. For example:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 A common use of <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> is in a conditional directive. It is also possible to generate a user-defined error with [#error](../vs140/sharperror--csharp-reference-.md).  
  
## Example  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
