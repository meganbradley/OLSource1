# CMFCPropertyGridProperty::GetHierarchyLevel
Retrieves the zero-based index of the property's hierarchy level.  
  
## Syntax  
  
```  
int GetHierarchyLevel() const;  
```  
  
## Return Value  
 The property's hierarchical level.
