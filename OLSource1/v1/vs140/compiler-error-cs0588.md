# Compiler Error CS0588
Internal Compiler Error: stage 'LEX'  
  
 Try to determine whether the compiler is failing because of its inability to parse unexpected syntax. If this is not the case, try [Getting Help from Microsoft Product Support Services (Visual Studio)](../vs140/talk-to-us.md).
