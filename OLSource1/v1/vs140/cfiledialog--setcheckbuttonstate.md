# CFileDialog::SetCheckButtonState
Sets the current state of a check button (check box) in the dialog.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 The ID of the check box.  
  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 The state of the check box. <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> indicates checked; <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> indicates Unchecked.
