# CMFCColorBar::EnableAutomaticButton
Shows or hides the automatic button.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 The text label of the *automatic* (default) color button, or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
 The standard label for the automatic button is **Automatic**.  
  
 [in] <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 The default color that the framework applies when you click the automatic button.  
  
 [in] <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>  
 <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> to enable the automatic button; <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> to disable the automatic button. The default value is <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>.
