# CMFCPopupMenuBar::SetOffset
Sets the row offset of the popup menu bar.  
  
## Syntax  
  
```  
void SetOffset(  
   int iOffset  
);  
```  
  
#### Parameters  
 [in] `iOffset`  
 The number of rows that the popup menu bar should be offset.
