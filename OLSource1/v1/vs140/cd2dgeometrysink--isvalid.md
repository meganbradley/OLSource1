# CD2DGeometrySink::IsValid
Checks geometry sink validity  
  
## Syntax  
  
```  
BOOL IsValid() const;  
```  
  
## Return Value  
 TRUE if geometry sink is valid; otherwise FALSE.

 **Header:** afxrendertarget.h
