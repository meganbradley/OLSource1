# CMFCRibbonButton::SetRightAlignMenu
Aligns the pop-up menu to the edge of the button.  
  
## Syntax  
  
```  
void SetRightAlignMenu(  
   BOOL bSet=TRUE   
);  
```  
  
#### Parameters  
 [in] `bSet`  
 If `TRUE`, the menu is right-aligned. Otherwise, the menu is left-aligned  
  
## Requirements  
 **Header:** afxribbonbutton.h
