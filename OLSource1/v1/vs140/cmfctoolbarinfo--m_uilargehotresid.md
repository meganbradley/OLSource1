# CMFCToolBarInfo::m_uiLargeHotResID
Specifies a resource ID for all the large highlighted images of a toolbar.  
  
## Syntax  
  
```  
UINT m_uiLargeHotResID;  
```  
  
## Requirements  
 **Header:** afxtoolbar.h
