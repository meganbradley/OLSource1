# CMFCColorBar::ContextToSize
Calculates the vertical and horizontal margins that are required to contain the buttons on the color bar control, and adjusts the location of those buttons.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
  
|Parameter|Description|  
|---------------|-----------------|  
|[in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>|<CodeContentPlaceHolder>2\</CodeContentPlaceHolder> to specify that the shape of the buttons on a color bar control are square; otherwise, <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>. The default value is <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.|  
|[in] <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>|<CodeContentPlaceHolder>6\</CodeContentPlaceHolder> to specify that the content on the face of a color bar control button is centered; otherwise, <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>. The default value is <CodeContentPlaceHolder>8\</CodeContentPlaceHolder>.|
