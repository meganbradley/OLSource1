# basic_stringbuf::traits_type
Associates a type name with the **Tr** template parameter.  
  
## Syntax  
  
```  
  
typedef Tr traits  
_  
type;  
  
```  
  
## Remarks  
 The type is a synonym for the template parameter **Tr**.  
  
## Requirements  
 **Header:** \<sstream>  
  
 **Namespace:** std
