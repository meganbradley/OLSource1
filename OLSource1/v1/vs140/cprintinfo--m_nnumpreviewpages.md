# CPrintInfo::m_nNumPreviewPages
Contains the number of pages displayed in preview mode; it can be either 1 or 2.  
  
## Remarks  
 The **m_nNumPreviewPages** member is a public variable of type **UINT**.  
  
## Requirements  
 **Header:** afxext.h
