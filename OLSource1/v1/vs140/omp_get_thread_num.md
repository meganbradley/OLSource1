# omp_get_thread_num
Returns the thread number of the thread executing within its thread team.  
  
## Syntax  
  
```  
int omp_get_thread_num( );  
```  
  
## Remarks  
 For more information, see [3.1.4 omp_get_thread_num Function](../vs140/3.1.4-omp_get_thread_num-function.md).
