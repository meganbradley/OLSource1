# CAnimationTimerEventHandler::OnPreUpdate
Handles events that occur before an animation update begins.  
  
## Syntax  
  
```  
IFACEMETHOD(  
   OnPreUpdate  
)();  
```  
  
## Return Value  
 S_OK if the method succeeds; otherwise E_FAIL.
