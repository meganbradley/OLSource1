# Structure &#39;&lt;structurename&gt;&#39; must contain at least one instance member variable or Event declaration
A functionally empty structure occurs. At least one variable member must be declared between the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement and the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement.  
  
 **Error ID:** BC30281  
  
### To correct this error  
  
-   Add a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement or an <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement to the structure.
