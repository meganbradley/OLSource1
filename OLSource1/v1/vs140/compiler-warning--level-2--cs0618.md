# Compiler Warning (level 2) CS0618
'member' is obsolete: 'text'  
  
 A class member was marked with the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> attribute, such that a warning will be issued when the class member is referenced. For more information, see [Common Attributes](../vs140/common-attributes--csharp-and-visual-basic-.md).  
  
 The following sample generates CS0618:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
