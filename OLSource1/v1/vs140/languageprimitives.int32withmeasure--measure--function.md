```  
// Signature:  
Int32WithMeasure : int -> int<'u>  
  
// Usage:  
Int32WithMeasure value  
```  
  
#### Parameters  
 `value`  
 Type: [int](../vs140/core.int-type-abbreviation--fsharp-.md)
