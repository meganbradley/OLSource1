# Compiler Warning (level 2) CS0469
The 'goto case' value is not implicitly convertible to type 'type'  
  
 When you use <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, there must be an implicit conversion from the value of the goto case to the type of the switch.  
  
## Example  
 The following sample generates CS0469.  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
