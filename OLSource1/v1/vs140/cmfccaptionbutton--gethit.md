# CMFCCaptionButton::GetHit
Returns the command represented by the button.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 The command represented by the button.  
  
 The following table lists possible return values.  
  
|Value|Command|  
|-----------|-------------|  
|<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>|Close button.|  
|<CodeContentPlaceHolder>2\</CodeContentPlaceHolder>|Minimize button.|  
|<CodeContentPlaceHolder>3\</CodeContentPlaceHolder>|Maximize button.|  
|<CodeContentPlaceHolder>4\</CodeContentPlaceHolder>|Left arrow button.|  
|<CodeContentPlaceHolder>5\</CodeContentPlaceHolder>|Right arrow button.|  
|<CodeContentPlaceHolder>6\</CodeContentPlaceHolder>|Down arrow menu button.|  
|<CodeContentPlaceHolder>7\</CodeContentPlaceHolder>|The default value; represents no command.|  
  
## Requirements  
 **Header:** afxcaptionbutton.h
