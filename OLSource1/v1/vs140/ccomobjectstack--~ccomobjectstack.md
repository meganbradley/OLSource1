# CComObjectStack::~CComObjectStack
The destructor.  
  
## Syntax  
  
```  
  
CComObjectStack( );  
  
```  
  
## Remarks  
 Frees all allocated resources and calls [FinalRelease](../vs140/ccomobjectrootex--finalrelease.md).
