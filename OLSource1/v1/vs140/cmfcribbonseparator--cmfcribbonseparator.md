## Syntax

#### Parameters  
 [in] <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 If <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, the separator is horizontal; if <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>, the separator is vertical.  
  
## Remarks  
 Horizontal separators are used in application menus. Vertical separators are used in toolbars.
