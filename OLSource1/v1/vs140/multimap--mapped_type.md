# multimap::mapped_type
A type that represents the data type stored in a multimap.  
  
## Syntax  
  
```  
typedef Type mapped_type;  
```  
  
## Remarks  
 `mapped_type` is a synonym for the template parameter `Type`.  
  
 For more information on `Type` see the [multimap Class](../vs140/multimap-class.md) topic.  
  
## Example  
 See the example for [value_type](../vs140/multimap--value_type.md) for an example of how to declare and use `key_type`.  
  
## Requirements  
 **Header:** \<map>  
  
 **Namespace:** std
