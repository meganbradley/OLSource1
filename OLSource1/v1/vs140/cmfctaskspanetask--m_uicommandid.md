# CMFCTasksPaneTask::m_uiCommandID
Specifies the command ID of the command that is executed when the user clicks the task. If this value is not a valid command ID, the task is treated as a simple label.  
  
## Syntax  
  
```  
UINT m_uiCommandID;  
```

## Requirements
