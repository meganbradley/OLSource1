# CComPtrBase::operator !
The NOT operator.  
  
## Syntax  
  
```  
  
bool operator !( ) const throw( );  
  
```  
  
## Return Value  
 Returns true if the `CComHeapPtr` pointer is equal to NULL, false otherwise.

 **Header:** atlcomcli.h
