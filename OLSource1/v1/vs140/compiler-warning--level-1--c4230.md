# Compiler Warning (level 1) C4230
anachronism used : modifiers/qualifiers interspersed; qualifier ignored  
  
 Using a qualifier before a Microsoft modifier such as <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> is an outdated practice.  
  
## Example  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
