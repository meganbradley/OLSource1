# CRichEditView::GetDocument
Call this function to get a pointer to the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> associated with this view.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Pointer to a [CRichEditDoc](../vs140/cricheditdoc-class.md) object associated with your <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> object.  
  
## Requirements  
 **Header:** afxrich.h
