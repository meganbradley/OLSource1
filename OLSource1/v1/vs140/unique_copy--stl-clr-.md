# unique_copy (STL-CLR)
Copies elements from a source range into a destination range except for the duplicate elements that are adjacent to each other.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This function behaves the same as the STL function <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. For more information, see [unique_copy](../vs140/unique_copy.md).

 **Header:** \<cliext/algorithm>  
  
 **Namespace:** cliext
