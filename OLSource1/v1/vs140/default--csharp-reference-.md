# default (C# Reference)
The `default` keyword.  
  
 The `default` keyword can be used in the `switch` statement or in generic code:  
  
-   [The switch statement](../vs140/switch--csharp-reference-.md): Specifies the default label.  
  
-   [Generic code](../vs140/default-keyword-in-generic-code--csharp-programming-guide-.md): Specifies the default value of the type parameter. This will be null for reference types and zero for value types.
