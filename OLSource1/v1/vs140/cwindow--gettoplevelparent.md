# CWindow::GetTopLevelParent
Retrieves the window's top-level parent window.  
  
## Syntax  
  
```  
  
HWND GetTopLevelParent( ) const throw();  
  
```  
  
## Return Value  
 The handle to the top-level parent window.  
  
## Requirements  
 **Header:** atlwin.h
