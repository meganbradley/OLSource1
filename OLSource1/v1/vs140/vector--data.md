# vector::data
Returns a pointer to the first element in the vector.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 A pointer to the first element in the [vector](../vs140/vector-class.md) or to the location succeeding an empty <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Example  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 **The vector c1 contains elements: 1 2**  
**The vector c1 now contains elements: 20 2**   
## Requirements  
 **Header:** \<vector>  
  
 **Namespace:** std
