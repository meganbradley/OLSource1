# C# Preprocessor Directives
This section contains information about the following C# preprocessor directives.  
  
 [#if](../vs140/sharpif--csharp-reference-.md)  
  
 [#else](../vs140/sharpelse--csharp-reference-.md)  
  
 [#elif](../vs140/sharpelif--csharp-reference-.md)  
  
 [#endif](../vs140/sharpendif--csharp-reference-.md)  
  
 [#define](../vs140/sharpdefine--csharp-reference-.md)  
  
 [#undef](../vs140/sharpundef--csharp-reference-.md)  
  
 [#warning](../vs140/sharpwarning--csharp-reference-.md)  
  
 [#error](../vs140/sharperror--csharp-reference-.md)  
  
 [#line](../vs140/sharpline--csharp-reference-.md)  
  
 [#region](../vs140/sharpregion--csharp-reference-.md)  
  
 [#endregion](../vs140/sharpendregion--csharp-reference-.md)  
  
 [#pragma](../vs140/sharppragma--csharp-reference-.md)  
  
 [#pragma warning](../vs140/sharppragma-warning--csharp-reference-.md)  
  
 [#pragma checksum](../vs140/sharppragma-checksum--csharp-reference-.md)  
  
 See the individual topics for more information and examples.  
  
 Although the compiler does not have a separate preprocessor, the directives described in this section are processed as if there were one. They are used to help in conditional compilation. Unlike C and C++ directives, you cannot use these directives to create macros.  
  
 A preprocessor directive must be the only instruction on a line.
