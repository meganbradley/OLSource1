# CMFCColorPickerCtrl::SetSaturation
Changes the saturation of the currently selected color.  
  
## Syntax  
  
```  
void SetSaturation(  
   double Saturation   
);  
```  
  
#### Parameters  
 [in] `Saturation`  
 A saturation value.
