# &#39;Error&#39; statements are not valid in the Immediate window
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statements can only be used in source code.  
  
 **Error ID:** BC30724  
  
### To correct this error  
  
-   Remove <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statements from debugging code.
