# basic_streambuf::pbase
A protected function that returns a pointer to the beginning of the output buffer.  
  
## Syntax  
  
```  
  
char  
_  
type *pbase( ) const;  
  
```  
  
## Return Value  
 A pointer to the beginning of the output buffer.  
  
## Requirements  
 **Header:** <streambuf\>  
  
 **Namespace:** std
