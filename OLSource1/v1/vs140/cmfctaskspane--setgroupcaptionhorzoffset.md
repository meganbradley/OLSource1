# CMFCTasksPane::SetGroupCaptionHorzOffset
Sets the horizontal offset of a group caption.  
  
## Syntax  
  
```  
void SetGroupCaptionHorzOffset(  
    int n = -1  
);  
```  
  
#### Parameters  
 [in] `n`  
 Specifies the horizontal offset of the group caption.

 **Header:** afxTasksPane.h
