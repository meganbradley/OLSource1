# Resource Compiler Fatal Error RC1022
expected '#endif'  
  
 An <CodeContentPlaceHolder>0\</CodeContentPlaceHolder>, **#ifdef**, or **#ifndef** directive was not terminated with an <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> directive.  
  
 Make sure that there is an <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, **#ifdef**, or **#ifndef** statement in effect before this statement.
