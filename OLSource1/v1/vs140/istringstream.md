# istringstream
Creates a type `basic_istringstream` specialized on a `char` template parameter.  
  
## Syntax  
  
```  
  
typedef basic  
_  
istringstream<char> istringstream;  
  
```  
  
## Remarks  
 The type is a synonym for template class [basic_istringstream](../vs140/basic_istringstream-class.md), specialized for elements of type `char`*.*  
  
## Requirements  
 **Header:** <sstream\>  
  
 **Namespace:** std
