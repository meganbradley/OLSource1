# CMFCRibbonEdit::IsHighlighted
Indicates whether the [CMFCRibbonEdit Class](../vs140/cmfcribbonedit-class.md) control is highlighted.  
  
## Syntax  
  
```  
virtual BOOL IsHighlighted() const;  
```  
  
## Return Value  
 `TRUE` if the [CMFCRibbonEdit Class](../vs140/cmfcribbonedit-class.md) control is highlighted; otherwise `FALSE`.

 **Header:** afxribbonedit.h
