# CSimpleRow::CSimpleRow
The constructor.  
  
## Syntax  
  
```  
  
      CSimpleRow(  
   DBCOUNTITEM iRowsetCur   
);  
```  
  
#### Parameters  
 `iRowsetCur`  
 [in] Index to the current rowset.  
  
## Remarks  
 Sets [m_iRowset](../vs140/csimplerow--m_irowset.md) to `iRowsetCur`.  
  
## Requirements  
 **Header:** atldb.h
