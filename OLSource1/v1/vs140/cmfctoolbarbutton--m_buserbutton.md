# CMFCToolBarButton::m_bUserButton
Specifies whether the button has a user-defined image  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This data member is set to <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> when the button has a user-defined image associated with it.

 **Header:** afxtoolbarbutton.h
