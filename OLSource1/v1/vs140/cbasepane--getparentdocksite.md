# CBasePane::GetParentDockSite
Returns a pointer to the parent dock site.  
  
## Syntax  
  
```  
virtual CDockSite* GetParentDockSite() const;  
```  
  
## Return Value  
 The parent dock site.

 **Header:** afxbasepane.h
