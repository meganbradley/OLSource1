# CMFCCaptionButton::GetSize
Returns the width and height of the button.  
  
## Syntax  
  
```  
static CSize GetSize();  
```  
  
## Return Value  
 The outer dimensions of the button.  
  
## Remarks  
 The size returned includes button margin and border.  
  
## Requirements  
 **Header:** afxcaptionbutton.h
