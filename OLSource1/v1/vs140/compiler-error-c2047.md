# Compiler Error C2047
illegal default  
  
 The keyword <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> can appear only in a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement.  
  
 The following sample generates C2047:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 Possible resolution:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
