# Compiler Error C2348
'type name' : is not a C-style aggregate, cannot be exported in embedded-IDL  
  
 To place a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> in a .idl file with the [export](../vs140/export.md) attribute, the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> must contain only data.  
  
 The following sample generates C2348:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
