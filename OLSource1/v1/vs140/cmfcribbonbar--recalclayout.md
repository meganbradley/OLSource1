# CMFCRibbonBar::RecalcLayout
Adjusts the layout of all controls on the ribbon bar.  
  
## Syntax  
  
```  
virtual void RecalcLayout();  
```  
  
## Remarks  
 After layout adjustment, the display of the ribbon bar is updated.  
  
## Requirements  
 **Header:** afxribbonbar.h
