# Compiler Warning (level 1) C4006
\#undef expected an identifier  
  
 The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> directive did not specify an identifier to undefine. The directive is ignored. To resolve the warning, be sure to specify an identifier. The following sample generates C4006:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
