# &#39;Let&#39; and &#39;Set&#39; assignment statements are no longer supported
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> cannot be used in assignments.  
  
 **Error ID:** BC30807  
  
### To correct this error  
  
-   Omit <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> from assignment statements; for example:  
  
     <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>
