# CDHtmlDialog::SetExternalDispatch
Sets the host's `IDispatch` interface.  
  
## Syntax  
  
```  
  
      void SetExternalDispatch(  
   IDispatch * pdispExternal   
);  
```  
  
#### Parameters  
 *pdispExternal*  
 The new `IDispatch` interface.  
  
## Requirements  
 **Header:** afxdhtml.h
