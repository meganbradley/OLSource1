# CComUnkArray::CComUnkArray
The constructor.  
  
## Syntax  
  
```  
  
CComUnkArray( );  
```  
  
## Remarks  
 Sets the collection to hold `nMaxSize` **IUnknown** pointers, and initializes the pointers to **NULL**.

 **Header:** atlcom.h
