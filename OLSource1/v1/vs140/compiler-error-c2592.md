# Compiler Error C2592
'class': 'base_class_2' is inherited from 'base_class_1' and cannot be re-specified  
  
 You can only specify base classes that do not inherit from other base classes. In this case, only <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> is needed in the specification of <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> because <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> already inherits <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.
