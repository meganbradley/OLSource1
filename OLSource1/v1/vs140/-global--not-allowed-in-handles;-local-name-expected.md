# &#39;Global&#39; not allowed in handles; local name expected
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> clause must refer to a local event. The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> keyword provides access to global programming elements.  
  
 **Error ID:** BC36002  
  
### To correct this error  
  
-   Change the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> clause to refer to a local instance of the event instead of the global instance.
