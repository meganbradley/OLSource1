# CMFCColorBar::AllowChangeTextLabels
Indicates whether the text label of color buttons can change.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Always <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
## Remarks  
 By default, this method always returns <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, which means text labels cannot be modified. Override this method to enable modifying text labels.  
  
## Requirements  
 **Header:** afxcolorbar.h
