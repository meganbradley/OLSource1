# CMFCToolBarButton::m_strText
The text label of the button.  
  
## Syntax  
  
```  
CString m_strText;  
```  
  
## Remarks  
 This data member contains the text label of the button. The text label can be empty.  
  
## Requirements  
 **Header:** afxtoolbarbutton.h
