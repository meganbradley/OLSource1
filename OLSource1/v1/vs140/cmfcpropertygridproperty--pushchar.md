# CMFCPropertyGridProperty::PushChar
Called from the property list control when the property is selected and the user enters a new character.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>

 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 A character.

 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> if the edit operation is continuing; otherwise, <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.
