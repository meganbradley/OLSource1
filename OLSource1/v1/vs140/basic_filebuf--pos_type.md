# basic_filebuf::pos_type
Makes this type within basic_filebuf's scope equivalent to the type of the same name in the **Tr** scope.  
  
## Syntax  
  
```  
  
typedef typename traits  
_  
type::pos  
_  
type pos  
_  
type;  
  
```  
  
## Requirements  
 **Header:** <fstream\>  
  
 **Namespace:** std
