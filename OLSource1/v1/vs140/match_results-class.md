# match_results Class
Holds a sequence of submatches.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>53\</CodeContentPlaceHolder>  
 The iterator type for submatches.  
  
 <CodeContentPlaceHolder>54\</CodeContentPlaceHolder>  
 The type of an allocator for managing storage.  
  
## Remarks  
 The template class describes an object that controls a non-modifiable sequence of elements of type <CodeContentPlaceHolder>55\</CodeContentPlaceHolder> generated by a regular expression search. Each element points to the subsequence that matched the capture group corresponding to that element.
