# Sids::AnonymousLogon
Returns the SECURITY_ANONYMOUS_LOGON_RID SID.  
  
> [!IMPORTANT]
>  This function cannot be used in applications that execute in the [!INCLUDE[wrt](../vs140/includes/wrt_md.md)].  
  
## Syntax  
  
```  
  
CSid AnonymousLogon( ) throw(...);  
  
```
