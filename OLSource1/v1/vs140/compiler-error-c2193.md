# Compiler Error C2193
'identifier' : already in a segment  
  
 A function was placed in two different segments using <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> pragmas.  
  
 The following sample generates C2193:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
