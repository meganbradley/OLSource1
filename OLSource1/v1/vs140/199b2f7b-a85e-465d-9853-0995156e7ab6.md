# &#39;AddHandler&#39; and &#39;RemoveHandler&#39; method parameters must have the same delegate type as the containing event
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> declaration must have <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> declarations, each of which takes a single parameter of the delegate type specified by the custom event's <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> clause.  
  
 **Error ID:** BC31136  
  
### To correct this error  
  
-   Change the type of the parameter to be the same as the delegate type specified by the custom event's <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> clause.
