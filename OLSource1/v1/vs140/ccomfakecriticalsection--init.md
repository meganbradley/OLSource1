# CComFakeCriticalSection::Init
Does nothing since there is no critical section.  
  
## Syntax  
  
```  
  
HRESULT Init( ) throw( );  
  
```  
  
## Return Value  
 Returns S_OK.

 **Header:** atlcore.h
