# Compiler Error CS0167
The delegate 'delegate' is missing the Invoke method  
  
 You imported and used a managed program (one that uses the .NET Framework common language runtime) that was created with another compiler. That compiler allowed an ill-formed [delegate](../vs140/delegate--csharp-reference-.md). Therefore, the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> method was not available. For more information, see [Delegates (C# Programmer's Reference)](../vs140/delegates--csharp-programming-guide-.md).
