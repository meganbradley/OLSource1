# My.Computer.Ports Object
Provides a property and a method for accessing the computer's serial ports.  
  
## Remarks  
 For information about the methods and properties of the `My.Computer.Ports` object, see <xref:Microsoft.VisualBasic.Devices.Ports?qualifyHint=False>.  
  
 For more information, see [Accessing the Computer's Ports (Visual Basic)](../vs140/accessing-the-computer-s-ports--visual-basic-.md).  
  
> [!NOTE]
>  You can also use properties and methods of the <xref:System.IO.Ports.SerialPort?qualifyHint=True> class to access the computer's serial ports.
