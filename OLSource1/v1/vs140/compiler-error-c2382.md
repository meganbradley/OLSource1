# Compiler Error C2382
'function' : redefinition; different exception specifications  
  
 Under [/Za](../vs140/-za---ze--disable-language-extensions-.md), this error indicates that a function overload was attempted only on the [exception specification](../vs140/exception-specifications--throw---c---.md).
