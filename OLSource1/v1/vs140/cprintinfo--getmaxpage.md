# CPrintInfo::GetMaxPage
Call this function to retrieve the number of the last page of the document.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 The number of the last page of the document.  
  
## Remarks  
 This value is stored in the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object referenced by the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> member.  
  
## Requirements  
 **Header:** afxext.h
