# list::swap
Exchanges the elements of two lists.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 The list providing the elements to be swapped, or the list whose elements are to be exchanged with those of the list <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.  
  
 <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>  
 A list whose elements are to be exchanged with those of the list <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>.  
  
## Example  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 **The original list c1 is: 1 2 3**  
**After swapping with c2, list c1 is: 10 20**  
**After swapping with c3, list c1 is: 100**
