# CSpinButtonCtrl::SetPos
Sets the current position for a spin button control.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 New position for the control. This value must be in the range specified by the upper and lower limits for the control.  
  
## Return Value  
 The previous position (16-bit precision for <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, 32-bit precision for <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>).  
  
## Remarks  
 <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> sets the 32-bit position.  
  
## Requirements  
 **Header:** afxcmn.h
