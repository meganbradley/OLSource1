# WORD
Allocates and optionally initializes a word (2 bytes) of storage for each <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. Can also be used as a type specifier anywhere a type is legal.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
