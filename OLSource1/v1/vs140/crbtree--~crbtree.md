# CRBTree::~CRBTree
The destructor.  
  
## Syntax  
  
```  
  
~CRBTree( ) throw( );  
  
```  
  
## Remarks  
 Frees any allocated resources. Calls [CRBTree::RemoveAll](../vs140/crbtree--removeall.md) to delete all elements.  
  
## Requirements  
 **Header:** atlcoll.h
