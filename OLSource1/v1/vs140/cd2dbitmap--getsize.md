# CD2DBitmap::GetSize
Returns the size, in device-independent pixels (DIPs), of the bitmap  
  
## Syntax  
  
```  
CD2DSizeF GetSize() const;  
```  
  
## Return Value  
 The size, in DIPs, of the bitmap.  
  
## Requirements  
 **Header:** afxrendertarget.h
