# Compiler Error CS0674
Do not use 'System.ParamArrayAttribute'. Use the 'params' keyword instead.  
  
 The C# compiler does not allow for the use of \<xref:System.ParamArrayAttribute*?displayProperty=fullName>; use [params](../vs140/params--csharp-reference-.md) instead.
