# CMFCShellListCtrl::OnSetColumns
The framework calls this method when it sets the names of the columns.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 By default, the framework creates four columns in a [CMFCShellListCtrl](../vs140/cmfcshelllistctrl-class.md) object. The names of these columns are <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, and <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>. You can override this method to customize the number of columns and their names.
