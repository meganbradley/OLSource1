# CD2DSolidColorBrush::Get
Returns ID2D1SolidColorBrush interface  
  
## Syntax  
  
```  
ID2D1SolidColorBrush* Get();  
```  
  
## Return Value  
 Pointer to an ID2D1SolidColorBrush interface or NULL if object is not initialized yet.  
  
## Requirements  
 **Header:** afxrendertarget.h
