# CRichEditView::GetTextLength
Call this function to retrieve the length of the text in this `CRichEditView` object.  
  
## Syntax  
  
```  
  
long GetTextLength( ) const;  
  
```  
  
## Return Value  
 The length of the text in this `CRichEditView` object.  
  
## Requirements  
 **Header:** afxrich.h
