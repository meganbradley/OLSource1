# CNoWorkerThread::GetThreadId
Non-functional equivalent of [CWorkerThread::GetThreadId](../vs140/cworkerthread--getthreadid.md).  
  
## Syntax  
  
```  
  
DWORD GetThreadId( ) throw( );  
  
```  
  
## Return Value  
 Always returns 0.  
  
## Remarks  
 The implementation provided by this class does nothing.  
  
## Requirements  
 **Header:** atlutil.h
