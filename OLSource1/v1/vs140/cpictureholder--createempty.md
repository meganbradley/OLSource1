# CPictureHolder::CreateEmpty
Creates an empty `CPictureHolder` object and connects it to an `IPicture` interface.  
  
## Syntax  
  
```  
  
BOOL CreateEmpty( );  
```  
  
## Return Value  
 Nonzero if the object is successfully created; otherwise 0.  
  
## Requirements  
 **Header:** afxwin.h
