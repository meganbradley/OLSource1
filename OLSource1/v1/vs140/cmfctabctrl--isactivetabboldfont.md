# CMFCTabCtrl::IsActiveTabBoldFont
Indicates whether the text of the active tab is displayed using a bold font.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the active tab is displayed using the bold font; otherwise, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Remarks  
 Use the [CMFCTabCtrl::SetActiveTabBoldFont](../vs140/cmfctabctrl--setactivetabboldfont.md) method to change the active tab font.  
  
## Requirements  
 **Header:** afxtabctrl.h
