# accelerator::get_default_view Method
Returns the default <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object that is associated with the [accelerator](../vs140/accelerator-class.md).  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>

 The default <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> object that is associated with the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.
