# vector::size
Returns the number of elements in the vector.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 The current length of the vector.  
  
## Example  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 **Vector length is 1.**  
**Vector length is now 2.**   
## Requirements  
 **Header:** \<vector>  
  
 **Namespace:** std
