# Compiler Error C2776
only one 'get' method can be specified per property  
  
 You can only specify one <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> function in the [property](../vs140/property--c---.md) extended attribute. This error occurs when multiple <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> functions are specified.  
  
 The following sample generates C2776:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
