# CDrawingManager::SetPixel
Changes a single pixel in a bitmap to the specified color.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
  
|||  
|-|-|  
|Parameter|Description|  
|[in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>|A pointer to the bit values of the bitmap.|  
|[in] <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>|The total width of the bitmap.|  
|[in] <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>|The total height of the bitmap.|  
|[in] <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>|The x-coordinate of the pixel in the bitmap to change.|  
|[in] <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>|The y-coordinate of the pixel in the bitmap to change.|  
|[in] <CodeContentPlaceHolder>6\</CodeContentPlaceHolder>|The new color for the pixel identified by the supplied coordinates.|
