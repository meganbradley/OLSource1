# CTokenGroups::Delete
Deletes a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> and its associated attributes from the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> object.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 The [CSid](../vs140/csid-class.md) object for which the security identifier (SID) and attributes should be removed.  
  
## Return Value  
 Returns true if the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> is removed, false otherwise.
