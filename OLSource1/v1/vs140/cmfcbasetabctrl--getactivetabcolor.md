# CMFCBaseTabCtrl::GetActiveTabColor
Retrieves the background color of the currently active tab.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 A [COLORREF](http://msdn.microsoft.com/library/windows/desktop/dd183449) value that specifies the background color of the active tab.  
  
## Remarks  
 By default, the background color of the active tab is <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. You can change the background color for the active tab by using the method [CMFCBaseTabCtrl::SetActiveTabColor](../vs140/cmfcbasetabctrl--setactivetabcolor.md).

 **Header:** afxbasetabctrl.h
