# Vbc Task
Wraps vbc.exe, which produces executables (.exe), dynamic-link libraries (.dll), or code modules (.netmodule). For more information on vbc.exe, see [Visual Basic Compiler](../vs140/visual-basic-command-line-compiler.md).
