# &#39;&lt;specifier&gt;&#39; is not valid on a WithEvents declaration
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement contains a keyword, such as <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, that is invalid in combination with the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> keyword.  
  
 **Error ID:** BC30234  
  
### To correct this error  
  
1.  Remove the invalid keyword from the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement.
