# CMFCRibbonCategory::IsVisible
Indicates whether the ribbon category is visible.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the ribbon category is visible; otherwise <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Remarks  
 Ribbon categories that are visible display a category tab.

 **Header:** afxribboncategory.h
