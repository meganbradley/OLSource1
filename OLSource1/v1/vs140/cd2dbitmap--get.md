# CD2DBitmap::Get
Returns ID2D1Bitmap interface  
  
## Syntax  
  
```  
ID2D1Bitmap* Get();  
```  
  
## Return Value  
 Pointer to an ID2D1Bitmap interface or NULL if object is not initialized yet.
