# CTreeCtrl::SetIndent
Call this function to set the width of indentation for a tree view control and redraw the control to reflect the new width.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 Width, in pixels, of the indentation. If <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> is less than the system-defined minimum width, the new width is set to the system-defined minimum.

 See the example for [CTreeCtrl::GetIndent](../vs140/ctreectrl--getindent.md).
