# CSimpleMap::GetValueAt
Retrieves the value at the specific index.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 The index of the value to return.  
  
## Return Value  
 Returns the value referenced by <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Remarks  
 The index passed by <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> must be valid for the return value to be meaningful.
