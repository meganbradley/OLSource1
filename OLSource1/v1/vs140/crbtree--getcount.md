# CRBTree::GetCount
Call this method to get the number of elements in the tree.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Returns the number of elements (each key/value pair is one element) stored in the tree.
