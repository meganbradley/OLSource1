# &#39;By&#39; expected
An <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> clause has been specified without the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> keyword.  
  
 **Error ID:** BC36605  
  
### To correct this error  
  
1.  Add the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> keyword to the <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> clause. The following is an example:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
