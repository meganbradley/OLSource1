# CAdapt::operator T&amp;
Returns a reference to the [m_T](../vs140/cadapt--m_t.md) member, allowing the adapter object to be treated as if it were an object of type `T`.  
  
## Syntax  
  
```  
  
operator T&( );  
  
```  
  
## Return Value  
 A reference to `m_T`.  
  
## Requirements  
 **Header:** atlcomcli.h
