# CMFCRibbonGallery::SetItemToolTip
Sets the tooltip text for an item in the gallery.  
  
## Syntax  
  
```  
void SetItemToolTip(  
    int nItemIndex,  
    LPCTSTR lpszToolTip   
);  
```  
  
#### Parameters  
 [in] `nItemIndex`  
 The zero-based index of the palette item with which to associate the tooltip.  
  
 [in] `lpszToolTip`  
 The text to appear on the tooltip.
