# CWindow::SetScrollInfo
Sets the parameters of a scroll bar.  
  
## Syntax  
  
```  
  
      int SetScrollInfo(  
   int nBar,  
   LPSCROLLINFO lpScrollInfo,  
   BOOL bRedraw = TRUE   
) throw();  
```
