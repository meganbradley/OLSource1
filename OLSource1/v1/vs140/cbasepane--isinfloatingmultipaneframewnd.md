# CBasePane::IsInFloatingMultiPaneFrameWnd
Specifies whether the pane is in a multi-pane frame window ([CMultiPaneFrameWnd Class](../vs140/cmultipaneframewnd-class.md)).  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the pane is in a multi-pane frame window; otherwise, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Remarks  
 Only dockable panes can float in a multi-pane frame window. Therefore, <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> always returns <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.
