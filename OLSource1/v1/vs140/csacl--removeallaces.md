# CSacl::RemoveAllAces
Removes all of the access-control entries (ACEs) contained in the `CSacl` object.  
  
## Syntax  
  
```  
  
void RemoveAllAces( ) throw( );  
  
```  
  
## Remarks  
 Removes every **ACE** structure (if any) in the `CSacl` object.  
  
## Requirements  
 **Header:** atlsecurity.h
