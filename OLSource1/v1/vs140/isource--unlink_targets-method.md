# ISource::unlink_targets Method
When overridden in a derived class, unlinks all target blocks from this `ISource` block.  
  
## Syntax  
  
```  
virtual void unlink_targets() = 0;  
```  
  
## Requirements  
 **Header:** agents.h  
  
 **Namespace:** concurrency
