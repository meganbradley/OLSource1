# CWnd::GetWindowedChildCount
Call this member function to retrieve the number of associated child windows.  
  
## Syntax  
  
```  
  
long GetWindowedChildCount( );  
  
```  
  
## Return Value  
 The number of child windows associated with the `CWnd` object.  
  
## Requirements  
 **Header:** afxwin.h
