# CMFCRibbonPanel::IsCenterColumnVert
Indicates whether the vertical positions of ribbon elements are centered within their display rectangle.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the vertical positions of ribbon elements are centered within their display rectangle; otherwise <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.
