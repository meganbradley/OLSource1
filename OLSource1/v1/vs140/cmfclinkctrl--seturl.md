# CMFCLinkCtrl::SetURL
Displays a specified URL as the button text.  
  
## Syntax  
  
```  
void SetURL(  
   LPCTSTR lpszURL   
);  
```  
  
#### Parameters  
 [in] `lpszURL`  
 The button text to display.
