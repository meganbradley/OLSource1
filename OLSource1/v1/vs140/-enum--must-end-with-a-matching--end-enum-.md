# &#39;Enum&#39; must end with a matching &#39;End Enum&#39;
An <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement occurs without a corresponding <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement. An <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement must be used to end the enumeration.  
  
 **Error ID:** BC30185  
  
### To correct this error  
  
1.  Check that the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> members are correctly formatted.  
  
2.  Add an <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> statement to the end of the enumeration.
