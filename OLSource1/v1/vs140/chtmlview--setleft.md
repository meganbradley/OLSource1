# CHtmlView::SetLeft
Sets the horizontal position of the Internet Explorer main window.  
  
## Syntax  
  
```  
  
      void SetLeft(  
   long nNewValue   
);  
```  
  
#### Parameters  
 `nNewValue`  
 The screen coordinate of the left edge of the main window.  
  
## Requirements  
 **Header:** afxhtml.h
