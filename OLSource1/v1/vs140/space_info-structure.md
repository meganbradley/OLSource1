# space_info Structure
Holds information about a volume.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Members  
  
### Public Data Members  
  
|Name|Description|  
|----------|-----------------|  
|<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>|Represents the number of bytes that are available to represent data on the volume.|  
|<CodeContentPlaceHolder>2\</CodeContentPlaceHolder>|Represents the total number of bytes that the volume can represent.|  
|<CodeContentPlaceHolder>3\</CodeContentPlaceHolder>|Represents the number of bytes that are not used to represent data on the volume.|  
  
## Requirements  
 **Header:** filesystem  
  
 **Namespace:** std::tr2::sys
