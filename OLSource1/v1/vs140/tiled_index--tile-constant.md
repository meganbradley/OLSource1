# tiled_index::tile Constant
Stores an [index](../vs140/index-class.md) object of rank 1, 2, or 3 that represents the coordinates of the current tile of a [tiled_extent](../vs140/tiled_extent-class.md) object.  
  
## Syntax  
  
```  
const index<rank> tile;  
```  
  
## Requirements  
 **Header:** amp.h  
  
 **Namespace:** Concurrency
