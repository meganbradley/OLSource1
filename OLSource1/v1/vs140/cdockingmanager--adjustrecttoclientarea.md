Adjusts the alignment of a rectangle.

<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>

 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 A reference to a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> object  
  
 [in] <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 The alignment of the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> object  
  
## Return Value  
 <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> if the alignment of the <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> object was adjusted; <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> otherwise.  
  
## Remarks  
 The <CodeContentPlaceHolder>8\</CodeContentPlaceHolder> parameter can have one of the following values:  
  
-   CBRS_ALIGN_TOP  
  
-   CBRS_ALIGN_BOTTOM  
  
-   CBRS_ALIGN_LEFT  
  
-   CBRS_ALIGN_RIGHT
