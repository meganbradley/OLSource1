# CMFCRibbonSlider::SetZoomIncrement
Set the zoom increment for the slider control.  
  
## Syntax  
  
```  
void SetZoomIncrement(  
   int nZoomIncrement   
);  
```  
  
#### Parameters  
 [in] `nZoomIncrement`  
 Specifies the zoom increment of the slider control.  
  
## Requirements  
 **Header:** afxribbonslider.h
