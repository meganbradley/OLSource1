# Property parameters cannot have the name &#39;Value&#39;
A parameter in a <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement is named <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. This name is reserved within a property definition for the new value being assigned to the property in the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> procedure.  
  
 **Error ID:** BC32201  
  
### To correct this error  
  
-   Change the name of the parameter in the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement.
