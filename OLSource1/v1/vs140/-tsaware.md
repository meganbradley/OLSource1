# -TSAWARE
```  
/TSAWARE[:NO]  
```  
  
## Remarks  
 The /TSAWARE option to the EDITBIN utility allows you to modify a program image the same way as if you had used the [/TSAWARE](../vs140/-tsaware--create-terminal-server-aware-application-.md) linker option.
