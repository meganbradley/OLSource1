# Comma or &#39;)&#39; expected (Procedure Parameter List)
A parameter list contains an unexpected element. Use commas (<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>) to separate parameters in such a list, and use a closing parenthesis (<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>) to end the list.  
  
 **Error ID:** BC30213  
  
### To correct this error  
  
-   Add a comma and continue the type parameter list, or add a closing parenthesis character to terminate the list.
