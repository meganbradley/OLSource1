# CMFCToolBarImages::m_bDisableTrueColorAlpha
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if truecolor alpha blending (32-bit color) is disabled.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 Set this member variable to <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> to enable truecolor alpha-blending for toolbar images.  
  
 The default value is <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> for backward compatibility.
