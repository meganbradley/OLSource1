# Directories and Files Summary (Visual Basic)
[!INCLUDE[vbprvb](../vs140/includes/vbprvb_md.md)] language keywords and run-time library members are organized by purpose and use.  
  
 Using the `My` feature gives you greater productivity and performance in file I/O operations than using these members. For more information, see <xref:Microsoft.VisualBasic.FileIO.FileSystem*>.  
  
|**Action**|**Language element**|  
|----------------|--------------------------|  
|Change a directory or folder.|[ChDir](assetId:///M:Microsoft.VisualBasic.FileSystem.ChDir(System.String)?qualifyHint=False&autoUpgrade=True)|  
|Change the drive.|[ChDrive](assetId:///M:Microsoft.VisualBasic.FileSystem.ChDrive(System.String)?qualifyHint=False&autoUpgrade=True)|  
|Copy a file.|[FileCopy](assetId:///M:Microsoft.VisualBasic.FileSystem.FileCopy(System.String,System.String)?qualifyHint=False&autoUpgrade=True)|  
|Make a directory or folder.|[MkDir](assetId:///M:Microsoft.VisualBasic.FileSystem.MkDir(System.String)?qualifyHint=False&autoUpgrade=True)|  
|Remove a directory or folder.|[RmDir](assetId:///M:Microsoft.VisualBasic.FileSystem.RmDir(System.String)?qualifyHint=False&autoUpgrade=True)|  
|Rename a file, directory, or folder.|[Rename](assetId:///M:Microsoft.VisualBasic.FileSystem.Rename(System.String,System.String)?qualifyHint=False&autoUpgrade=True)|  
|Return the current path.|<xref:Microsoft.VisualBasic.FileSystem.CurDir*>|  
|Return a file's date/time stamp.|[FileDateTime](assetId:///M:Microsoft.VisualBasic.FileSystem.FileDateTime(System.String)?qualifyHint=False&autoUpgrade=True)|  
|Return file, directory, or label attributes.|[GetAttr](assetId:///M:Microsoft.VisualBasic.FileSystem.GetAttr(System.String)?qualifyHint=False&autoUpgrade=True)|  
|Return a file's length.|[FileLen](assetId:///M:Microsoft.VisualBasic.FileSystem.FileLen(System.String)?qualifyHint=False&autoUpgrade=True)|  
|Return a file's name or volume label.|<xref:Microsoft.VisualBasic.FileSystem.Dir*>|  
|Set attribute information for a file.|[SetAttr](assetId:///M:Microsoft.VisualBasic.FileSystem.SetAttr(System.String,Microsoft.VisualBasic.FileAttribute)?qualifyHint=False&autoUpgrade=True)|
