# COleControl::InternalGetText
Accesses the stock Text or Caption property of your control.  
  
## Syntax  
  
```  
  
const CString& InternalGetText( );  
```  
  
## Return Value  
 A reference to the control text string.  
  
## Requirements  
 **Header:** afxctl.h
