# CWinTraits::GetWndExStyle
Call this function to retrieve the extended styles of the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 Extended styles used for creation of a window. If <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> is 0, the template style values (<CodeContentPlaceHolder>4\</CodeContentPlaceHolder>) are returned. If <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> is nonzero, <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> is returned.  
  
## Return Value  
 The extended window styles of the object.
