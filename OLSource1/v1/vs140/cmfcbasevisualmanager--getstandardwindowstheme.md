# CMFCBaseVisualManager::GetStandardWindowsTheme
Gets the current Windows theme.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 The currently selected Windows Theme color. Can be one of the following enumerated values:  
  
-   <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> - there is no theme enabled.  
  
-   <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> - non standard theme is selected (meaning a theme is selected, but none from the list below).  
  
-   <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> - blue theme (Luna).  
  
-   <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> - olive theme.  
  
-   <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> - silver theme.

 **Header:** afxvisualmanager.h
