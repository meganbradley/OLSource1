# Character Testing
**ANSI 4.3.1** The sets of characters tested for by the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>, and <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> functions  
  
 The following list describes these functions as they are implemented by the Microsoft C compiler.  
  
|Function|Tests For|  
|--------------|---------------|  
|<CodeContentPlaceHolder>6\</CodeContentPlaceHolder>|Characters 0 – 9, A–Z, a–z ASCII 48–57, 65–90, 97–122|  
|<CodeContentPlaceHolder>7\</CodeContentPlaceHolder>|Characters A–Z, a–z ASCII 65–90, 97–122|  
|<CodeContentPlaceHolder>8\</CodeContentPlaceHolder>|ASCII 0 –31, 127|  
|<CodeContentPlaceHolder>9\</CodeContentPlaceHolder>|Characters a–z ASCII 97–122|  
|<CodeContentPlaceHolder>10\</CodeContentPlaceHolder>|Characters A–Z, a–z, 0 – 9, punctuation, space ASCII 32–126|  
|<CodeContentPlaceHolder>11\</CodeContentPlaceHolder>|Characters A–Z ASCII 65–90|
