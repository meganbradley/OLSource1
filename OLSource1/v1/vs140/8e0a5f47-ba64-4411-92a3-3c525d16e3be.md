# _ismbcgraph, _ismbcgraph_l, _ismbcprint, _ismbcprint_l, _ismbcpunct, _ismbcpunct_l, _ismbcblank, _ismbcblank_l, _ismbcspace, _ismbcspace_l
Determines whether character is a graphical character, a display character, a punctuation character, or a space character.  
  
> [!IMPORTANT]
>  This API cannot be used in applications that execute in the [!INCLUDE[wrt](../vs140/includes/wrt_md.md)]. For more information, see[CRT functions not supported with /ZW](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx).  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 Character to be determined.  
  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 Locale to use.  
  
## Return Value  
 Each of these routines returns a nonzero value if the character satisfies the test condition, or 0 if it does not. If <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> <= 255 and there is a corresponding <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> routine (for example, <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> corresponds to <CodeContentPlaceHolder>6\</CodeContentPlaceHolder>), the result is the return value of the corresponding <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> routine.  
  
 The versions of these functions are identical, except that the ones that have the <CodeContentPlaceHolder>8\</CodeContentPlaceHolder> suffix use the locale that's passed in for their locale-dependent behavior, instead of the current locale. For more information, see [Locale](../vs140/locale.md).  
  
## Remarks  
 Each of these functions tests a given multibyte character for a given condition.  
  
|Routine|Test condition|Code page 932 example|  
|-------------|--------------------|---------------------------|  
|<CodeContentPlaceHolder>9\</CodeContentPlaceHolder>|Graphic|Returns nonzero if and only if <CodeContentPlaceHolder>10\</CodeContentPlaceHolder> is a single-byte representation of any ASCII or katakana printable character except a white space ( ).|  
|<CodeContentPlaceHolder>11\</CodeContentPlaceHolder>|Printable|Returns nonzero if and only if <CodeContentPlaceHolder>12\</CodeContentPlaceHolder> is a single-byte representation of any ASCII or katakana printable character including a white space ( ).|  
|<CodeContentPlaceHolder>13\</CodeContentPlaceHolder>|Punctuation|Returns nonzero if and only if <CodeContentPlaceHolder>14\</CodeContentPlaceHolder> is a single-byte representation of any ASCII or katakana punctuation character.|  
|<CodeContentPlaceHolder>15\</CodeContentPlaceHolder>|Space or horizontal tab|Returns nonzero if and only if <CodeContentPlaceHolder>16\</CodeContentPlaceHolder> is a space or horizontal tab character: <CodeContentPlaceHolder>17\</CodeContentPlaceHolder>=0x20 or <CodeContentPlaceHolder>18\</CodeContentPlaceHolder>=0x09.|  
|<CodeContentPlaceHolder>19\</CodeContentPlaceHolder>|White space|Returns nonzero if and only if <CodeContentPlaceHolder>20\</CodeContentPlaceHolder> is a white-space character: <CodeContentPlaceHolder>21\</CodeContentPlaceHolder>=0x20 or 0x09<=<CodeContentPlaceHolder>22\</CodeContentPlaceHolder><=0x0D.|
