# Compiler Error CS0214
Pointers and fixed size buffers may only be used in an unsafe context  
  
 Pointers can only be used with the [unsafe](../vs140/unsafe--csharp-reference-.md) keyword. For more information, see [Unsafe Code and Pointers (C# Programmer's Reference)](../vs140/unsafe-code-and-pointers--csharp-programming-guide-.md).
