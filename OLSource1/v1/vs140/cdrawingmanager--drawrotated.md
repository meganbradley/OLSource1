# CDrawingManager::DrawRotated
Rotates a source DC content inside the given rectangle by 90 degrees.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 Destination rectangle.  
  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 The source device context.  
  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> indicates rotate +90 degrees; <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> indicates rotate -90 degrees.
