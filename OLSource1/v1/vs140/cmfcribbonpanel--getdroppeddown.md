# CMFCRibbonPanel::GetDroppedDown
Retrieves a pointer to a ribbon element if its pop-up menu is dropped down.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Pointer to the ribbon element that has its pop-up menu dropped down; otherwise <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if no ribbon element has its pop-up menu dropped down.  
  
## Remarks  
 Only ribbon elements that are contained in the ribbon panel are tested.  
  
## Requirements  
 **Header:** afxribbonpanel.h
