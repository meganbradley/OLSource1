# CMFCToolBar::IsUserDefined
Specifies whether the toolbar is user-defined.  
  
## Syntax  
  
```  
BOOL IsUserDefined() const;  
```  
  
## Return Value  
 `TRUE` if the toolbar was created by the user; otherwise `FALSE`.  
  
## Requirements  
 **Header:** afxtoolbar.h
