# &#39;Dir&#39; function must first be called with a &#39;PathName&#39; argument
An initial call to the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> function does not include the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> argument. The first call to <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> must include a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, but subsequent calls to <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> do not need to include parameters to retrieve the next item.  
  
### To correct this error  
  
1.  Supply a <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> argument in the function call.
