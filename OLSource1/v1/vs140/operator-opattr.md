# operator OPATTR
Returns a word defining the mode and scope of *expression*. The low byte is identical to the byte returned by [.TYPE](../vs140/operator-.type.md). The high byte contains additional information.  
  
## Syntax  
  
```  
  
OPATTR expression  
```
