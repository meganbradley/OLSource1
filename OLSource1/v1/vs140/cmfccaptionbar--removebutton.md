# CMFCCaptionBar::RemoveButton
Removes the button from the caption bar.  
  
## Syntax  
  
```  
void RemoveButton();  
```  
  
## Remarks  
 The layout of caption bar elements are adjusted automatically.  
  
## Requirements  
 **Header:** afxcaptionbar.h
