# Running Automated Tests Using Microsoft Visual Studio
To run your automated tests locally or use a test controller and test agents, use Visual Studio. Or, if you have to plan your test effort and run tests as part of a test plan, use [!INCLUDE[TCMext](../vs140/includes/tcmext_md.md)], which is included with Visual Studio Enterprise and Visual Studio Test Professional. You can also create and run manual tests with [!INCLUDE[TCMext](../vs140/includes/tcmext_md.md)].
