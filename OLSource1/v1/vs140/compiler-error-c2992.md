# Compiler Error C2992
'class' : invalid or missing type parameter list  
  
 The class is preceded by a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or **generic** keyword with missing or invalid parameters.  
  
## Example  
 The following sample generates C2992:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
