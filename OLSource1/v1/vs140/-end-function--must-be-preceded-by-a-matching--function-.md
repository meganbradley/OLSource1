# &#39;End Function&#39; must be preceded by a matching &#39;Function&#39;
An <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement appears in your code with no matching <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> procedure definition preceding it.  
  
 **Error ID:** BC30430  
  
### To correct this error  
  
1.  Remove the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement if it is redundant.  
  
2.  Supply the missing <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> procedure if one is missing.  
  
3.  Move the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> to the appropriate place in the code.
