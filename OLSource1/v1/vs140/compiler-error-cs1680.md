# Compiler Error CS1680
Invalid reference alias option: 'alias=' -- missing filename.  
  
 This error occurs when you use the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> feature with the **/reference** compiler option without specifying a valid file name.  
  
 The following sample generates CS1680.  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
