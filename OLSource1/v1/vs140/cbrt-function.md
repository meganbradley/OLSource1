# cbrt Function
Computes the real cube root of the argument  
  
## Syntax  
  
```  
inline float cbrt(  
   float _X  
) restrict(amp);  
inline double cbrt(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the real cube root of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
