# -DISASM
```  
/DISASM  
```  
  
## Remarks  
 This option displays disassembly of code sections, using symbols if present in the file.  
  
 /DISASM should only be used on native (not managed) images.  
  
 Only the [/HEADERS](../vs140/-headers.md) DUMPBIN option is available for use on files produced with the [/GL](../vs140/-gl--whole-program-optimization-.md) compiler option.
