# &#39;SyncLock&#39; statements are not valid in the Immediate window
The <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement synchronizes threads and is not permitted in a debugging context.  
  
 **Error ID:** BC30135  
  
### To correct this error  
  
-   Do not issue a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement in the **Immediate** window.
