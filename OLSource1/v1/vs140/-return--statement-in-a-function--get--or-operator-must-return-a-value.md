# &#39;Return&#39; statement in a Function, Get, or Operator must return a value
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statements must be used to return a value to a calling procedure. You cannot use <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statements by themselves to control program flow.  
  
 **Error ID:** BC30654  
  
### To correct this error  
  
1.  Specify a value that the function or procedure can return.  
  
2.  Use the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement to cause the program to exit the current procedure.
