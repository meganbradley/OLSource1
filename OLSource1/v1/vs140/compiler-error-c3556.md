# Compiler Error C3556
'expression': incorrect argument to 'decltype'  
  
 The compiler cannot deduce the type of the expression that is the argument to the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> type specifier. In the following code example, the compiler cannot deduce the type of the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> argument because <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> is overloaded.
