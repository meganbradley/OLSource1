# CFileDialog::AddMenu
Adds a menu to the dialog.  
  
## Syntax  
  
```  
HRESULT AddMenu(  
   DWORD dwIDCtl,  
   const CString& strLabel  
);  
```  
  
#### Parameters  
 `dwIDCtl`  
 The ID of the menu to add.  
  
 `strLabel`  
 The menu name.
