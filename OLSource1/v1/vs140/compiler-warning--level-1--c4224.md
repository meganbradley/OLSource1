# Compiler Warning (level 1) C4224
nonstandard extension used : formal parameter 'identifier' was previously defined as a type  
  
 The identifier was previously used as a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. This causes a warning under ANSI compatibility ([/Za](../vs140/-za---ze--disable-language-extensions-.md)).  
  
## Example  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
