# CMFCAutoHideBar::m_nShowAHWndDelay
The time delay between the moment when the user places the mouse cursor over a [CMFCAutoHideButton Class](../vs140/cmfcautohidebutton-class.md) and the moment when the framework shows the associated window.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 When the user places the mouse cursor over a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, there is a slight delay before the framework displays the associated window. This parameter determines the length of that delay in milliseconds.  
  
## Requirements  
 **Header:** afxautohidebar.h
