# CMFCRibbonBar::GetCategoryIndex
Retrieves the index of the specified ribbon category.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 Pointer to a ribbon category.  
  
## Return Value  
 The zero-based index of a ribbon category specified by <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>; or -1 if the ribbon category is not found.

 **Header:** afxribbonbar.h
