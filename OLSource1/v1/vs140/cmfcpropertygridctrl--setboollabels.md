# CMFCPropertyGridCtrl::SetBoolLabels
Specifies the text of Boolean labels.  
  
## Syntax  
  
```  
void SetBoolLabels(  
   LPCTSTR lpszTrue,  
   LPCTSTR lpszFalse   
);  
```  
  
#### Parameters  
 [in] `lpszTrue`  
 The text string to display for the Boolean value of true.  
  
 [in] `lpszFalse`  
 The text string to display for the Boolean value of false.
