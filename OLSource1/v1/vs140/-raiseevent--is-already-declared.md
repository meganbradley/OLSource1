# &#39;RaiseEvent&#39; is already declared
More than one <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> declaration appeared in a custom event declaration. A <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> declaration declares a procedure used to raise the event.  
  
 **Error ID:** BC31129  
  
### To correct this error  
  
-   Remove the redundant <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement.
