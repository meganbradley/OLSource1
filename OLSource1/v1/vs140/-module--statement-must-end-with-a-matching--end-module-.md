# &#39;Module&#39; statement must end with a matching &#39;End Module&#39;
Module blocks must end with an `End``Module` statement.  
  
 **Error ID:** BC30625  
  
### To correct this error  
  
-   Add an `End Module` statement to the end of the `Module` block.
