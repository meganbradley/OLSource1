# Compiler Error C2135
'bit operator' : illegal bit field operation  
  
 The address-of operator (<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>) cannot be applied to a bit field.  
  
 The following sample generates C2135:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
