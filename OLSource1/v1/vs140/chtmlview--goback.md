# CHtmlView::GoBack
Navigates backward one item in the history list.  
  
## Syntax  
  
```  
  
void GoBack( );  
  
```  
  
## Remarks  
 Applies to Internet Explorer and WebBrowser.  
  
## Requirements  
 **Header:** afxhtml.h
