# Fatal Error C1192
\#using failed on 'file'  
  
 Only a file in the Microsoft Intermediate Language (MSIL) format can be passed to a [#using](../vs140/sharpusing-directive--c---.md) directive. The [/clr](../vs140/-clr--common-language-runtime-compilation-.md) compiler option lets you create an MSIL output file. Other Visual Studio languages also produce MSIL files.
