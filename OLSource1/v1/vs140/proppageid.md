# PROPPAGEID
Adds a property page for use by your OLE control.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 The unique class ID of a property page.  
  
## Remarks  
 All <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> macros must be placed between the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> macros in your control's implementation file.
