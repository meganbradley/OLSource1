# Modifiers (C# Reference)
Modifiers are used to modify declarations of types and type members. This section introduces the C# modifiers.  
  
|Modifier|Purpose|  
|--------------|-------------|  
|[Access Modifiers](../vs140/access-modifiers--csharp-reference-.md)<br /><br /> -   [public](../vs140/public--csharp-reference-.md)<br />-   [private](../vs140/private--csharp-reference-.md)<br />-   [internal](../vs140/internal--csharp-reference-.md)<br />-   [protected](../vs140/protected--csharp-reference-.md)|Specifies the declared accessibility of types and type members.|  
|[abstract](../vs140/abstract--csharp-reference-.md)|Indicates that a class is intended only to be a base class of other classes.|  
|[async](../vs140/async--csharp-reference-.md)|Indicates that the modified method, lambda expression, or anonymous method is asynchronous.|  
|[const](../vs140/const--csharp-reference-.md)|Specifies that the value of the field or the local variable cannot be modified.|  
|[event](../vs140/event--csharp-reference-.md)|Declares an event.|  
|[extern](../vs140/extern--csharp-reference-.md)|Indicates that the method is implemented externally.|  
|[new](../vs140/new--csharp-reference-.md)|Explicitly hides a member inherited from a base class.|  
|[override](../vs140/override--csharp-reference-.md)|Provides a new implementation of a virtual member inherited from a base class.|  
|[partial](../vs140/partial--type---csharp-reference-.md)|Defines partial classes, structs and methods throughout the same assembly.|  
|[readonly](../vs140/readonly--csharp-reference-.md)|Declares a field that can only be assigned values as part of the declaration or in a constructor in the same class.|  
|[sealed](../vs140/sealed--csharp-reference-.md)|Specifies that a class cannot be inherited.|  
|[static](../vs140/static--csharp-reference-.md)|Declares a member that belongs to the type itself instead of to a specific object.|  
|[unsafe](../vs140/unsafe--csharp-reference-.md)|Declares an unsafe context.|  
|[virtual](../vs140/virtual--csharp-reference-.md)|Declares a method or an accessor whose implementation can be changed by an overriding member in a derived class.|  
|[volatile](../vs140/volatile--csharp-reference-.md)|Indicates that a field can be modified in the program by something such as the operating system, the hardware, or a concurrently executing thread.|
