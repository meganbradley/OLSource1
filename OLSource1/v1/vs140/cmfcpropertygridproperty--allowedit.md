# CMFCPropertyGridProperty::AllowEdit
Makes a property either editable or read-only.  
  
## Syntax  
  
```  
void AllowEdit(  
   BOOL bAllow=TRUE   
);  
```  
  
#### Parameters  
 [in] `bAllow`  
 `TRUE` to make the property editable; `FALSE` to make the property read-only. The default value is `TRUE`.
