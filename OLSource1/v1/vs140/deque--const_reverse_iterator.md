# deque::const_reverse_iterator
A type that provides a random-access iterator that can read any **const** element in the deque.

<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 A type <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> cannot modify the value of an element and is used to iterate through the deque in reverse.
