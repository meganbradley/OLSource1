# Compiler Error CS1105
Extension methods must be static.  
  
 Extension methods must be declared as static methods in a non-generic static class.  
  
## Example  
 The following example generates CS1105 because <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> is not static:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
