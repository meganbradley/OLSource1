# CAtlPreviewCtrlImpl::DoPaint
Called by the framework to render the preview.  
  
## Syntax  
  
```  
virtual void DoPaint(  
   HDC hdc  
);  
```  
  
#### Parameters  
 `hdc`  
 A handle to a device context for painting.
