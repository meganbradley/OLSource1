# Compiler Error CS0226
An __arglist expression may only appear inside of a call or new expression.  
  
 The unsupported keyword <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> can only appear in a method call or a new expression.  
  
## Example  
 The following code generates CS0226:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
