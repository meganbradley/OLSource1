# OperatorIntrinsics.PowIntPtr Function (F#)
This is a library intrinsic. Calls to this function may be generated by uses of the generic [pown](../vs140/operators.pown-^t--function--fsharp-.md) operator on values of type `nativeint`.  
  
 **Namespace/Module Path:** Microsoft.FSharp.Core.Operators.OperatorIntrinsics  
  
 **Assembly:** FSharp.Core (in FSharp.Core.dll)  
  
## Syntax  
  
```  
// Signature:  
PowIntPtr : nativeint -> int -> nativeint  
  
// Usage:  
PowIntPtr x n  
```  
  
#### Parameters  
 `x`  
 Type: [nativeint](../vs140/core.nativeint-type-abbreviation--fsharp-.md)  
  
 The value to raise to a power.  
  
 `n`  
 Type: [int](../vs140/core.int-type-abbreviation--fsharp-.md)  
  
 The power to raise the value to.  
  
## Return Value  
 The result of the exponentiation operation.  
  
## Remarks  
 This function is for use by compiled F# code and should not be used directly.
