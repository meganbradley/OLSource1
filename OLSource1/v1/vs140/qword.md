# QWORD
Allocates and optionally initializes 8 bytes of storage for each <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. Also can be used as a type specifier anywhere a type is legal.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
