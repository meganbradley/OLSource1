# CPropExchange::GetVersion
Call this function to retrieve the version number of the control.  
  
## Syntax  
  
```  
  
DWORD GetVersion( );  
```  
  
## Return Value  
 The version number of the control.  
  
## Requirements  
 **Header:** afxctl.h
