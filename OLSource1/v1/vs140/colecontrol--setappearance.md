# COleControl::SetAppearance
Sets the stock Appearance property value of your control.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 *sAppearance*  
 A **short** (<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>) value to be used for the appearance of your control. A value of zero sets the control's appearance to flat and a value of 1 sets the control's appearance to 3D.  
  
## Remarks  
 For more about stock properties, see [ActiveX Controls: Properties](../vs140/mfc-activex-controls--properties.md).  
  
## Requirements  
 **Header:** afxctl.h
