# CMFCRibbonButton::IsRightAlignMenu
Specifies whether the menu is right-aligned.  
  
## Syntax  
  
```  
BOOL IsRightAlignMenu() const;  
```  
  
## Return Value  
 `TRUE` if menu is right-aligned; otherwise `FALSE`.  
  
## Requirements  
 **Header:** afxribbonbutton.h
