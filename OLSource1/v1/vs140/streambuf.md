# streambuf
A specialization of <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> that uses <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> as the template parameters.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks
