# &#39;Option Compare&#39; must be followed by &#39;Text&#39; or &#39;Binary&#39;
An <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement contains an incorrect setting or no setting. The only settings allowed in <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> are <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.  
  
 **Error ID:** BC30207  
  
### To correct this error  
  
1.  Check to see if the setting specifier is misspelled.  
  
2.  Add either <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> to the <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> statement; for example, <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>.
