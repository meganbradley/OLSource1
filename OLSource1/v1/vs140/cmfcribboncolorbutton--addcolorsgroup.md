# CMFCRibbonColorButton::AddColorsGroup
Adds a group of colors to the regular color area.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 The group name.  
  
 [in] <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 The list of colors.  
  
 [in] <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 Controls how the color items are displayed in the group. If <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>, the color items are drawn without a vertical spacing. If <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>, the color items are drawn with a vertical spacing.  
  
## Remarks  
 Use this function to make the color pop-up display several groups of colors. You can control how the colors are displayed in group.
