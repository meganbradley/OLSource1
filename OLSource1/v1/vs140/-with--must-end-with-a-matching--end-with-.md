# &#39;With&#39; must end with a matching &#39;End With&#39;
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement occurs without a corresponding <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement. An <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement must be used to end the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> block.  
  
 **Error ID:** BC30085  
  
### To correct this error  
  
-   If this <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> block is part of a set of nested <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> blocks, make sure each block is properly terminated.  
  
-   Add an <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> statement to the end of the <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> block.
