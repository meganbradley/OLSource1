# &#39;Finally&#39; can only appear once in a &#39;Try&#39; statement
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder> is used to complete a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> block; hence it can only appear once at the end of the block.  
  
 **Error ID:** BC30381  
  
### To correct this error  
  
1.  Locate and remove the redundant <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.
