# CMFCRibbonCategory::SetData
Sets the user-defined data to be associated with the ribbon category.  
  
## Syntax  
  
```  
void SetData(  
   DWORD_PTR dwData   
);  
```  
  
#### Parameters  
 [in] `dwData`  
 The user-defined data.  
  
## Requirements  
 **Header:** afxribboncategory.h
