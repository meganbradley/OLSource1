# CSimpleArray::RemoveAll
Removes all elements from the array.  
  
## Syntax  
  
```  
  
void RemoveAll( );  
  
```  
  
## Remarks  
 Removes all elements currently stored in the array.

 **Header:** atlsimpcoll.h
