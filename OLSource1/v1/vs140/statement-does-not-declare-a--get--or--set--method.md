# Statement does not declare a &#39;Get&#39; or &#39;Set&#39; method

A property is defined as a block of code enclosed within the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> statements. Inside this block, each <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> procedure appears as an internal block enclosed within a declaration statement and an end statement.  
  
 **Error ID:** BC30576  
  
### To correct this error
