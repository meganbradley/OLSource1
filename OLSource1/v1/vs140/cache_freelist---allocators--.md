# CACHE_FREELIST (&lt;allocators&gt;)
Yields `stdext::allocators::cache_freelist<sizeof(Type), max>`.  
  
## Syntax  
  
```  
#define CACHE_FREELIST(max) <cache_class>  
```  
  
## Remarks  
  
## Requirements  
 **Header:** <allocators\>  
  
 **Namespace:** stdext
