# CMFCShellListCtrl::GetCurrentItemIdList
Returns the PIDL of the currently selected item.  
  
## Syntax  
  
```  
LPITEMIDLIST GetCurrentItemIdList() const;  
```  
  
## Return Value  
 The PIDL of the current item.  
  
## Requirements  
 **Header:** afxshelllistctrl.h
