# IDataObjectImpl::FireDataChange
Sends a change notification back to each advise sink that is currently being managed.  
  
## Syntax  
  
```  
  
HRESULT FireDataChange( );  
  
```  
  
## Return Value  
 A standard `HRESULT` value.  
  
## Requirements  
 **Header:** atlctl.h
