# END_COLUMN_MAP
Marks the end of the column map entries.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 It is used with a single accessor on a rowset. The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> macro is completed with the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> macro.  
  
## Example  
 See [BEGIN_COLUMN_MAP](../vs140/begin_column_map.md).  
  
## Requirements  
 **Header:** atldbcli.h
