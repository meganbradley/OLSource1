# Partial methods must be declared &#39;Private&#39;
The access modifier <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> is required in partial method declarations. The following example shows the use of <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> in the method signature and its implementation.  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 **Error ID:** BC31432  
  
### To correct this error  
  
-   Add the keyword <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> before <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> in the signature and implementation declarations.
