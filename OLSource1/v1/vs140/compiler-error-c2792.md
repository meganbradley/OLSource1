# Compiler Error C2792
'super' : this keyword must be followed by '::'  
  
 The only token that can follow the keyword <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> is <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
 The following sample generates C2792:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
