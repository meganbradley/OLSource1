# Compiler Error C2775
'identifier' : no 'get' method is associated with this property  
  
 A data member declared with the [property](../vs140/property--c---.md) extended attribute does not have a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> function specified, but an expression tries to retrieve its value.  
  
 The following sample generates C2775:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
