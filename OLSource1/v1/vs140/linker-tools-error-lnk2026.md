# Linker Tools Error LNK2026
module unsafe for SAFESEH image  
  
 [/SAFESEH](../vs140/-safeseh--image-has-safe-exception-handlers-.md) was specified, but a module was not compatible with the safe exception handling feature. If you want to use this module with **/SAFESEH**, then you will need to recompile the module using the Visual C++ .NET 2003 (or later) compiler.
