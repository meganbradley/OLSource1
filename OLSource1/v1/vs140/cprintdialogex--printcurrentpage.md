# CPrintDialogEx::PrintCurrentPage
Call this function after calling <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> to determine whether to print the current page in the document.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 **TRUE** if **Print Current Page** is selected in the print dialog; otherwise **FALSE**.
