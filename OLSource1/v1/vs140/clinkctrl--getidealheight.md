# CLinkCtrl::GetIdealHeight
Retrieves the ideal height of the link control.  
  
## Syntax  
  
```  
  
int GetIdealHeight( ) const;  
  
```  
  
## Return Value  
 The ideal height of the control, in pixels.  
  
## Remarks  
 This member function implements the behavior of the Win32 message [LM_GETIDEALHEIGHT](http://msdn.microsoft.com/library/windows/desktop/bb760716), as described in the [!INCLUDE[winSDK](../vs140/includes/winsdk_md.md)].  
  
## Requirements  
 **Header:** afxcmn.h
