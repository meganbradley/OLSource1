# tuple_size Class &lt;array&gt;
Wraps the size of an array.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Template Parameters  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 The type of an element.  
  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 The size of the array.  
  
## Remarks  
 This template is a specialization of the template class [tuple_size](../vs140/tuple_size-class--tuple-.md). It has a member <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> that is an integral constant expression whose value is <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>, which is the size of the array.
