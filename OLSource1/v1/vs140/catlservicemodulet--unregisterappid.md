# CAtlServiceModuleT::UnregisterAppId
Removes the service from the registry.  
  
## Syntax  
  
```  
  
HRESULT UnregisterAppId( ) throw( );  
  
```  
  
## Return Value  
 Returns S_OK on success, or an error HRESULT on failure.

 **Header:** atlbase.h
