# CreatorMap::serverName Data Member
Supports the WRL infrastructure and is not intended to be used directly from your code.  
  
## Syntax  
  
```  
const wchar_t* serverName;  
```  
  
## Remarks  
 Stores the server name for the CreatorMap.
