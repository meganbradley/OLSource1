# CComBSTR::operator =
Sets the [m_str](../vs140/ccombstr--m_str.md) member to a copy of <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or to a copy of the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> member of *src*.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 The <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> parameter specifies either an **LPCOLESTR** for Unicode versions or <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> for ANSI versions.  
  
## Example  
 See the example for [CComBSTR::Copy](../vs140/ccombstr--copy.md).
