# Compiler Error C2777
only one 'put' method can be specified per property  
  
 A [property](../vs140/property--c---.md) declspec modifier had more than one <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> property.  
  
 The following sample generates C2777:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
