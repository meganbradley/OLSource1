# Compiler Error CS0620
Indexers cannot have void type  
  
 The return type of an [indexer](../vs140/indexers--csharp-programming-guide-.md) cannot be <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. An indexer must return a value.
