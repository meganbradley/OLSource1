# no_smart_pointers
**C++ Specific**  
  
 Suppresses the creation of smart pointers for all interfaces in the type library.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 By default, when you use <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, you get a smart pointer declaration for all interfaces in the type library. These smart pointers are of type [_com_ptr_t Class](../vs140/_com_ptr_t-class.md).  
  
 **END C++ Specific**
