# CBasePane::IsDialogControl
Specifies whether the pane is a dialog box control.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the pane is a dialog box control; otherwise, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Remarks  
 The framework uses this method to ensure layout consistency for all panes.

 **Header:** afxbasepane.h
