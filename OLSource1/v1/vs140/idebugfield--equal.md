# IDebugField::Equal
This method compares this field with the specified field for equality.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 [in] The field to compare to this one.  
  
## Return Value  
 If the fields are the same, returns <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>. If the fields are different, returns <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> Otherwise, returns an error code.
