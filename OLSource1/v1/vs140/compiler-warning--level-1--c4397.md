# Compiler Warning (level 1) C4397
DefaultCharSetAttribute is ignored  
  
 \<xref:System.Runtime.InteropServices.DefaultCharSetAttribute*> is ignored by the Visual C++ compiler. To specify a character set for the DLL, use the CharSet option of DllImport. For more information, see [Using C++ Interop (Implicit PInvoke)](../vs140/using-c---interop--implicit-pinvoke-.md).
