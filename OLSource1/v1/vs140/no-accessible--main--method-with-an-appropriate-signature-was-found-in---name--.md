# No accessible &#39;Main&#39; method with an appropriate signature was found in &#39;&lt;name&gt;&#39;
Command-line applications must have a <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> defined. <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> must be declared as <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> if it is defined in a class, or as <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> if defined in a module.  
  
 For more information on <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>, see [NIB: Visual Basic Version of Hello, World](assetId:///9d030b60-e148-4366-a462-69532f02294c).  
  
 **Error ID:** BC30737  
  
### To correct this error  
  
-   Define a <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> procedure for your project. Declare it as <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> if and only if you define it inside a class.
