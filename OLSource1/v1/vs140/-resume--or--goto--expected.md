# &#39;Resume&#39; or &#39;GoTo&#39; expected
An <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement does not contain either a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> clause or a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> clause. It must use one clause or the other.  
  
 **Error ID:** BC32019  
  
### To correct this error  
  
-   Add either a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> clause or a <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> clause to the <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> statement.
