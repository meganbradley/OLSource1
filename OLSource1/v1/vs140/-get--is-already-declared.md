# &#39;Get&#39; is already declared
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement appears more than once in a code block. The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement declares a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> property procedure used to retrieve a value of a property.  
  
 **Error ID:** BC30443  
  
### To correct this error  
  
-   Remove the redundant <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement.
