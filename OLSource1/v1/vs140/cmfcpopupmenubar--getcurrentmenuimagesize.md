# CMFCPopupMenuBar::GetCurrentMenuImageSize
Indicates the size of menu-button images.  
  
## Syntax  
  
```  
virtual CSize GetCurrentMenuImageSize( ) const;  
```  
  
## Return Value  
 Returns the size of menu-button images in the toolbar.
