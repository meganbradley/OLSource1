# &#39;ReDim&#39; Preserve operand cannot be Nothing
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement attempts to use the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> keyword to change a dimension of an array that is not the last dimension, but does not supply a valid value for its operand.  
  
### To correct this error  
  
-   Change the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> operand to a valid value.
