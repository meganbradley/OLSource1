# CPaneDivider::m_nDefaultWidth
Specifies the default width, in pixels, of all pane dividers in the application.  
  
## Syntax  
  
```  
AFX_IMPORT_DATA static int m_nDefaultWidth;  
```  
  
## Requirements  
 **Header:** afxPaneDivider.h
