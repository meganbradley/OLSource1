# Go To Line
This dialog box allows you to move to a specific line in the active document. To access this dialog box, open a document for editing, then select **Go To** on the **Edit** menu.  
  
 Line number (1 - <CodeContentPlaceHolder>0\</CodeContentPlaceHolder>)  
 Allows you to enter the number of the line in the active document to which you want to move. The number entered must fall between 1 and <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, the number of lines in the current document.
