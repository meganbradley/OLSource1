# CWinApp::SupportsAutosaveAtInterval
Determines whether the restart manager autosaves open documents at a regular interval.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> indicates the restart manager autosaves open documents; <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> indicates the restart manager does not.  
  
## Requirements  
 **Header:** afxwin.h
