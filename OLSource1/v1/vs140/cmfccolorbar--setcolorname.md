# CMFCColorBar::SetColorName
Sets a new name for a specified color.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 The RGB value of a color.  
  
 [in] <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 The new name for the specified color.  
  
## Remarks  
 This method changes the name of the specified color in all <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> objects in your application.
