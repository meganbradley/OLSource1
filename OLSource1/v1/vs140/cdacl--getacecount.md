# CDacl::GetAceCount
Returns the number of ACEs (access-control entries) in the `CDacl` object.  
  
## Syntax  
  
```  
  
UINT GetAceCount( ) const throw( );  
  
```  
  
## Return Value  
 Returns the number of ACEs contained in the `CDacl` object.  
  
## Requirements  
 **Header:** atlsecurity.h
