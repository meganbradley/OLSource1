# forward_list::pop_front
Deletes the element at the beginning of a forward list.  
  
## Syntax  
  
```  
void pop_front();  
```  
  
## Remarks  
 The first element of the forward list must be non-empty.  
  
 The member function never throws an exception.  
  
## Requirements  
 **Header:** \<forward_list>  
  
 **Namespace:** std
