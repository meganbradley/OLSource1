# CDockingManager::AddHiddenMDITabbedBar
Adds a handle to a bar pane to the list of hidden MDI tabbed bar panes.  
  
## Syntax  
  
```  
void AddHiddenMDITabbedBar(  
   CDockablePane* pBar  
);  
```  
  
#### Parameters  
 [in] `pBar`  
 A pointer to a bar pane  
  
## Requirements  
 **Header:** afxdockingmanager.h
