# CAtlException::m_hr
The `HRESULT` data member.  
  
## Syntax  
  
```  
  
HRESULT m_hr;  
  
```  
  
## Remarks  
 The data member that stores the error condition. The HRESULT value is set by the constructor, [CAtlException::CAtlException](../vs140/catlexception--catlexception.md).  
  
## Requirements  
 **Header:** atlexcept.h
