# CStringData::nAllocLength
Length of the allocated character buffer.  
  
## Syntax  
  
```  
  
int nAllocLength;  
  
```  
  
## Remarks  
 Stores the length of the allocated data buffer in `XCHAR`s (not including terminating null).  
  
## Requirements  
 **Header:** atlsimpstr.h
