# Iteration Statements (C# Reference)
You can create loops by using the iteration statements. Iteration statements cause embedded statements to be executed a number of times, subject to the loop-termination criteria. These statements are executed in order, except when a [jump statement](../vs140/jump-statements--csharp-reference-.md) is encountered.  
  
 The following keywords are used in iteration statements:  
  
-   [do](../vs140/do--csharp-reference-.md)  
  
-   [for](../vs140/for--csharp-reference-.md)  
  
-   [foreach](../vs140/foreach--in--csharp-reference-.md)  
  
-   [in](../vs140/foreach--in--csharp-reference-.md)  
  
-   [while](../vs140/while--csharp-reference-.md)
