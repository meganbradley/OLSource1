# IPointerInactiveImpl::OnInactiveMouseMove
Notifies the object that the mouse pointer has moved over it, indicating the object can fire mouse events.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Returns **E_NOTIMPL**.
