# CMFCRibbonEdit::OnLButtonUp
Called by the framework when the user releases the left mouse button.  
  
## Syntax  
  
```  
virtual void OnLButtonUp(  
   CPoint point  
);  
```  
  
#### Parameters  
 [in] `point`  
 This parameter is not used.
