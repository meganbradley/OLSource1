# Compiler Error C2319
'try/catch' must be followed by a compound statement. Missing '{'  
  
 A <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> block is not found following the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> statement. The block must be enclosed in curly braces.  
  
 The following sample generates C2319:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
