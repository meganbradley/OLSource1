# Compiler Error C2583
'identifier' : 'const/volatile' 'this' pointer is illegal for constructors/destructors  
  
 A constructor or destructor is declared <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>. This is not allowed.  
  
 The following sample generates C2583:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
