# Relational operator expected
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement contains an <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> clause but no comparison operator such as <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>. If a <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> statement does not include an operator, <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> is assumed and <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> is not used.  
  
 **Error ID:** BC30239  
  
### To correct this error  
  
-   Remove the <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> keyword or follow it with a comparison operator.
