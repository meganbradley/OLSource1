# CPrintDialogEx::GetCopies
Call this function after calling `DoModal` to retrieve the number of copies requested.  
  
## Syntax  
  
```  
  
int GetCopies( ) const;  
  
```  
  
## Return Value  
 The number of copies requested.  
  
## Requirements  
 **Header:** afxdlgs.h
