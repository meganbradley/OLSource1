# CJumpList::AddKnownCategory
Appends a Known Category to the list.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 Specifies a known category type. Can be either <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, or <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.  
  
## Return Value  
  
## Remarks  
 Known Categories are the Frequent and Recent categories that we will automatically calculate for every application that utilizes <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> (or indirectly uses it as the shell will call it on the application's behalf in some scenarios).
