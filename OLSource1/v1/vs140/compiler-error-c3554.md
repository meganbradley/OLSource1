# Compiler Error C3554
'decltype' cannot be combined with any other type-specifier  
  
 You cannot qualify the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> keyword with any type specifier. For example, the following code fragment yields error C3554.  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
