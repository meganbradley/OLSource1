# Compiler Error C2267
'function' : static functions with block scope are illegal  
  
 A local function is declared <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. Static functions must have global scope.  
  
 The following sample generates C2267:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
