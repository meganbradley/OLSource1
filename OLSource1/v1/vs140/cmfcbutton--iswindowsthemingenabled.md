# CMFCButton::IsWindowsThemingEnabled
Indicates whether the style of the button border corresponds to the current Windows theme.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the style of the button border corresponds to the current Windows theme; otherwise, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Requirements  
 **Header:** afxbutton.h
