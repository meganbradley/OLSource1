# ENDM
Terminates a macro or repeat block.  
  
## Syntax  
  
```  
  
ENDM  
  
```

 See [MACRO](../vs140/macro.md), [FOR](../vs140/for--masm-.md), [FORC](../vs140/forc.md), [REPEAT](../vs140/repeat.md), or **WHILE**.
