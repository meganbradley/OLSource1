# CAxDialogImpl::GetIDD
Call this method to get the dialog template resource ID.  
  
## Syntax  
  
```  
  
int GetIDD( );  
  
```  
  
## Return Value  
 Returns the dialog template resource ID.  
  
## Requirements  
 **Header:** atlwin.h
