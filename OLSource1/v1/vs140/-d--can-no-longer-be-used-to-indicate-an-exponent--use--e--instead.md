# &#39;D&#39; can no longer be used to indicate an exponent, use &#39;E&#39; instead
The 'D' character cannot be used to indicate exponentiation.  
  
 **Error ID:** BC30827  
  
### To correct this error  
  
-   Use the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> operator or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> characters to indicate an exponent is present. For example:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
