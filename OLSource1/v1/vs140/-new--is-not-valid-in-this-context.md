# &#39;New&#39; is not valid in this context
The <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> keyword occurs where a type name is required. The context calls for a value type that cannot be combined with <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
 **Error ID:** BC30200  
  
### To correct this error  
  
-   Remove the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> keyword.
