# CMFCRibbonBaseElement::GetParentRibbonBar
Retrieves the parent ribbon bar for the ribbon element.  
  
## Syntax  
  
```  
CMFCRibbonBar* GetParentRibbonBar() const;  
```  
  
## Return Value  
 A pointer to the parent ribbon bar for the ribbon element.
