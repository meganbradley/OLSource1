# CMFCToolBarButton::m_nID
The command ID of the button.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 A command ID of -1 indicates that the button is a separator. All button separators have the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> style. See [CMFCToolBarButton::m_nStyle](../vs140/cmfctoolbarbutton--m_nstyle.md) for more information about button styles.  
  
## Requirements  
 **Header:** afxtoolbarbutton.h
