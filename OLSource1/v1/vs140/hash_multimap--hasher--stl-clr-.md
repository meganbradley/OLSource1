# hash_multimap::hasher (STL-CLR)
The hashing delegate for a key.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 The type describes a delegate that converts a key value to an integer.
