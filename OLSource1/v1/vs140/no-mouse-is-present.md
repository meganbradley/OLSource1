# No mouse is present
One of the properties of the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> object was called, but the computer has no mouse or mouse port installed.  
  
### To correct this error  
  
-   Add a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> block around the call to the property of the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> object.  
  
     — or —  
  
-   Install a mouse on the computer.
