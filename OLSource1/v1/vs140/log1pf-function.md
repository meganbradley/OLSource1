# log1pf Function
Calculates the base-e logarithm of 1 plus the argument  
  
## Syntax  
  
```  
inline float log1pf(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the base-e logarithm of 1 plus the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
