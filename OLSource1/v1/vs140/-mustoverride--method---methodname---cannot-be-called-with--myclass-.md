# &#39;MustOverride&#39; method &#39;&lt;methodname&gt;&#39; cannot be called with &#39;MyClass&#39;
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder> is equivalent to <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, but all method invocations on it are treated as if the method being invoked were <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
 **Error ID:** BC30614  
  
### To correct this error  
  
-   Remove the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> modifier, or declare the method in a base class and inherit and override that class.
