# CJumpList::CJumpList
Constructs a `CJumpList` object.  
  
## Syntax  
  
```  
CJumpList(  
   BOOL bAutoCommit = TRUE  
);  
```  
  
#### Parameters  
 `bAutoCommit`  
 If this parameter is FALSE the list is not automatically committed in  destructor.  
  
## Requirements  
 **Header:** afxadv.h
