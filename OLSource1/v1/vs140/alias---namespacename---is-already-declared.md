# Alias &#39;&lt;namespacename&gt;&#39; is already declared
You have attempted to use <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> to import a namespace that is already declared.  
  
 **Error ID:** BC30572  
  
### To correct this error  
  
-   Check the namespace you are attempting to import.
