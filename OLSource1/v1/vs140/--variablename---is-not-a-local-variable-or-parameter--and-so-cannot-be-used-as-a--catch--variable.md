# &#39;&lt;variablename&gt;&#39; is not a local variable or parameter, and so cannot be used as a &#39;Catch&#39; variable
Variables in <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statements must be declared locally, or be parameters for the current procedure.  
  
 **Error ID:** BC31082  
  
### To correct this error  
  
1.  Declare local variables or parameters for <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statements.
