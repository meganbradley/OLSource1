# DT
Allocates and optionally initializes 10 bytes of storage for each <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. Can also be used as a type specifier anywhere a type is legal. <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> is a synonym of [TBYTE](../vs140/tbyte.md).  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
