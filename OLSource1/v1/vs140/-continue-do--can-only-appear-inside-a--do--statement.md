# &#39;Continue Do&#39; can only appear inside a &#39;Do&#39; statement
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement can only appear within a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> loop.  
  
 **Error ID:** BC30782  
  
### To correct this error  
  
1.  If the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement is in a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> loop, change the statement to <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
2.  If the <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> statement is in a <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> loop, change the statement to <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>.  
  
3.  Otherwise, remove the <CodeContentPlaceHolder>8\</CodeContentPlaceHolder> statement.
