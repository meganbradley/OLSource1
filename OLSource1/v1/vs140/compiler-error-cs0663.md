# Compiler Error CS0663
Cannot define overloaded methods that differ only on ref and out.  
  
 Methods that differ only on their use of [ref](../vs140/ref--csharp-reference-.md) and [out](../vs140/out--csharp-reference-.md) on a parameter are not allowed.
