# Compiler Error CS1524
Expected catch or finally  
  
 A <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> block must be followed by a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> block.  
  
 For more information on exceptions, see [Exception Handling Statements](../vs140/exception-handling-statements--csharp-reference-.md).  
  
## Example  
 The following sample generates CS1524:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
