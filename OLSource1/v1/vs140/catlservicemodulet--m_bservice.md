# CAtlServiceModuleT::m_bService
Flag indicating the program is running as a service.  
  
## Syntax  
  
```  
  
BOOL m_bService;  
  
```  
  
## Remarks  
 Used to distinguish a Service EXE from an Application EXE.  
  
## Requirements  
 **Header:** atlbase.h
