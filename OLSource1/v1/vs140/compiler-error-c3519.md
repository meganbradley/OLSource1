# Compiler Error C3519
'invalid_param' : invalid parameter to embedded_idl attribute  
  
 A parameter was passed to the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> attribute of [#import](../vs140/sharpimport-directive--c---.md), but the compiler did not recognize the parameter.  
  
 The only parameters that are allowed for <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> are <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
 The following sample generates C3519:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
