# Container Class::value_type
> [!NOTE]
>  This topic is in the Visual C++ documentation as a nonfunctional example of containers used in the Standard C++ Library. For more information, see [STL Containers](../vs140/stl-containers.md).  
  
 Acts a synonym for the template parameter **Ty**.  
  
## Syntax  
  
```  
  
typedef T4 value_type;  
  
```  
  
## Remarks  
 It is described here as a synonym for the unspecified type **T4** (typically **Alloc::value_type**).
