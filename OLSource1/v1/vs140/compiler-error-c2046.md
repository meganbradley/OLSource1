# Compiler Error C2046
illegal case  
  
 The keyword <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> can appear only in a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement.  
  
 The following sample generates C2046:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 Possible resolution:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
