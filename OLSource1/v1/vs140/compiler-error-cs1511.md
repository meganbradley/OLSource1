# Compiler Error CS1511
Keyword 'base' is not available in a static method  
  
 The [base](../vs140/base--csharp-reference-.md) keyword was used in a [static](../vs140/static--csharp-reference-.md) method. <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> can only be called in an instance constructor, instance method, or instance accessor.
