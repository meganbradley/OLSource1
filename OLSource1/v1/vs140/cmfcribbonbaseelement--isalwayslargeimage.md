# CMFCRibbonBaseElement::IsAlwaysLargeImage
Indicates whether the ribbon element image size is always large.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the ribbon element image size is always large; otherwise <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Remarks  
 Large image size is 32 x 32 pixels.  
  
## Requirements  
 **Header:** afxbaseribbonelement.h
