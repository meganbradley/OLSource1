# CD2DResource::IsValid
Checks resource validity  
  
## Syntax  
  
```  
virtual BOOL IsValid() const = 0;  
```  
  
## Return Value  
 TRUE if resource is valid; otherwise FALSE.  
  
## Requirements  
 **Header:** afxrendertarget.h
