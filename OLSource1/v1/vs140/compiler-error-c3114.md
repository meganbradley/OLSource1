# Compiler Error C3114
'argument': not a valid named attribute argument  
  
 In order for an attribute class data member to be a valid named argument, it must not be marked <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, or <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>. If a property, the property must not be <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> and must have get and set accessors.  
  
 For more information, see [property](../vs140/property---c---component-extensions-.md) and [User-Defined Attributes](../vs140/user-defined-attributes---c---component-extensions-.md).

 The following sample generates C3114.
