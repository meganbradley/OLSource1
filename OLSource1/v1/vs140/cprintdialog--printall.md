# CPrintDialog::PrintAll
Determines whether to print all pages of the document.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Nonzero if all pages in the document are to be printed; otherwise 0.  
  
## Remarks  
 Call this function after calling <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> to determine whether to print all pages in the document.
