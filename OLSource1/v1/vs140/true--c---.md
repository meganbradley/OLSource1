## Remarks  
 This keyword is one of the two values for a variable of type [bool](../vs140/bool--c---.md) or a conditional expression (a conditional expression is now a true boolean expression). If <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> is of type <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, then the statement <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> assigns **true** to <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>.
