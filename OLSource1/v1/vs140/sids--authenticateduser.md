# Sids::AuthenticatedUser
Returns the SECURITY_AUTHENTICATED_USER_RID SID.  
  
> [!IMPORTANT]
>  This function cannot be used in applications that execute in the [!INCLUDE[wrt](../vs140/includes/wrt_md.md)].  
  
## Syntax  
  
```  
  
CSid AuthenticatedUser( ) throw(...);  
  
```
