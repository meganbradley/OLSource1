# CMFCRibbonCheckBox::GetRegularSize
Gets the regular size of the checkbox.

<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 Pointer to the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> object associated with this checkbox.  
  
## Return Value  
 Returns a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> object that contains the regular size of the checkbox.
