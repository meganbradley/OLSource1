# source_link_manager::set_bound Method
Sets the maximum number of source links that can be added to this `source_link_manager` object.  
  
## Syntax  
  
```  
void set_bound(  
   size_t _MaxLinks  
);  
```  
  
#### Parameters  
 `_MaxLinks`  
 The maximum number of links.  
  
## Requirements  
 **Header:** agents.h  
  
 **Namespace:** concurrency
