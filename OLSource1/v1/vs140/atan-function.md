# atan Function
Calculates the arctangent of the argument  
  
## Syntax  
  
```  
inline float atan(  
   float _X  
) restrict(amp);  
inline double atan(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the arctangent value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
