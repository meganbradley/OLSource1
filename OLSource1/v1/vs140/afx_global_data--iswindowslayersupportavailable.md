# AFX_GLOBAL_DATA::IsWindowsLayerSupportAvailable
Indicates whether the operating system supports layered windows.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if layered windows are supported; otherwise, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Remarks  
 If layered windows are supported, *smart docking* markers use layered windows.  
  
## Requirements  
 **Header:** afxglobals.h
