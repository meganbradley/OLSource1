# CDockablePane::IsTracked
Specifies whether a pane is being moved by the user.  
  
## Syntax  
  
```  
BOOL IsTracked() const;  
```  
  
## Return Value  
 `TRUE` if the pane is being moved; otherwise, `FALSE`.  
  
## Requirements  
 **Header:** afxdockablepane.h
