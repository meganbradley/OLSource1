# CDataRecoveryHandler::DeleteAllAutosavedFiles
Deletes all the current autosaved files.  
  
## Syntax  
  
```  
virtual BOOL DeleteAllAutosavedFiles();  
```  
  
## Return Value  
 The default implementation always returns `TRUE`.  
  
## Requirements  
 **Header:** afxdatarecovery.h
