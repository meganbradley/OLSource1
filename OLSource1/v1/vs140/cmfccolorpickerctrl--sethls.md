# CMFCColorPickerCtrl::SetHLS
Sets the current color to the specified HLS color value.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>

 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 A hue value.  
  
 [in] <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 A luminance value.  
  
 [in] <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 A saturation value.  
  
 [in] <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>  
 <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> to force the window to immediately update to the new color; otherwise, <CodeContentPlaceHolder>6\</CodeContentPlaceHolder>. The default is <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>.
