# &#39;RemoveHandler&#39; declaration must end with a matching &#39;End RemoveHandler&#39;
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> declaration must end with an <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement.  
  
 **Error ID:** BC31116  
  
### To correct this error  
  
-   Check that the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> declaration ends with an <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement.
