# Compiler Error CS1615
Argument 'number' should not be passed with the 'keyword' keyword  
  
 One of the keywords <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or **out** was used when the function did not take a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> or **out** parameter for that argument. To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.  
  
 The following sample generates CS1615:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
