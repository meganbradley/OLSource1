# CArray::IsEmpty
Determines whether the array is empty.  
  
## Syntax  
  
```  
  
BOOL IsEmpty( ) const;  
  
```  
  
## Return Value  
 Nonzero if the array contains no elements; otherwise 0.

 **Header:** afxtempl.h
