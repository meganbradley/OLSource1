# Compiler Error C2180
controlling expression has type 'type'  
  
 The controlling expression in an <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, or <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> statement is an expression cast to <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>. To fix this issue, change the controlling expression to one that produces a <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> or a type that can be converted to <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>.
