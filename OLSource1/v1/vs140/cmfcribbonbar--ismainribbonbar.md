# CMFCRibbonBar::IsMainRibbonBar
Indicates whether the ribbon bar is the primary ribbon bar.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Always returns <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
## Remarks  
 By default this method always returns <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>. Override this method to indicate whether the ribbon bar is the primary ribbon bar.  
  
## Requirements  
 **Header:** afxribbonbar.h
