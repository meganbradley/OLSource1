# CMFCTasksPane::IsHistoryMenuButtonsEnabled
Indicates whether the **Next** and **Previous** navigation buttons have drop-down menus.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the **Next** and **Previous** navigation buttons have drop-down menus; otherwise, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.
