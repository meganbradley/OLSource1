# CComAutoThreadModule::dwThreadID
As of ATL 7.0, `CComAutoThreadModule` is obsolete: see [ATL Module Classes](../vs140/atl-module-classes.md) for more details.  
  
## Syntax  
  
```  
  
DWORD dwThreadID;  
  
```  
  
## Remarks  
 Contains the identifier of the current thread.
