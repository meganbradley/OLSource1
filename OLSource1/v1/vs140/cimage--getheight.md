# CImage::GetHeight
Retrieves the height, in pixels, of an image.  
  
## Syntax  
  
```  
  
int GetHeight( ) const throw( );  
  
```  
  
## Return Value  
 The height, in pixels, of an image.  
  
## Requirements  
 **Header:** atlimage.h
