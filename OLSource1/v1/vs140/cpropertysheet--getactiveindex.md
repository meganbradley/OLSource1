# CPropertySheet::GetActiveIndex
Gets the index number of the property sheet window's active page and then uses the returned index number as the parameter for <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 The index number of the active page.
