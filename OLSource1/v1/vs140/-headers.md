# -HEADERS
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This option displays the file header and the header for each section. When used with a library, it displays the header for each member object.  
  
 Only the **/HEADERS** DUMPBIN option is available for use on files produced with the [/GL](../vs140/-gl--whole-program-optimization-.md) compiler option.
