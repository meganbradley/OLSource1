# using (C# Reference)
The <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> keyword has two major uses:  
  
-   As a directive, when it is used to create an alias for a namespace or to import types defined in other namespaces. See [using Directive](../vs140/using-directive--csharp-reference-.md).  
  
-   As a statement, when it defines a scope at the end of which an object will be disposed. See [using Statement](../vs140/using-statement--csharp-reference-.md).
