# SizeSettings::IsHorizontal
Checks if the resize data specifies a nonzero horizontal resizing.  
  
## Syntax  
  
```vb  
BOOL IsHorizontal() const  
```  
  
## Return Value  
 TRUE if the `SizeSettings` object specifies a nonzero horizontal resizing.

## Requirements  
 **Header:** afxlayout.h
