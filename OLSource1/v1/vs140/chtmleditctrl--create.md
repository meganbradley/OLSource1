# CHtmlEditCtrl::Create
Creates a WebBrowser ActiveX control and attaches it to the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object. The WebBrowser ActiveX control automatically navigates to a default document and then is placed in edit mode by this function.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 This parameter is unused.  
  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 This parameter is unused.  
  
 <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>  
 Specifies the control's size and position.  
  
 <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>  
 Specifies the control's parent window. It must not be **NULL**.  
  
 <CodeContentPlaceHolder>6\</CodeContentPlaceHolder>  
 Specifies the control's ID.  
  
 <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>  
 This parameter is unused.  
  
## Return Value  
 Returns **TRUE** on success, **FALSE** on failure.
