# Compiler Error C3233
'type' : generic type parameter already constrained  
  
 It is not valid to constrain a generic parameter in more than one <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> clause.  
  
 The following sample generates C3233:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
