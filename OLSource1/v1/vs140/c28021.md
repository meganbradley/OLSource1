# C28021
warning C28021: The parameter \<param> being annotated with \<anno> must be a pointer  
  
 This warning is reported when the object being annotated is not a pointer type. This annotation cannot be used with <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> or integral types.
