# CMFCRibbonButton::GetMenu
Returns a handle to a Windows menu that is assigned to the ribbon button.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 A handle to a Windows menu assigned to the button; <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if there is no menu assigned.
