# CHtmlView::GetSource
Call this member function to retrieve the HTML source code for the web page.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Nonzero if successful; otherwise zero.  
  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 A [CString](../vs140/cstringt-class.md) that will hold the source code.  
  
## Remarks  
 This function is equivalent to the "View Source" command in Internet Explorer, except that the source code is returned in a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.

 **Header:** afxhtml.h
