# CMFCRibbonEdit::Redraw
Updates the display of the [CMFCRibbonEdit](../vs140/cmfcribbonedit-class.md) control.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This method redraws the display rectangle for the [CMFCRibbonEdit](../vs140/cmfcribbonedit-class.md) object by indirectly calling [CWnd::RedrawWindow](http://msdn.microsoft.com/library/windows/desktop/dd162911) with the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, and <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> flags set.  
  
## Requirements  
 **Header:** afxribbonedit.h
