# CSimpleMap::GetSize
Returns the number of entries in the mapping array.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Returns the number of entries (a key and value is one entry) in the mapping array.
