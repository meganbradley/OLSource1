# Compiler Error C3762
unable to process attribute 'attribute'  
  
 A user-defined attribute that inherits from <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> is being used to define a security attribute. Such an attribute cannot be used in the same assembly where it is defined.
