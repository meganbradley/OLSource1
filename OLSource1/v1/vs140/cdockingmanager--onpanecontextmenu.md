# CDockingManager::OnPaneContextMenu
Called by the framework when it builds a menu that has a list of panes.  
  
## Syntax  
  
```  
void OnPaneContextMenu(  
   CPoint point  
);  
```  
  
#### Parameters  
 [in] `point`  
 Specifies the location of the menu.  
  
## Requirements  
 **Header:** afxdockingmanager.h
