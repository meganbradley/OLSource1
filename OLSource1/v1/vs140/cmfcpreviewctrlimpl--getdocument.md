# CMFCPreviewCtrlImpl::GetDocument
Returns a document connected to this preview control.  
  
## Syntax  
  
```  
ATL::IDocument* GetDocument();  
```  
  
## Return Value  
 A pointer to a document, whose content is previewed in the control.
