# cospif Function
Calculates the cosine value of pi * _X  
  
## Syntax  
  
```  
inline float cospif(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the cosine value of pi * _X  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
