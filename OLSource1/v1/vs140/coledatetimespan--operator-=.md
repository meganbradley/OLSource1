# COleDateTimeSpan::operator =
Copies a `COleDateTimeSpan` value.  
  
## Syntax  
  
```  
  
      COleDateTimeSpan& operator =(   
   double dblSpanSrc    
) throw( );  
```  
  
## Remarks  
 This overloaded assignment operator copies the source date/time-span value into this `COleDateTimeSpan` object.  
  
## Requirements  
 **Header:** atlcomtime.h
