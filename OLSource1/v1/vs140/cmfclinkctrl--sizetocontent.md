# CMFCLinkCtrl::SizeToContent
Resizes the button to contain the button text or bitmap.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> to center the button text and bitmap vertically between the top and bottom of the link control; otherwise, <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>. The default value is <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
 [in] <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>  
 <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> to center the button text and bitmap horizontally between the left and right sides of the link control; otherwise, <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>. The default value is <CodeContentPlaceHolder>8\</CodeContentPlaceHolder>.
