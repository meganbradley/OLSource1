# C6509
warning C6509: invalid annotation: 'return' cannot be referenced from a precondition  
  
 This warning indicates that the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  keyword cannot be used in a precondition. The <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> keyword is used to terminate the execution of a function and return control to the calling function.  
  
## Example  
 The following code generates this warning because <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> is used in a precondition:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 To correct this warning, use the following code:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
