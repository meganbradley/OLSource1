# Methods or events that implement interface members cannot be declared &#39;Shared&#39;
You have attempted to declare as <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> a method or event that implements an interface member. Methods and events being implemented in a class cannot be designated <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, except in a non-inheritable class.  
  
 **Error ID:** BC30505  
  
### To correct this error  
  
-   Remove the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> keyword.
