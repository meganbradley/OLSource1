# CMFCToolBarButton::m_bText
Specifies whether a text label is displayed on the button.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 If this data member is set to <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, the framework displays the text label of the toolbar button; otherwise the framework does not display the text label. This member affects the return value of the [CMFCToolBarButton::m_bText](../vs140/cmfctoolbarbutton--m_btext.md) method.  
  
## Requirements  
 **Header:** afxtoolbarbutton.h
