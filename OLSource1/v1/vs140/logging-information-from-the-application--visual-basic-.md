# Logging Information from the Application (Visual Basic)
This section contains topics that cover how to log information from your application using the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object, and how to extend the application's logging capabilities.  
  
 The <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> object provides methods for writing information to the application's log listeners, and the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> object's advanced <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> property provides detailed configuration information. The <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> object is configured by the application's configuration file.  
  
 The <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> object is available only for ASP.NET applications. For client applications, use <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>. For more information, see \<xref:Microsoft.VisualBasic.Logging.Log*>.
