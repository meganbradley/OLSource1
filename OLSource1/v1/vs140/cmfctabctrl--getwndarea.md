# CMFCTabCtrl::GetWndArea
Retrieves the boundary of the client area of the current tab control.  
  
## Syntax  
  
```  
void GetWndArea(  
   CRect& rect  
) const;  
```  
  
#### Parameters  
 [in, out] `rect`  
 When this method returns, this parameter contains a rectangle that bounds the current tab control.
