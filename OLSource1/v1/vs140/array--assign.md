# array::assign
Replaces all elements.

<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>

 The member function replaces the sequence controlled by <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> with a repetition of <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> elements of value <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>.
