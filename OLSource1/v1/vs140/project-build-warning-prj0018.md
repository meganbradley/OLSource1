# Project Build Warning PRJ0018
The following environment variables were not found:  
  
 An environment variable is not defined. This error lists the environment variables that were not defined.  
  
 See [Macros for Custom Build Commands](../vs140/common-macros-for-build-commands-and-properties.md) for information on macros in the build process.
