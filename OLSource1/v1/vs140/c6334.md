# C6334
warning C6334: sizeof operator applied to an expression with an operator may yield unexpected results  
  
 This warning indicates a misuse of the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> operator. The <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> operator, when applied to an expression, yields the size of the type of the resulting expression.  
  
 For example, in the following code:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> will be assigned the value 4, not 9, because the resulting expression is no longer a pointer to the array <CodeContentPlaceHolder>6\</CodeContentPlaceHolder>, but simply a pointer.  
  
## Example  
 The following code generates this warning:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 To correct this warning, use the following code:  
  
<CodeContentPlaceHolder>2\</CodeContentPlaceHolder>
