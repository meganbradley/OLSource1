# CD2DGeometry::Get
Returns ID2D1Geometry interface  
  
## Syntax  
  
```  
ID2D1Geometry* Get();  
```  
  
## Return Value  
 Pointer to an ID2D1Geometry interface or NULL if object is not initialized yet.
