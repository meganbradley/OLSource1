# &#39;Implements&#39; not valid in Modules
An <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement or clause occurs inside a module. Modules cannot implement interfaces.  
  
 **Error ID:** BC30231  
  
### To correct this error  
  
-   Remove the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement or clause, or retype the module as a class.
