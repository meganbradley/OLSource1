# Compiler Warning (level 4) C4680
'class' : coclass does not specify a default interface  
  
 A [default](../vs140/default--c---.md) interface was not specified for a class that was marked with the [coclass](../vs140/coclass.md) attribute. In order for an object to be useful, it must implement an interface.
