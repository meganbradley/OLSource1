# CMFCToolBarButton::m_bImage
Specifies whether an image is displayed on the button.  
  
## Syntax  
  
```  
BOOL m_bImage;  
```  
  
## Remarks  
 If this data member is set to `TRUE`, the framework displays the image that is associated with the toolbar button; otherwise the framework does not display the image. This member affects the return value of the [CMFCToolBarButton::m_bImage](../vs140/cmfctoolbarbutton--m_bimage.md) method.  
  
## Requirements  
 **Header:** afxtoolbarbutton.h
