# Compiler Error C2467
illegal declaration of anonymous 'user-defined-type'  
  
 A nested user-defined type was declared. This is an error when compiling C source code with the ANSI compatibility option ([/Za](../vs140/-za---ze--disable-language-extensions-.md)) enabled.  
  
 The following sample generates C2467:  
  
```  
//C2467.c  
// compile with: /Za   
int main() {  
   struct X {  
      union { int i; };   // C2467, nested declaration  
   };  
}  
```
