# &#39;Sub New&#39; cannot handle events
You attempted to combine <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> with <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, which is invalid. Use the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> keyword at the end of a procedure declaration to cause it to handle events raised by an object variable declared using the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> keyword.  
  
 **Error ID:** BC30497  
  
### To correct this error  
  
1.  Do not use <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> in this context.
