# CMFCColorBar::GetHighlightedColor
Retrieves the color that signifies that a color button has the focus; that is, the button is *hot*.  
  
## Syntax  
  
```  
COLORREF GetHighlightedColor() const;  
```  
  
## Return Value  
 An RGB value.

## Requirements  
 **Header:** afxcolorbar.h
