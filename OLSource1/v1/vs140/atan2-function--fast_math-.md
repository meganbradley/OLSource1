# atan2 Function (fast_math)
Calculates the arctangent of _Y/_X  
  
## Syntax  
  
```  
inline float atan2(  
   float _Y,  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_Y`  
 Floating-point value  
  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the arctangent value of _Y/_X  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
