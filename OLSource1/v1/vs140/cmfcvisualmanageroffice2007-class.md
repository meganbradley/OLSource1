## Remarks  
 Use <CodeContentPlaceHolder>118\</CodeContentPlaceHolder> to change the visual appearance of your application to resemble that of Microsoft Office 2007. This visual manager requires that you set the style before you use it. Before you pass this visual manager to <CodeContentPlaceHolder>119\</CodeContentPlaceHolder>, you must call the static function <CodeContentPlaceHolder>120\</CodeContentPlaceHolder>.
