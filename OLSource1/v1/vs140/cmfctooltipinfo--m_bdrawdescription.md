# CMFCToolTipInfo::m_bDrawDescription
Specifies whether each tooltip displays description text.  
  
## Syntax  
  
```  
BOOL m_bDrawDescription;  
```  
  
## Remarks  
 Set this member to `TRUE` to display the description, or `FALSE` to hide the description. You can specify the description on a tooltip by calling [CMFCToolTipCtrl::SetDescription](../vs140/cmfctooltipctrl--setdescription.md)  
  
## Requirements  
 **Header:** afxToolTipCtrl.h
