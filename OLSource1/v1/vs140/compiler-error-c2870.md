# Compiler Error C2870
'name' : a namespace definition must appear either at file scope or immediately within another namespace definition  
  
 You defined namespace <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> incorrectly. Namespaces must be defined at file scope (outside all blocks and classes) or immediately within another namespace.  
  
 The following sample generates C2870:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
