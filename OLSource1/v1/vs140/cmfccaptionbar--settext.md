# CMFCCaptionBar::SetText
Sets the text label for the caption bar.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 The text string to set.  
  
 [in] <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 The text alignment.  
  
## Remarks  
 The text label is aligned as specified by the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> parameter. It can be one of the following <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> values:  
  
-   ALIGN_INVALID  
  
-   ALIGN_LEFT  
  
-   ALIGN_RIGHT  
  
-   ALIGN_CENTER  
  
## Requirements  
 **Header:** afxcaptionbar.h
