# &#39;&lt;modifier&gt;&#39; is not valid on a Declare
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement contains an invalid keyword, such as <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
 **Error ID:** BC30244  
  
### To correct this error  
  
-   Remove the invalid keyword from the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement.
