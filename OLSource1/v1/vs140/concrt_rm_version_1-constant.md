# CONCRT_RM_VERSION_1 Constant
Indicates support of the Resource Manager interface defined in Visual Studio 2010.  
  
## Syntax  
  
```  
const unsigned int CONCRT_RM_VERSION_1 = 0x00010000;  
```  
  
## Requirements  
 **Header:** concrtrm.h  
  
 **Namespace:** concurrency
