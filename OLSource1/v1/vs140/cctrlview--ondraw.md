# CCtrlView::OnDraw
Called by the framework to draw the contents of the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object using the specified device context.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 A pointer to the device context in which the drawing occurs.  
  
## Remarks  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> is typically called for screen display, passing a screen device context specified by <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
## Requirements  
 **Header:** afxwin.h
