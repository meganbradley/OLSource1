# CComDynamicUnkArray::end
Returns a pointer to one past the last **IUnknown** pointer in the collection.  
  
## Syntax  
  
```  
  
IUnknown** end( );  
```  
  
## Return Value  
 A pointer to an **IUnknown** interface pointer.  
  
## Requirements  
 **Header:** atlcom.h
