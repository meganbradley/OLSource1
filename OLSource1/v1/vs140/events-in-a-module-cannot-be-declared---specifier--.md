# Events in a Module cannot be declared &#39;&lt;specifier&gt;&#39;
You have used a specifier that is not valid on an event within a <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement. Modules can never be instantiated, do not support inheritance, and cannot implement interfaces.  
  
 **Error ID:** BC30434  
  
### To correct this error  
  
-   Remove the specifier.
