# CMFCTasksPane::RemoveAllGroups
Removes all groups on the specified page.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 Specifies the zero-based index of the page.  
  
## Remarks  
 Removes all groups on the page specified by <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, or all groups if there is only a default page.  
  
## Requirements  
 **Header:** afxTasksPane.h
