# CMFCBaseTabCtrl::GetTabsNum
Retrieves the number of tabs in the tab control.  
  
## Syntax  
  
```  
virtual int GetTabsNum() const;  
```  
  
## Return Value  
 The number of tabs in the tab control.  
  
## Requirements  
 **Header:** afxbasetabctrl.h
