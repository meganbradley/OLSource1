# Compiler Error CS0821
Implicitly typed locals cannot be fixed  
  
 Implicitly typed local variables and anonymous types are not supported in the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> context.  
  
### To correct this error  
  
1.  Either remove the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> modifier from the variable or else give the variable an explicit type.
