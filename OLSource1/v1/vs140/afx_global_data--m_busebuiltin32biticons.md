# AFX_GLOBAL_DATA::m_bUseBuiltIn32BitIcons
Indicates whether the framework uses predefined 32-bit color icons or icons of a lower resolution.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> specifies that the framework use 32-bit color icons; <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> specifies lower resolution icons. The <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> constructor initializes this member to <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
 This member must be set at application startup.  
  
## Requirements  
 **Header:** afxglobals.h
