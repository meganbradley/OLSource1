# MSBuild Error MSB3072
**The "Exec" task needs a command to execute.**  
  
 The attribute <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> is a required attribute for the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> task.  
  
### To correct this error  
  
1.  In the project file, specify the attribute <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> for the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> task.
