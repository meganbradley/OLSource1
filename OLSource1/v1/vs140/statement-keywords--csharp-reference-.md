# Statement Keywords (C# Reference)
Statements are program instructions. Except as described in the topics referenced in the following table, statements are executed in sequence. The following table lists the C# statement keywords. For more information about statements that are not expressed with any keyword, see [Statements (C# Programming Guide)](../vs140/statements--csharp-programming-guide-.md).

|[Iteration statements](../vs140/iteration-statements--csharp-reference-.md)|[do](../vs140/do--csharp-reference-.md), [for](../vs140/for--csharp-reference-.md), [foreach](../vs140/foreach--in--csharp-reference-.md), [in](../vs140/foreach--in--csharp-reference-.md), [while](../vs140/while--csharp-reference-.md)|
