# unordered_multimap::iterator
The type of an iterator for the controlled sequence.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 The type describes an object that can serve as a forward iterator for the controlled sequence. It is described here as a synonym for the implementation-defined type <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.
