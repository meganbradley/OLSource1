# CMFCFilterChunkValueImpl::IsValid
Checks whether this property value is valid or not.  
  
## Syntax  
  
```  
BOOL IsValid() const;  
```  
  
## Return Value  
 `TRUE` if the current chunk value is valid; otherwise `FALSE`.
