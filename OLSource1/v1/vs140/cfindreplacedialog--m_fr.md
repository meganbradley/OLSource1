# CFindReplaceDialog::m_fr
Used to customize a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> is a structure of type [FINDREPLACE](http://msdn.microsoft.com/library/windows/desktop/ms646835). Its members store the characteristics of the dialog-box object. After constructing a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> object, you can use <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> to modify various values in the dialog box.  
  
 For more information on this structure, see the **FINDREPLACE** structure in the [!INCLUDE[winSDK](../vs140/includes/winsdk_md.md)].
