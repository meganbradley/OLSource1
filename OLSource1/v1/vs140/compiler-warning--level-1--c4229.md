# Compiler Warning (level 1) C4229
anachronism used : modifiers on data are ignored  
  
 Using a Microsoft modifier such as <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> on a data declaration is an outdated practice.  
  
## Example  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
