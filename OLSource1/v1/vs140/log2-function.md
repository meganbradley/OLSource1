# log2 Function
Calculates the base-2 logarithm of the argument  
  
## Syntax  
  
```  
inline float log2(  
   float _X  
) restrict(amp);  
inline double log2(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the base-10 logarithm of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
