# WithEvents variable name length cannot exceed 1019 characters
The name or identifier of every variable declared using <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> is limited to 1019 characters.  
  
 **Error ID:** BC32205  
  
### To correct this error  
  
-   Reduce the length of the identifier.
