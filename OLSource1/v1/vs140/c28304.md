# C28304
warning C28304: For C++ reference-parameter \<parameter_name>, an improperly placed <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> operator was found applied to \<token>.  
  
 The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> operator should only be used in special circumstances involving C++ reference parameters and only in system-provided macros. It must be immediately followed by a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> operator or a functional annotation.
