# &#39;End Get&#39; must be preceded by a matching &#39;Get&#39;
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder> is used to terminate <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> property procedures. The <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> construct was encountered outside a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> property procedure.  
  
 **Error ID:** BC30630  
  
### To correct this error  
  
1.  Make sure that the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> property procedure is declared after a <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> keyword and before the <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> construct.  
  
2.  Make sure that the <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> property procedure begins with the <CodeContentPlaceHolder>8\</CodeContentPlaceHolder> keyword and ends with <CodeContentPlaceHolder>9\</CodeContentPlaceHolder> construct.
