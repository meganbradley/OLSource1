# Compiler Error C3723
'function': could not resolve event  
  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> could not resolve which event to call.  
  
 The following sample generates C3723:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> are not compatible with /clr programming.  Use the += and -= operators instead.  
  
 The following sample generates C3723:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
