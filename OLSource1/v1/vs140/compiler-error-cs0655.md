# Compiler Error CS0655
'parameter' is not a valid named attribute argument because it is not a valid attribute parameter type  
  
 See [Attributes (C# Programmers Reference)](../vs140/attributes--csharp-and-visual-basic-.md) for a discussion of valid parameter types for an attribute.
