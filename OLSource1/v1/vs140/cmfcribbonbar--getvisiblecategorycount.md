# CMFCRibbonBar::GetVisibleCategoryCount
Retrieves the number of visible categories on the ribbon bar.  
  
## Syntax  
  
```  
int GetVisibleCategoryCount() const;  
```  
  
## Return Value  
 The number of visible categories on the ribbon bar.
