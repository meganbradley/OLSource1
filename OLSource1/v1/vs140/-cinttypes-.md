# &lt;cinttypes&gt;
Includes the Standard C library header \<inttypes.h>. Including this header also includes \<cstdint>.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 Including this header ensures that the names declared by using external linkage in the Standard C library header are declared in the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> namespace.
