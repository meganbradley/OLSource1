# CMFCButton::SetTextHotColor
Sets the color of the button text for a button that is selected.  
  
## Syntax  
  
```  
void SetTextHotColor(  
   COLORREF clrTextHot   
);  
```  
  
#### Parameters  
 [in] `clrTextHot`  
 An RGB color value.
