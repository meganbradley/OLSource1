# XML namespace prefix &#39;prefix&#39; is already declared
An XML namespace prefix defined by an <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement matches a previously defined XML namespace prefix.  
  
 **Error ID:** BC30573  
  
### To correct this error  
  
-   Use a unique name for the XML namespace prefix.
