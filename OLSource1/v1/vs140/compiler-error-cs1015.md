# Compiler Error CS1015
An object, string, or class type expected  
  
 An attempt was made to pass a predefined data type into a [catch](../vs140/try-catch--csharp-reference-.md) block. Only data types that derive from \<xref:System.Exception*?displayProperty=fullName> can be passed into a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> block. For more information on exceptions, see [Exception Handling Statements](../vs140/exception-handling-statements--csharp-reference-.md).
