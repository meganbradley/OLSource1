# @@:
Defines a code label recognizable only between *label1* and *label2*, where *label1* is either start of code or the previous `@@:` label, and *label2* is either end of code or the next `@@:` label. See [@B](../vs140/@b.md) and [@F](../vs140/@f.md).  
  
## Syntax  
  
```  
  
@@:  
  
```
