# Compiler Error CS0585
Internal Compiler Error: stage 'stage'  
  
 Try to determine if the compiler is failing due to its inability to parse unexpected syntax. If that is not the case, try [Getting Help from Microsoft Product Support Services (Visual Studio)](../vs140/talk-to-us.md).
