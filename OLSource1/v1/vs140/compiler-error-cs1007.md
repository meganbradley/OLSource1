# Compiler Error CS1007
Property accessor already defined  
  
 When you declare a [property](../vs140/using-properties--csharp-programming-guide-.md), you must also declare its accessor methods. However, a property cannot have more than one <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> accessor method or more than one <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> accessor method.
