# CMFCToolBarImages::ConvertTo32Bits
Converts underlined bitmaps to 32 bpp images.  
  
## Syntax  
  
```  
BOOL ConvertTo32Bits(  
   COLORREF clrTransparent = (COLORREF)-1  
);  
```  
  
#### Parameters  
 `clrTransparent`  
 Specifies transparent color of underlined bitmaps.
