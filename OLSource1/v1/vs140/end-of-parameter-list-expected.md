# End of parameter list expected
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> specification is followed by one or more other argument specifications in a procedure declaration.  
  
 **Error ID:** BC30192  
  
### To correct this error  
  
-   Move the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> to the end of the argument list.
