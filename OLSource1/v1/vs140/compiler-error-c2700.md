# Compiler Error C2700
'identifier' : cannot be thrown (use /W4 for more info)  
  
 The object cannot be thrown. Compile with [/W4](../vs140/-w---w0---w1---w2---w3---w4---w1---w2---w3---w4---wall---wd---we---wo---wv---wx--warning-level-.md) for more diagnostic information.
