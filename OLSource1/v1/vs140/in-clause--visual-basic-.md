# In Clause (Visual Basic)
Specifies the group that the loop variable is to traverse in a `For Each` loop, or specifies the collection to query in a `From`, `Join`, or `Group Join` clause.  
  
## Remarks  
 The `In` keyword can be used in the following contexts:  
  
 [For Each...Next Statement](../vs140/for-each...next-statement--visual-basic-.md)  
  
 [From Clause (Visual Basic)](../vs140/from-clause--visual-basic-.md)  
  
 [Join Clause (Visual Basic)](../vs140/join-clause--visual-basic-.md)  
  
 [Group Join Clause (Visual Basic)](../vs140/group-join-clause--visual-basic-.md)
