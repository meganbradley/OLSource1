# CWnd::OnHelpFinder
Handles the **ID_HELP_FINDER** and **ID_DEFAULT_HELP** commands.  
  
## Syntax  
  
```  
  
afx_msg void OnHelpFinder( );  
  
```  
  
## Remarks  
 See [CWinApp::OnHelpFinder](../vs140/cwinapp--onhelpfinder.md) for more information.
