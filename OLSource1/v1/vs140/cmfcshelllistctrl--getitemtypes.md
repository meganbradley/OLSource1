# CMFCShellListCtrl::GetItemTypes
Returns the type of items displayed by the [CMFCShellListCtrl](../vs140/cmfcshelllistctrl-class.md) object.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 A [SHCONTF](http://msdn.microsoft.com/library/windows/desktop/bb762539) value that contains the type of items listed in the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
## Remarks  
 To set the type of items listed in a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, call [CMFCShellListCtrl::SetItemTypes](../vs140/cmfcshelllistctrl--setitemtypes.md).  
  
## Requirements  
 **Header:** afxshelllistctrl.h
