# Directives to the Preprocessor
A "directive" instructs the C preprocessor to perform a specific action on the text of the program before compilation. [Preprocessor directives](../vs140/preprocessor-directives.md) are fully described in the *Preprocessor Reference*. This example uses the preprocessor directive <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 This statement tells the compiler to replace each occurrence of <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> by <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> before compilation. The C compiler preprocessor directives are:  
  
|#define|#endif|#ifdef|#line|
