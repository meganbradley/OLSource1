# pow Function
Calculates _X raised to the power of _Y  
  
## Syntax  
  
```  
inline float pow(  
   float _X,  
   float _Y  
) restrict(amp);  
inline double pow(  
   double _X,  
   double _Y  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value, base  
  
 `_Y`  
 Floating-point value, exponent  
  
## Return Value  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
