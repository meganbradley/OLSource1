# CDHtmlDialog::GetElementProperty
Retrieves the value of the property identified by <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> from the HTML element identified by <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 The ID of an HTML element.  
  
 <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>  
 The dispatch ID of a property.  
  
## Return Value  
 The value of the property or an empty variant if the property or element could not be found.
