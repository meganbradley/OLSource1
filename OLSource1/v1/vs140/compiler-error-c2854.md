# Compiler Error C2854
syntax error in #pragma hdrstop  
  
 The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> gives an invalid filename. The pragma can be followed by an optional filename in parentheses and quotation marks:  
  
 The following sample generates C2854:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
