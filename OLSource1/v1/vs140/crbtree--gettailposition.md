# CRBTree::GetTailPosition
Call this method to get the position value for the element at the tail of the tree.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Returns the position value for the element at the tail of the tree.  
  
## Remarks  
 The value returned by <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> can be used with methods such as [CRBTree::GetKeyAt](../vs140/crbtree--getkeyat.md) or [CRBTree::GetPrev](../vs140/crbtree--getprev.md) to traverse the tree and retrieve values.
