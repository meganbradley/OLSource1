# CDaoRecordset::m_pDAORecordset
Contains a pointer to the OLE interface for the DAO recordset object underlying the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> object.  
  
## Remarks  
 Use this pointer if you need to access the DAO interface directly.  
  
 For related information, see the topic "Recordset Object" in DAO Help.  
  
## Requirements  
 **Header:** afxdao.h
