# CKeyboardManager::CKeyboardManager
Constructs a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 In most cases, you do not have to create a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> directly. By default, the framework creates one for you. To get a pointer to the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, call [CWinAppEx::GetKeyboardManager](../vs140/cwinappex--getkeyboardmanager.md). If you do create one manually, you must initialize it with the method [CWinAppEx::InitKeyboardManager](../vs140/cwinappex--initkeyboardmanager.md).  
  
## Requirements  
 **Header:** afxkeyboardmanager.h
