# CAdapt::m_T
Holds the data being adapted.  
  
## Syntax  
  
```  
  
T m_T;  
  
```  
  
## Remarks  
 This **public** data member can be accessed directly or indirectly with [operator const T&](../vs140/cadapt--operator-const-t-.md) and [operator T&](../vs140/cadapt--operator-t-.md).
