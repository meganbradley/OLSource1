# MSBuild Error MSB1004
**Specify the name of the target.**  
  
 At least one target must be specified with the **/target** switch.  
  
### To correct this error  
  
1.  Specify a target or targets. You can use either a comma or a semicolon to separate a list of targets, for example, <CodeContentPlaceHolder>0\</CodeContentPlaceHolder>. Alternatively, you can repeat the switch, for example, <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
