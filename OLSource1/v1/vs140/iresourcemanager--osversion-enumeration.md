# IResourceManager::OSVersion Enumeration
An enumerated type that represents the operating system version.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Members  
  
### Values  
  
|Name|Description|  
|----------|-----------------|  
|<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>|Preserved for legacy reasons.|  
|<CodeContentPlaceHolder>2\</CodeContentPlaceHolder>|An operating system prior to Windows XP. The Concurrency Runtime is not supported on operating systems with a version earlier than Windows XP with Service Pack 3.|  
|<CodeContentPlaceHolder>3\</CodeContentPlaceHolder>|The Windows Vista and Windows Server 2008 operating systems.|  
|<CodeContentPlaceHolder>4\</CodeContentPlaceHolder>|The Windows 2003 Server operating system.|  
|<CodeContentPlaceHolder>5\</CodeContentPlaceHolder>|The Windows 7 and Windows Server 2008 R2 operating systems.|  
|<CodeContentPlaceHolder>6\</CodeContentPlaceHolder>|Any operating system with version Windows 8 or higher.|  
|<CodeContentPlaceHolder>7\</CodeContentPlaceHolder>|The Windows XP operating system.|
