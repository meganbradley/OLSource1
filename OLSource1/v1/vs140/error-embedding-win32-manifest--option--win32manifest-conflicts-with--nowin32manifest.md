# Error embedding Win32 manifest: Option -win32manifest conflicts with -nowin32manifest
The Visual Basic compiler has been passed both a <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> compiler option and a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> compiler option. Only one of these options can be passed to the Visual Basic compiler.  
  
 **Error ID:** BC2033  
  
### To correct this error  
  
1.  Remove either the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> compiler option or the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> compiler option.
