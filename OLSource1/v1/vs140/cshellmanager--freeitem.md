# CShellManager::FreeItem
Deletes an item list.  
  
## Syntax  
  
```  
void FreeItem(  
   LPITEMIDLIST pidl   
);  
```  
  
#### Parameters  
 [in] `pidl`  
 An item list to delete.  
  
## Requirements  
 **Header:** afxshellmanager.h
