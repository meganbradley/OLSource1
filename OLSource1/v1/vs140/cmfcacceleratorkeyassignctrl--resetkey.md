# CMFCAcceleratorKeyAssignCtrl::ResetKey
Resets the shortcut key.  
  
## Syntax  
  
```  
void ResetKey();  
```  
  
## Remarks  
 The function clears the edit control text. This includes any shortcut keys that the user pressed.  
  
## Requirements  
 **Header:**afxacceleratorkeyassignctrl.h
