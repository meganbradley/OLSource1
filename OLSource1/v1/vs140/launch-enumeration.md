# launch Enumeration
Represents a bitmask type that describes the possible modes for the template function [async](../vs140/async-function.md).  
  
## Syntax  
  
```  
enum class launch{  
   async,  
   deferred  
};  
```  
  
## Requirements  
 **Header:** future  
  
 **Namespace:** std::launch
