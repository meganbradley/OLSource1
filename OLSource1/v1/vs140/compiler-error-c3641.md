# Compiler Error C3641
'function' : invalid calling convention 'calling_convention' for function compiled with /clr:pure or /clr:safe  
  
 Only [__clrcall](../vs140/__clrcall.md) calling convention is allowed with [/clr:pure](../vs140/-clr--common-language-runtime-compilation-.md).
