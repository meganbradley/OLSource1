# CFrameWndEx::IsFullScreen
Determines whether the frame window is in full screen mode.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the frame window is in full screen mode; otherwise <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Remarks  
 You can set the full screen mode by calling the [CFrameWndEx::EnableFullScreenMode](../vs140/cframewndex--enablefullscreenmode.md) method.
