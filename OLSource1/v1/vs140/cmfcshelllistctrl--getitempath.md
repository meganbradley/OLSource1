# CMFCShellListCtrl::GetItemPath
Retrieves the path for an item.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [out] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 A reference to a string that receives the path.  
  
 [in] <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 The index of the list item.  
  
## Return Value  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> if successful; <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> otherwise.  
  
## Remarks  
 The index supplied by <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> is based on the items currently displayed by the [CMFCShellListCtrl Class](../vs140/cmfcshelllistctrl-class.md) object.
