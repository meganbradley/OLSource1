# Compiler Error C3274
__finally/finally without matching try  
  
 A [__finally](../vs140/try-finally-statement.md) or [finally](../vs140/finally.md) statement was found without a matching <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. To resolve this, either delete the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement or add a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement for the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
 The following sample generates C3274:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
