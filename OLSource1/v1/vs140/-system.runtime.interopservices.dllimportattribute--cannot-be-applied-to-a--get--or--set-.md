# &#39;System.Runtime.InteropServices.DllImportAttribute&#39; cannot be applied to a &#39;Get&#39; or &#39;Set&#39;
The <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> attribute was applied to a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> property procedure.  
  
 **Error ID:** BC31524  
  
### To correct this error  
  
1.  Remove <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> from <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> property procedures.
