# Expression Evaluator Error CXX0044
use of _based(void) pointer requires :> operator  
  
 A pointer based on <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> cannot be used directly. You must form a complete pointer using the **:>** operator.  
  
 This error is identical to CAN0044.
