# Compiler Error C2502
'identifier' : too many access modifiers on the base class  
  
 The base class has more than one access modifier. Only one access modifier (<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, or <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>) is allowed.
