# Fatal Error C1021
invalid preprocessor command 'string'  
  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> is not a valid [preprocessor directive](../vs140/preprocessor-directives.md). To resolve the error, use a valid preprocessor name for <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
 The following sample generates C1021:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
