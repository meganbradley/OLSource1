# CAtlTransactionManager::IsFallback
Determines whether the fallback calls are enabled.  
  
## Syntax  
  
```  
BOOL IsFallback() const;  
```  
  
## Return Value  
 Returns `TRUE` is the class supports fallback calls. `FALSE` otherwise.
