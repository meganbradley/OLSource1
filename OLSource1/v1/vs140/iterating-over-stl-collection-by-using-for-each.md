# Iterating Over STL Collection By Using for each
The <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> keyword can be used to iterate over a Standard C++ Library (STL) collection.  
  
## All Platforms  
 **Remarks**  
  
 An STL collection is also known as a *container*. For more information, see [STL Containers](../vs140/stl-containers.md).  
  
## Examples  
 **Example**  
  
 The following code example uses <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> to iterate over a [\<map>](../vs140/-map-.md).  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 **Output**  
  
 **Months with 30 days = 4** **Example**
