# Resource Compiler Error RC2116
expecting number for ID  
  
 Expecting a number for the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> field of a control statement in the **DIALOG** statement. Make sure you have a number or <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement for the control ID.
