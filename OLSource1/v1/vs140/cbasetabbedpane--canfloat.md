# CBaseTabbedPane::CanFloat
Specifies whether the tabbed pane can float.  
  
## Syntax  
  
```  
virtual BOOL CanFloat() const;  
```  
  
## Return Value  
 `TRUE` if the pane can float; otherwise, `FALSE`.  
  
## Requirements  
 **Header:** afxbasetabbedpane.h
