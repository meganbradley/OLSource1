# CD2DResource::IsAutoDestroy
Check auto destroy flag.  
  
## Syntax  
  
```  
BOOL IsAutoDestroy() const;  
```  
  
## Return Value  
 TRUE if the object will be destroyed by its owner; otherwise FALSE.  
  
## Requirements  
 **Header:** afxrendertarget.h
