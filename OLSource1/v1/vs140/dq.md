# DQ
Allocates and optionally initializes 8 bytes of storage for each <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. Also can be used as a type specifier anywhere a type is legal. <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> is a synonym of [QWORD](../vs140/qword.md).  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
