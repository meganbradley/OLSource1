# list::size
Returns the number of elements in a list.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 The current length of the list.  
  
## Example  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 **List length is 1.**  
**List length is now 2.**   
## Requirements  
 **Header:** \<list>  
  
 **Namespace:** std
