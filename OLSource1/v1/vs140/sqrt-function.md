# sqrt Function
Calculates the squre root of the argument  
  
## Syntax  
  
```  
inline float sqrt(  
   float _X  
) restrict(amp);  
inline double sqrt(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the squre root of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
