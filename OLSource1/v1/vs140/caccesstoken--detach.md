# CAccessToken::Detach
Call this method to revoke ownership of the access token.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Returns the handle to the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> which has been detached.  
  
## Remarks  
 This method revokes the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>'s ownership of the access token.
