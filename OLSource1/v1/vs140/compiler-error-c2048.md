# Compiler Error C2048
more than one default  
  
 A <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement contains multiple <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> labels. Delete one of the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> labels to resolve the error.  
  
 The following sample generates C2048:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 Possible resolution:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
