# SBYTE (MASM)
Allocates and optionally initializes a signed byte of storage for each `initializer`. Can also be used as a type specifier anywhere a type is legal.  
  
## Syntax  
  
```  
  
name SBYTEinitializer [[, initializer]]...  
```
