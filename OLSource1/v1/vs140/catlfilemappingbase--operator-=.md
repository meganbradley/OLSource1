# CAtlFileMappingBase::operator =
Sets the current file-mapping object to another file-mapping object.  
  
## Syntax  
  
```  
  
      CAtlFileMappingBase& operator =(  
   CAtlFileMappingBase& orig   
);  
```  
  
#### Parameters  
 `orig`  
 The current file-mapping object.  
  
## Return Value  
 Returns a reference to the current object.  
  
## Requirements  
 **Header:** atlfile.h
