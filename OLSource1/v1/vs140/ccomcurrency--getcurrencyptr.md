# CComCurrency::GetCurrencyPtr
Returns the address of an `m_currency` data member.  
  
## Syntax  
  
```  
  
CURRENCY * GetCurrencyPtr( ) throw();  
  
```  
  
## Return Value  
 Returns the address of an `m_currency` data member  
  
## Requirements  
 **Header:** atlcur.h
