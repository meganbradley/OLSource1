# CComApartment::Apartment
Marks the thread's starting address.  
  
## Syntax  
  
```  
  
DWORD Apartment( );  
  
```  
  
## Return Value  
 Always 0.  
  
## Remarks  
 Automatically set during [CComAutoThreadModule::Init](../vs140/ccomautothreadmodule--init.md).  
  
## Requirements  
 **Header:** atlbase.h
