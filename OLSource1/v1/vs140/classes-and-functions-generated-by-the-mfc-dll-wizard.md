# Classes and Functions Generated by the MFC DLL Wizard
The code that the MFC DLL Wizard generates depends on the kind of DLL you are creating and the options you have selected. The MFC DLL Wizard generates the same code for both forms of regular DLLs.  
  
|Kind of DLL|Option|Classes|Functions|  
|-----------------|------------|-------------|---------------|  
|[Extension](../vs140/extension-dlls--overview.md)|None|None|<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>|  
|[Regular](../vs140/regular-dlls-dynamically-linked-to-mfc.md)|None|Application class derived from <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>|None|  
|[Regular](../vs140/regular-dlls-dynamically-linked-to-mfc.md)|Automation|Application class derived from <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>|**DllGetClassObjectDllCanUnloadNowDllRegisterServer**|  
|[Extension](../vs140/extension-dlls--overview.md)|Window Sockets|None|<CodeContentPlaceHolder>3\</CodeContentPlaceHolder>|  
|[Regular](../vs140/regular-dlls-dynamically-linked-to-mfc.md)|Window Sockets|Application class derived from <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>|<CodeContentPlaceHolder>5\</CodeContentPlaceHolder> contains call to <CodeContentPlaceHolder>6\</CodeContentPlaceHolder>|
