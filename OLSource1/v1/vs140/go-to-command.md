# Go To Command
Moves the cursor to the specified line.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Arguments  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 Optional. An integer representing the number of the line to go to.  
  
## Remarks  
 The line numbering begins at one. If the value of <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> is less than one, the first line displays. If the value of <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> is greater than the number of the last line, the last line displays.  
  
 If a value for <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> is not specified, the **Go To Line** dialog box displays.  
  
 The alias for this command is GoToLn.  
  
## Example  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
