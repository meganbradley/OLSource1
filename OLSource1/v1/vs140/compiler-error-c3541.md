# Compiler Error C3541
'type': typeid cannot be applied to a type that contains 'auto'  
  
 The [typeid](../vs140/typeid---c---component-extensions-.md) operator cannot be applied to the indicated type because it contains the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> specifier.  
  
## Example  
 The following example yields C3541.  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
