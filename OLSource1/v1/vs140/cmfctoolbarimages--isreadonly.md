# CMFCToolBarImages::IsReadOnly
Specifies whether the toolbar images are read-only.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the toolbar images are read-only, otherwise <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Remarks  
 The <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> object is read-only when the bitmap with toolbar images was loaded from a read-only file, or when the bitmap was copied in using the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>method.  
  
## Requirements  
 **Header:** afxtoolbarimages.h
