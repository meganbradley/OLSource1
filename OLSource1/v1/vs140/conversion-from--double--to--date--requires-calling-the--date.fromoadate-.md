# Conversion from &#39;Double&#39; to &#39;Date&#39; requires calling the &#39;Date.FromOADate&#39;
You have attempted to cast a <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> value to a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> value, which cannot be done without using the \<xref:System.DateTime.FromOADate*?displayProperty=fullName> method.  
  
 **Error ID:** BC30533  
  
### To correct this error  
  
-   Use the \<xref:System.DateTime.FromOADate*> method to convert the value.
