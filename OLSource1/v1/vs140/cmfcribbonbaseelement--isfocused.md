# CMFCRibbonBaseElement::IsFocused
Specifies whether the ribbon element has the focus.  
  
## Syntax  
  
```  
virtual BOOL IsFocused() const;  
```  
  
## Return Value  
 `TRUE` if the ribbon element has the focus; otherwise `FALSE`.  
  
## Requirements  
 **Header:** afxbaseribbonelement.h
