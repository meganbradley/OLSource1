# CHtmlView::GetWidth
Retrieves the width of the Internet Explorer main window.  
  
## Syntax  
  
```  
  
long GetWidth( ) const;  
  
```  
  
## Return Value  
 The current width of the window, in pixels.  
  
## Requirements  
 **Header:** afxhtml.h
