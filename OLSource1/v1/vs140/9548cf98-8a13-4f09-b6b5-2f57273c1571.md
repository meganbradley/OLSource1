# &#39;System.Runtime.InteropServices.DllImportAttribute&#39; cannot be applied to a Sub, Function, or Operator with a non-empty body
The <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> attribute was applied to a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, or <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> that is not empty.  
  
 **Error ID:** BC31522  
  
### To correct this error  
  
-   Remove all code from the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>, or <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> to use this attribute.
