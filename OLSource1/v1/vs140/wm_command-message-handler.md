# WM_COMMAND Message Handler
The following map entry corresponds to the function prototype.  
  
|Map entry|Function prototype|  
|---------------|------------------------|  
|ON_COMMAND(<id\>, <memberFxn\>)|afx_msg void memberFxn( );|
