# &#39;Inherits&#39; can appear only once within a &#39;Class&#39; statement and can only specify one class
More than one <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement appears in the same class, or an <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement specifies more than one class. A class can inherit from only one base class.  
  
 **Error ID:** BC30121  
  
### To correct this error  
  
-   Remove any extra <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statements and make sure the remaining <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statement specifies only one base class.
