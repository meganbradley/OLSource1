# The -moduleassemblyname option may only be specified when building a target of type &#39;module&#39;
The Visual Basic compiler has been passed the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> compiler option when the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> option is set to a value other than <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
 **Error ID:** BC2030  
  
### To correct this error  
  
1.  Remove the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> compiler option or set the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> option to <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>.
