# COleControl::GetHwnd
Implements the Get function of the stock hWnd property.  
  
## Syntax  
  
```  
  
OLE_HANDLE GetHwnd( );  
```  
  
## Return Value  
 The OLE control's window handle, if any; otherwise **NULL**.  
  
## Requirements  
 **Header:** afxctl.h
