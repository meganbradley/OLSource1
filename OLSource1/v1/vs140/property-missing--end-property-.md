# Property missing &#39;End Property&#39;
An <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement must match the block it terminates. A <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> definition has not been properly terminated.  
  
 **Error ID:** BC30025  
  
### To correct this error  
  
-   Add an <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement to the end of the property definition.
