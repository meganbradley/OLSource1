# Compiler Error C2450
switch expression of type 'type' is illegal  
  
 The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> expression evaluates to an invalid type. It must evaluate to an integer type or a class type with unambiguous conversion to an integer type. If it evaluates to a user-defined type, you must supply a conversion operator.  
  
 The following sample generates C2450:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
