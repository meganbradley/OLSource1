# find_end (STL-CLR)
Looks in a range for the last subsequence that is identical to a specified sequence or that is equivalent in a sense specified by a binary predicate.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This function behaves the same as the STL function <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>. For more information, see [find_end](../vs140/find_end.md).  
  
## Requirements  
 **Header:** \<cliext/algorithm>  
  
 **Namespace:** cliext
