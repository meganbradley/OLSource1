# ParamArray parameter must be a one-dimensional array
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> argument must be declared as a one-dimensional array type. The declaration specifies more than one dimension for the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> argument.  
  
 **Error ID:** BC30051  
  
### To correct this error  
  
-   Remove all commas inside the parentheses following the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> argument name.
