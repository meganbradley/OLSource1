# CBasePane::StretchPane
Stretches a pane vertically or horizontally.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 The length by which to stretch the pane.  
  
 [in] <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 If <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, stretch the pane vertically. If <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>, stretch the pane horizontally.  
  
## Return Value  
 The size of the stretched pane.
