# Linker Tools Warning LNK4076
invalid incremental status file 'filename'; linking nonincrementally  
  
 LINK cannot write to the incremental status (.ilk) file. Either <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> is corrupt or it is not an incremental linking database. Remove the file and relink.
