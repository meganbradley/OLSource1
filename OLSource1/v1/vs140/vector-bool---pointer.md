# vector&lt;bool&gt;::pointer
A type that describes an object that can serve as a pointer to a Boolean element of the sequence contained by the `vector<bool>` object.  
  
## Syntax  
  
```  
typedef iterator pointer;  
```  
  
## Requirements  
 **Header:** <vector\>  
  
 **Namespace:** std
