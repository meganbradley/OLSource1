# ICommandUI::Text
Sets the text of the user interface item for this command.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This property sets the text of the user interface item for this command. Set <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> to a text string handle.  
  
## Requirements  
 **Header:** afxwinforms.h
