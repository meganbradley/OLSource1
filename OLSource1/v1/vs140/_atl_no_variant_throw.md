# _ATL_NO_VARIANT_THROW
Suppresses the [CComVariant Class](../vs140/ccomvariant-class.md) from throwing exceptions.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 Define this macro if your module must not throw exceptions. When this macro is defined, <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> will not throw exceptions. By default, this macro is not defined.
