# floor Function
Calculates the floor of the argument  
  
## Syntax  
  
```  
inline float floor(  
   float _X  
) restrict(amp);  
inline double floor(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the floor of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
