# CV_CPU_TYPE_e
Specifies the target processor.  
  
> [!NOTE]
>  The processor is identified in the enumeration element after the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> prefix.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 The values in this enumeration are returned by a call to the [IDiaSymbol::get_platform](../vs140/idiasymbol--get_platform.md) method.  
  
## Requirements  
 Header: cvconst.h
