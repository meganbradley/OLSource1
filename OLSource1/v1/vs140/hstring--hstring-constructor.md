# HString::HString Constructor
Initializes a new instance of the HString class.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 An HSTRING handle.  
  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 An existing HString object.  
  
## Remarks  
 The first constructor initializes a new HString object that is empty.  
  
 The second constructor initializes a new HString object to the value of the existing <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> parameter, and then destroys the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> parameter.
