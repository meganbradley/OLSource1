# Operator &#39;operator&#39; is not defined for types &#39;type1&#39; and &#39;type2&#39;
Operator 'operator' is not defined for types 'type1' and 'type2'. You can use the 'Value' property to get the string value of the first element of '\<parentElement>'.  
  
 An XML literal has been used in a binary or <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> comparison.  
  
 **Error ID:** BC31195  
  
### To correct this error  
  
-   Use the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> property of the XML literal for the binary or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> comparison.
