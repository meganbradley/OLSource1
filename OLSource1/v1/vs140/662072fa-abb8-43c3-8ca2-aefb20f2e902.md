# Parameter &#39;&lt;parametername&gt;&#39; in extension method &#39;&lt;methodname&gt;&#39; defined in &#39;&lt;typename&gt;&#39; already has a matching omitted argument
A procedure call to an extension method omits an argument by position and then supplies the argument by name. For example, the following call to extension method <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> first omits an argument for parameter <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, and then supplies it by name.  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 **Error ID:** BC36583  
  
### To correct this error  
  
-   Supply the argument by position, or remove the comma that omits it.
