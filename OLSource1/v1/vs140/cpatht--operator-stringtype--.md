# CPathT::operator StringType &amp;
This operator allows the object to be treated like a string.  
  
## Syntax  
  
```  
  
operator StringType&( ) throw( );  
  
```  
  
## Return Value  
 Returns a string representing the current path managed by this object.  
  
## Requirements  
 **Header:** atlpath.h
