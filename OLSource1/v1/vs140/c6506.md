# C6506
warning C6506: invalid annotation: \<name> property may only be used on values of pointer or array types  
  
 This warning indicates that a property is used on a type other than pointer or array types. The Access, Tainted, and Valid properties can be used on all data types. Other properties, such as ValidBytesConst, ValidElementsConst, ElementSize, and NullTerminted support pointer, pointer to members, or array types. For a complete list of properties and the supported data types, see [Annotation Properties](assetId:///f77b4370-6bda-4294-bd2a-e7d0df182a3d).  
  
## Example  
 The following code generates this warning:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 To correct this warning, use a pointer or an array type, as shown in the following sample code:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
