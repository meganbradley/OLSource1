# CRecentFileList::GetSize
Retrieves the number of files in the MRU file list.  
  
## Syntax  
  
```  
  
int GetSize( ) const;  
  
```  
  
## Return Value  
 The number of files in the current most recently used (MRU) file list.  
  
## Requirements  
 **Header:** afxadv.h
