# CReBarCtrl::GetBarHeight
Retrieves the height of the rebar bar.  
  
## Syntax  
  
```  
  
UINT GetBarHeight( ) const;  
  
```  
  
## Return Value  
 Value that represents the height, in pixels, of the control.  
  
## Requirements  
 **Header:** afxcmn.h
