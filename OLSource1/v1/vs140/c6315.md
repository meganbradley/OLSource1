# C6315
warning C6315: Incorrect order of operations: bitwise-and has higher precedence than bitwise-or. Add parentheses to clarify intent  
  
 This warning indicates that an expression in a test context contains both bitwise-and (<CodeContentPlaceHolder>2\</CodeContentPlaceHolder>) and bitwise-or (<CodeContentPlaceHolder>3\</CodeContentPlaceHolder>) operations, but causes a constant because the bitwise-or operation happens last. Parentheses should be added to clarify intent.  
  
## Example  
 The following code generates this warning:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 To correct this warning, add parenthesis as shown in the following code:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
