# ParameterGroup Element
Contains an optional list of parameters that will be present on the task that is generated by a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>. For more information, see [UsingTask](../vs140/usingtask-element--msbuild-.md).  
  
 \<Project>  
 \<UsingTask>  
 \<ParameterGroup>  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Attributes and Elements  
 The following sections describe attributes, child elements, and parent elements.  
  
### Attributes  
 None.  
  
### Child Elements  
  
|Element|Description|  
|-------------|-----------------|  
|[Parameter](../vs140/parameter-element.md)|Contains information about a specific parameter for a task that is generated by a <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>. The name of the element is the name of the parameter.|  
  
### Parent Elements  
  
|Element|Description|  
|-------------|-----------------|  
|[UsingTask](../vs140/usingtask-element--msbuild-.md)|Provides a way to register tasks in [!INCLUDE[vstecmsbuild](../vs140/includes/vstecmsbuild_md.md)]. There may be zero or more <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> elements in a project.|

 The following example shows how to use the <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> element.
