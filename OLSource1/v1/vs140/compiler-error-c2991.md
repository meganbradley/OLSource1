# Compiler Error C2991
redefinition of type parameter 'parameter'  
  
 There was a type conflict between two generic or template definitions of <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>. When defining multiple generic or template parameters, you must use equivalent types.  
  
 The following sample generates C2991:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 C2991 can also occur when using generics:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
