# Compiler Error C3168
'type' : illegal underlying type for enum  
  
 The underlying type you specified for the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> type was not valid. The underlying type must be an integral C++ type or a corresponding CLR type.  
  
 The following sample generates C3168:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 The following sample generates C3168:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
