# COleControlSite::IsDefaultButton
Determines if the control is the default button.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 Nonzero if the control is the default button on the window, otherwise zero.

 **Header:** afxocc.h
