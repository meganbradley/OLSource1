# Compiler Error CS1675
Enums cannot have type parameters  
  
 To resolve this error, remove the type parameter from the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> declaration.  
  
## Example  
 The following sample generates CS1675:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
