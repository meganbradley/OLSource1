# virtual (C++)
The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> keyword declares a virtual function or a virtual base class.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 Specifies the return type of the virtual member function.  
  
 <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>  
 Declares a member function.  
  
 <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>  
 Defines the level of access to the base class, <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>7\</CodeContentPlaceHolder>. Can appear before or after the <CodeContentPlaceHolder>8\</CodeContentPlaceHolder> keyword.  
  
 <CodeContentPlaceHolder>9\</CodeContentPlaceHolder>  
 Identifies a previously declared class type.
