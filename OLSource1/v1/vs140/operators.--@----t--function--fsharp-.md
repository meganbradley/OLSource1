# Operators.( @ )&lt;&#39;T&gt; Function (F#)

 **Namespace/Module Path:** Microsoft.FSharp.Core.Operators

## Syntax  
  
```  
// Signature:  
( @ ) : 'T list -> 'T list -> 'T list  
  
// Usage:  
list1 @ list2  
```  
  
#### Parameters  
 `list1`  
 Type: `'T list`

 `list2`  
 Type: `'T list`

## Return Value
