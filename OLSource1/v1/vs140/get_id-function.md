# get_id Function
Uniquely identifies the current thread of execution.  
  
## Syntax  
  
```  
thread::id this_thread::get_id() _NOEXCEPT;  
```  
  
## Return Value  
 An object of type [thread::id](../vs140/thread--id-class.md) that uniquely identifies the current thread of execution.  
  
## Requirements  
 **Header:** thread  
  
 **Namespace:** std
