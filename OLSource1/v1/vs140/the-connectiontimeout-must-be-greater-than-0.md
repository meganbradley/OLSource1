# The ConnectionTimeout must be greater than 0
When uploading and downloading files with the [My.Computer.Network Object](../vs140/my.computer.network-object.md), you must specify a <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> greater than <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.  
  
### To correct this error  
  
-   Supply a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> that is greater than <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.
