# CTokenGroups::GetLength
Returns the size of the **CTokenGroup** object.  
  
## Syntax  
  
```  
  
UINT GetLength( ) const throw( );  
  
```  
  
## Remarks  
 Returns the total size of the **CTokenGroup** object, in bytes.  
  
## Requirements  
 **Header:** atlsecurity.h
