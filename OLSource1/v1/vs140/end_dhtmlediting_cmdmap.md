# END_DHTMLEDITING_CMDMAP
Marks the end of a DHTML editing command map.  
  
## Syntax  
  
```  
  
END_DHTMLEDITING_CMDMAP( )  
```  
  
## Remarks  
 Use in conjunction with [BEGIN_DHTMLEDITING_CMDMAP](../vs140/begin_dhtmlediting_cmdmap.md).
