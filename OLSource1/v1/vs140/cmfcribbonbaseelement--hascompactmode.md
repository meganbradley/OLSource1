# CMFCRibbonBaseElement::HasCompactMode
Specifies whether the ribbon element has a compact mode.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the ribbon element has a compact mode. <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> otherwise.  
  
## Remarks  
  
> [!NOTE]
>  In the compact mode, an element displays a small image only.

 **Header:** afxbaseribbonelement.h
