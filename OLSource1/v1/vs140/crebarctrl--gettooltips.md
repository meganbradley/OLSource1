# CReBarCtrl::GetToolTips
Implements the behavior of the Win32 message [RB_GETTOOLTIPS](http://msdn.microsoft.com/library/windows/desktop/bb774477), as described in the [!INCLUDE[winSDK](../vs140/includes/winsdk_md.md)].  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 A pointer to a [CToolTipCtrl](../vs140/ctooltipctrl-class.md) object.  
  
## Remarks  
 Note that the MFC implementation of <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> returns a pointer to a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, rather than an <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.  
  
## Requirements  
 **Header:** afxcmn.h
