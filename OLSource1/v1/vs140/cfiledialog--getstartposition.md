# CFileDialog::GetStartPosition
Call this member function to retrieve the position of the first file pathname in the list, if <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> has the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> flag set.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 A **POSITION** value that can be used for iteration; **NULL** if the list is empty.  
  
## Requirements  
 **Header:** afxdlgs.h
