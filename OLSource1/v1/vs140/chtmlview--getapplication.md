# CHtmlView::GetApplication
Call this member function to retrieve the automation object supported by the application that contains the WebBrowser control.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 A pointer to the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> interface of the active document object. For more information, see [Implementing the IDispatch Interface](assetId:///0e171f7f-0022-4e9b-ac8e-98192828e945).  
  
## Remarks  
 Applies to Internet Explorer and WebBrowser.
