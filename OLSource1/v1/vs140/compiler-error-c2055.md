# Compiler Error C2055
expected formal parameter list, not a type list  
  
 A function definition contains a parameter type list instead of a formal parameter list. ANSI C requires formal parameters to be named unless they are void or an ellipsis (<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>).  
  
 The following sample generates C2055:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
