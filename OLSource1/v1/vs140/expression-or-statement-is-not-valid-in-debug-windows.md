# Expression or statement is not valid in debug windows
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statements cannot be used in the debugging context.  
  
 **Error ID:** BC30706  
  
### To correct this error  
  
1.  Remove <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statements from debugging code.
