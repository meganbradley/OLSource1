# Compiler Error CS0027
Keyword 'this' is not available in the current context  
  
 The [this](../vs140/this--csharp-reference-.md) keyword was found outside of a property, method, or constructor.  
  
 To fix this error, either modify the statement to eliminate use of the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> keyword, and/or move part or all of the statement inside a property, method, or constructor.  
  
 The following example generates CS0027:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
