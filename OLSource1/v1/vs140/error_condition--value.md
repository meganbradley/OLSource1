# error_condition::value
Returns the stored error code value.  
  
## Syntax  
  
```  
value_type value() const;  
```  
  
## Return Value  
 The stored error code value of type [value_type](../vs140/error_condition--value_type.md).  
  
## Remarks  
  
## Requirements  
 **Header:** \<system_error>  
  
 **Namespace:** std
