# &lt;publisherIdentity&gt; Element (ClickOnce Deployment)
Contains information about the publisher that signed this deployment manifest.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Elements and Attributes  
 The <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> element is required for signed manifests. The following table shows the attributes that the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> element supports.  
  
|Attribute|Description|  
|---------------|-----------------|  
|<CodeContentPlaceHolder>3\</CodeContentPlaceHolder>|Required. Describes the identity of the party that published this application.|  
|<CodeContentPlaceHolder>4\</CodeContentPlaceHolder>|Required. Contains the SHA-1 hash of the public key of the certificate issuer.|
