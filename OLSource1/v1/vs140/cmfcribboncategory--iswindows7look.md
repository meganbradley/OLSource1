# CMFCRibbonCategory::IsWindows7Look
Indicates whether the parent ribbon has Windows 7 look (small rectangular application button).  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the parent ribbon has Windows 7 look; otherwise <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.
