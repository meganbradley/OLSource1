# move_iterator::base
Returns the stored iterator for this `move_iterator`.  
  
## Syntax  
  
```  
RandomIterator base() const;  
```  
  
## Property Value/Return Value  
 Returns the stored iterator for the `move_iterator`.  
  
## Remarks  
 The member function returns the stored iterator.  
  
## Requirements  
 **Header:** \<iterator>  
  
 **Namespace:** std
