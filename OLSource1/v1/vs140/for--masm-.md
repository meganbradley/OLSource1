# FOR (MASM)
Marks a block that will be repeated once for each <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, with the current <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> replacing <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> on each repetition.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 Same as [IRP](../vs140/irp.md).
