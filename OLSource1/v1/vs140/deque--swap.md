# deque::swap
Exchanges the elements of two deques.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 The deque providing the elements to be swapped, or the deque whose elements are to be exchanged with those of the deque <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>.  
  
 <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>  
 A deque whose elements are to be exchanged with those of the deque <CodeContentPlaceHolder>5\</CodeContentPlaceHolder>.  
  
## Example  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 **The original deque c1 is: 1 2 3**  
**After swapping with c2, deque c1 is: 10 20**  
**After swapping with c3, deque c1 is: 100**  
**After swapping with c2, deque c1 is: 1 2 3**
