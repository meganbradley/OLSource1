# COleSafeArray::GetElemSize
Retrieves the size of an element in a `COleSafeArray` object.  
  
## Syntax  
  
```  
  
DWORD GetElemSize( );  
```  
  
## Return Value  
 The size, in bytes, of the elements of a safe array.  
  
## Requirements  
 **Header:** afxdisp.h
