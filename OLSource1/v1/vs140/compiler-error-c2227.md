# Compiler Error C2227
left of '->member' must point to class/struct/union/generic type  
  
 The operand to the left of <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> is not a pointer to a class, structure, or union.  
  
 The following sample generates C2227:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
