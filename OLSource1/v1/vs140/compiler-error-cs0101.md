# Compiler Error CS0101
The namespace 'namespace' already contains a definition for 'type'  
  
 A [namespace](../vs140/namespace--csharp-reference-.md) has duplicate identifiers. Rename or delete one of the duplicate identifiers. For more information, see [Namespaces (C# Programmer's Reference)](../vs140/namespaces--csharp-programming-guide-.md)
