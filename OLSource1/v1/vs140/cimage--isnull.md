# CImage::IsNull
Determines if a bitmap is currently loaded.  
  
## Syntax  
  
```  
  
bool IsNull( ) const throw( );  
  
```  
  
## Remarks  
 This method returns **True** if a bitmap is not currently loaded; otherwise **False**.  
  
## Requirements  
 **Header:** atlimage.h
