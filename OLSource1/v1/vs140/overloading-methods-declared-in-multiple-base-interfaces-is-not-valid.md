# Overloading methods declared in multiple base interfaces is not valid
Multiple inherited interfaces implicitly overload the same method.  
  
 **Error ID:** BC31410  
  
### To correct this error  
  
-   Use the <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> modifier instead of the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> modifier.
