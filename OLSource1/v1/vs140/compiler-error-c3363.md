# Compiler Error C3363
'type' : 'typeid' can only be applied to a type  
  
 The [typeid](../vs140/typeid---c---component-extensions-.md) operator was used incorrectly.  
  
## Example  
 The following sample generates C3363.  
  
```  
// C3363.cpp  
// compile with: /clr  
int main() {  
   System::typeid;   // C3363  
}  
```
