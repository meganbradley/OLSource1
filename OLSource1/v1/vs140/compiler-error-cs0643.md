# Compiler Error CS0643
'arg' duplicate named attribute argument  
  
 A parameter, <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>, on a user-defined attribute was specified twice. For more information, see [Attributes (C# Programmers Reference)](../vs140/attributes--csharp-and-visual-basic-.md).  
  
## Example  
 The following sample generates CS0643:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
