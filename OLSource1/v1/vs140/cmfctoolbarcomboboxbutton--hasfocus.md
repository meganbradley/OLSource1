# CMFCToolBarComboBoxButton::HasFocus
Indicates whether the combo box currently has the focus.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the combo box currently has the focus; otherwise, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Remarks  
 This method also returns <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> if any child window of the combo box currently has the focus.
