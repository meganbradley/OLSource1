# CHtmlEditCtrlBase::SetDesignMode
Set design mode.  
  
## Syntax  
  
```  
  
      BOOL SetDesignMode(  
   BOOL bMode   
) const;  
```  
  
#### Parameters  
 `bMode`  
 If true, turns design mode on.  
  
## Return Value  
 Returns TRUE on success, FALSE on failure.  
  
## Requirements  
 **Header:** afxhtml.h
