# CMFCPopupMenu::IsQuickCustomize
Determines whether the associated [CMFCToolBarMenuButton Class](../vs140/cmfctoolbarmenubutton-class.md) is in QuickCustomize mode.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if the associated menu button is in QuickCustomize mode; otherwise <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>. This method will also return <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> if the pop-up menu is not associated with a <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
## Remarks  
 In QuickCustomize mode the user selects a button on a toolbar to customize the button directly.

 **Header:** afxpopupmenu.h
