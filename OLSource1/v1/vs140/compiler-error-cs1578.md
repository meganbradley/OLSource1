# Compiler Error CS1578
Filename, single-line comment or end-of-line expected  
  
 After a [#line](../vs140/sharpline--csharp-reference-.md) directive, only a file name (in double quotation marks) or a single-line comment is allowed.  
  
 The following sample generates CS1578:
