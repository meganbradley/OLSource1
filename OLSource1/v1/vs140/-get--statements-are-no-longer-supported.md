# &#39;Get&#39; statements are no longer supported
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statements are no longer supported. File I/O functionality is available in the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> namespace.  
  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> is not supported for file operations, and can only be used in property procedure syntax.  
  
 **Error ID:** BC30829  
  
### To correct this error  
  
1.  Perform file operations using the members of <CodeContentPlaceHolder>3\</CodeContentPlaceHolder>, <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>, and [!INCLUDE[vbprvb](../vs140/includes/vbprvb_md.md)] run-time functions.
