# Compiler Warning (level 1) C4227
anachronism used : qualifiers on reference are ignored  
  
 Using qualifiers like <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> with C++ references is an outdated practice.  
  
## Example  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
