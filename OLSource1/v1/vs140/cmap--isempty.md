# CMap::IsEmpty
Determines whether the map is empty.  
  
## Syntax  
  
```  
  
BOOL IsEmpty( ) const;  
```  
  
## Return Value  
 Nonzero if this map contains no elements; otherwise 0.  
  
## Example  
 See the example for [CMap::RemoveAll](../vs140/cmap--removeall.md).
