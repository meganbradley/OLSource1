# Compiler Error C2856
\#pragma hdrstop cannot be inside an #if block  
  
 The <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> pragma cannot be placed inside the body of a conditional compilation block.  
  
 Move the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement to an area that is not contained in an <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> block.
