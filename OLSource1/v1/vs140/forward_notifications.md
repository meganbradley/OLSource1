# FORWARD_NOTIFICATIONS
Forwards notification messages to the parent window.  
  
## Syntax  
  
```  
  
FORWARD_NOTIFICATIONS( )  
  
```  
  
## Remarks  
 Specify this macro as part of your message map.  
  
## Requirements  
 **Header:** atlwin.h
