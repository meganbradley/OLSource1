# CMFCTabCtrl::HideNoTabs
Enables or disables drawing of the tab area if there are no visible tabs.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> to enable drawing the tab area; <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> to disable drawing. The default value is <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.
