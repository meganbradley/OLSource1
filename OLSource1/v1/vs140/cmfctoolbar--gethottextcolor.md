# CMFCToolBar::GetHotTextColor
Returns the text color of the highlighted toolbar buttons.  
  
## Syntax  
  
```  
static COLORREF GetHotTextColor();  
```  
  
## Return Value  
 A [COLORREF](http://msdn.microsoft.com/library/windows/desktop/dd183449) value that represent the current highlighted text color.  
  
## Remarks  
 Call [CMFCToolBar::SetHotTextColor](../vs140/cmfctoolbar--sethottextcolor.md) to set a new text color for highlighted toolbar buttons.  
  
## Requirements  
 **Header:** afxtoolbar.h
