# IConnectionPointImpl::m_vec
Manages the connections between the connection point object and a sink.  
  
## Syntax  
  
```  
  
CDV m_vec;  
```  
  
## Remarks  
 By default, `m_vec` is of type [CComDynamicUnkArray](../vs140/ccomdynamicunkarray-class.md).  
  
## Requirements  
 **Header:** atlcom.h
