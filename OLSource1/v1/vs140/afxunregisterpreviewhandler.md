# AfxUnRegisterPreviewHandler
A helper to unregister a preview handler.  
  
## Syntax  
  
```  
BOOL AFXAPI AfxUnRegisterPreviewHandler(LPCTSTR lpszCLSID);  
```  
  
#### Parameters  
 `lpszCLSID`  
 Specifies the CLSID of the handler to be unregistered.
