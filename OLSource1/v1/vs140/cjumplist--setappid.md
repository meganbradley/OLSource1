# CJumpList::SetAppID
Sets the Application User Model ID for the list that will be built.  
  
## Syntax  
  
```  
void SetAppID(  
   LPCTSTR strAppID  
);  
```  
  
#### Parameters  
 `strAppID`  
 A string that specifies the Application User Model ID.  
  
## Requirements  
 **Header:** afxadv.h
