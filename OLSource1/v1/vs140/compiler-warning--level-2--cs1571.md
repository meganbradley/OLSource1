# Compiler Warning (level 2) CS1571
XML comment on 'construct' has a duplicate param tag for 'parameter'  
  
 When using the [/doc](../vs140/-doc--csharp-compiler-options-.md) compiler option, multiple comments were found for the same method parameter. Remove one of the duplicate lines.
