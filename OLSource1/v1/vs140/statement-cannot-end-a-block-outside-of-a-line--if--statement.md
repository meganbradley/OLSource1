# Statement cannot end a block outside of a line &#39;If&#39; statement
A single-line <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement contains several statements separated by colons (:), one of which is an <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> statement for a control block outside the single-line <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>. Single-line <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> statements do not use the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> statement.  
  
 **Error ID:** BC32005  
  
### To correct this error  
  
-   Move the single-line <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> statement outside the control block that contains the <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> statement.
