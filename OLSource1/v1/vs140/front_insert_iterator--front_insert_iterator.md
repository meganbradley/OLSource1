# front_insert_iterator::front_insert_iterator
Creates an iterator that can insert elements at the front of a specified container object.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>  
 The container object into which the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> is to insert elements.  
  
## Return Value  
 A <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> for the parameter container object.  
  
## Example  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 **The list L is:**  
 **( -2 0 2 4 6 8 10 12 14 16 ).**  
**After the front insertions, the list L is:**  
 **( 30 20 -2 0 2 4 6 8 10 12 14 16 ).**
