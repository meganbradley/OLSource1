# CJumpList::AddTaskSeparator
Adds a separator between tasks.  
  
## Syntax  
  
```  
BOOL AddTaskSeparator();  
```  
  
## Return Value  
 Nonzero if it is successful, 0 if it is not.  
  
## Requirements  
 **Header:** afxadv.h
