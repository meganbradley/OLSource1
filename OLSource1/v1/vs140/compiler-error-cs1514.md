# Compiler Error CS1514
{ expected  
  
 The compiler expected an opening curly brace (<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>) that was not found.  
  
 The following sample generates CS1514:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>
