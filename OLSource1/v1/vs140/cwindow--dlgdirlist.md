# CWindow::DlgDirList
Fills a list box with the names of all files matching a specified path or file name.  
  
## Syntax  
  
```  
  
      int DlgDirList(  
   LPTSTR lpPathSpec,  
   int nIDListBox,  
   int nIDStaticPath,  
   UINT nFileType   
) throw();  
```
