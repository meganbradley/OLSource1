# atanh Function
Calculates the inverse hyperbolic tangent of the argument  
  
## Syntax  
  
```  
inline float atanh(  
   float _X  
) restrict(amp);  
inline double atanh(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the inverse hyperbolic tangent value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
