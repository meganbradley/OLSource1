# OperatorIntrinsics.PowGeneric&lt;&#39;T&gt; Function (F#)
This is a library intrinsic. Calls to this function may be generated by uses of the generic [pown](../vs140/operators.pown-^t--function--fsharp-.md) operator.  
  
 **Namespace/Module Path:** Microsoft.FSharp.Core.Operators.OperatorIntrinsics  
  
 **Assembly:** FSharp.Core (in FSharp.Core.dll)  
  
## Syntax  
  
```  
// Signature:  
PowGeneric : 'T * ('T -> 'T -> 'T) * 'T * int -> 'T  
  
// Usage:  
PowGeneric (one, mul, value, exponent)  
```  
  
#### Parameters  
 `one`  
 Type: `'T`  
  
 The value representing 1 for the type parameter.  
  
 `mul`  
 Type: `'T -> 'T -> 'T`  
  
 The multiplication operation for the type parameter.  
  
 `value`  
 Type: `'T`  
  
 The value to be raised to a power.  
  
 `exponent`  
 Type: [int](../vs140/core.int-type-abbreviation--fsharp-.md)  
  
 The power to raise the value to.  
  
## Return Value  
 The result of the exponentiation operation.  
  
## Remarks  
 This function is for use by compiled F# code and should not be used directly.
