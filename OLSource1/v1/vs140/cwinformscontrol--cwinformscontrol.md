# CWinFormsControl::CWinFormsControl
Constructs an MFC Windows Forms control wrapper object.  
  
## Syntax  
  
```  
CWinFormsControl( );  
```  
  
## Remarks  
 The Windows Forms control is instantiated when you call [CreateManagedControl](../vs140/cwinformscontrol--createmanagedcontrol.md).  
  
## Requirements  
 **Header:** afxwinforms.h
