# AfxThrowResourceException
Throws a resource exception.  
  
## Syntax  
  
```  
  
void AfxThrowResourceException( );  
```  
  
## Remarks  
 This function is normally called when a Windows resource cannot be loaded.  
  
## Requirements  
 **Header:** afxwin.h
