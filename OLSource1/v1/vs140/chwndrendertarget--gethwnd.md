# CHwndRenderTarget::GetHwnd
Returns the HWND associated with this render target.  
  
## Syntax  
  
```  
HWND GetHwnd() const;  
```  
  
## Return Value  
 The HWND associated with this render target.  
  
## Requirements  
 **Header:** afxrendertarget.h
