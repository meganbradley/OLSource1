# &#39;Next&#39; must be preceded by a matching &#39;For&#39;
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement occurs without a corresponding <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> statement. <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> must be preceded by a corresponding <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> or <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> statement.  
  
 **Error ID:** BC30092  
  
### To correct this error  
  
1.  If this <CodeContentPlaceHolder>6\</CodeContentPlaceHolder> loop is part of a set of nested <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> loops, make sure each loop is properly terminated.  
  
2.  Verify that other control structures within the <CodeContentPlaceHolder>8\</CodeContentPlaceHolder> loop are correctly terminated.  
  
3.  Ensure that this <CodeContentPlaceHolder>9\</CodeContentPlaceHolder> loop is correctly formatted.
