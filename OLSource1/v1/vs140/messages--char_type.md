# messages::char_type
A character type that is used display messages.  
  
## Syntax  
  
```  
typedef CharType char_type;  
```  
  
## Remarks  
 The type is a synonym for the template parameter **CharType**.  
  
## Requirements  
 **Header:** \<locale>  
  
 **Namespace:** std
