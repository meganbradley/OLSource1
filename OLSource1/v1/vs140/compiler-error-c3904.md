# Compiler Error C3904
'property_accessor': must specify number parameter(s)  
  
 Check the number of parameters in your <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> and <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> methods against property dimensions.  
  
-   The number of parameters for the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> method must equal the number of dimensions of the property or be zero for nonindexed properties.  
  
-   The number of parameters of the <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> method must be one more than the number of dimensions of the property.  
  
 For more information, see [property](../vs140/property---c---component-extensions-.md).
