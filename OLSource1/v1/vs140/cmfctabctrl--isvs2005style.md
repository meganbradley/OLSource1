# CMFCTabCtrl::IsVS2005Style
Indicates whether tabs are drawn using the style of Visual Studio 2005.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> if tabs are drawn using the style of Visual Studio 2005; otherwise, <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.  
  
## Remarks  
 Use the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> parameter of the [CMFCTabCtrl::Create](../vs140/cmfctabctrl--create.md) method to specify how tabs are drawn.

 **Header:** afxtabctrl.h
