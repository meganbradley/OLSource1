# Compiler Error C2129
static function 'function' declared but not defined  
  
 A forward reference is made to a <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> function that is never defined.  
  
 A <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> function must be defined within file scope. If the function is defined in another file, it must be declared <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.
