# CToolBar::SetBitmap
Call this member function to set the bitmap image for the toolbar.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 *hbmImageWell*  
 Handle of a bitmap image that is associated with a toolbar.  
  
## Return Value  
 Nonzero if successful; otherwise 0.  
  
## Remarks  
 For example, call <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> to change the bitmapped image after the user takes an action on a document that changes the action of a button.  
  
## Requirements  
 **Header:** afxext.h
