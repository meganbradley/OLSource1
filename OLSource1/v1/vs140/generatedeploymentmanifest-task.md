# GenerateDeploymentManifest Task
Generates a [!INCLUDE[ndptecclick](../vs140/includes/ndptecclick_md.md)] deployment manifest. A [!INCLUDE[ndptecclick](../vs140/includes/ndptecclick_md.md)] deployment manifest describes the deployment of an application by defining a unique identity for the deployment, identifying deployment traits such as install or online mode, specifying application update settings and update locations, and indicating the corresponding [!INCLUDE[ndptecclick](../vs140/includes/ndptecclick_md.md)] application manifest.
