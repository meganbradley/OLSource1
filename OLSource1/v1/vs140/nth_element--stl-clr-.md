# nth_element (STL-CLR)
Partitions a range of elements, correctly locating the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>th element of the sequence in the range so that all the elements in front of it are less than or equal to it and all the elements that follow it in the sequence are greater than or equal to it.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 This function behaves the same as the STL function <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>. For more information, see [nth_element](../vs140/nth_element.md).  
  
## Requirements  
 **Header:** \<cliext/algorithm>  
  
 **Namespace:** cliext
