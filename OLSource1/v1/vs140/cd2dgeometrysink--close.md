# CD2DGeometrySink::Close
Closes the geometry sink  
  
## Syntax  
  
```  
BOOL Close();  
```  
  
## Return Value  
 Nonzero if successful; otherwise FALSE.  
  
## Requirements  
 **Header:** afxrendertarget.h
