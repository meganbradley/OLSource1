# ref (C++)
Identifies a reference pointer.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 The <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> C++ attribute has the same functionality as the [ref](http://msdn.microsoft.com/library/windows/desktop/aa367153) MIDL attribute.  
  
## Example  
 The following code shows how to use the <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> attribute:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
