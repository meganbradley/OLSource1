# CWindow::GetScrollInfo
Retrieves the parameters of a scroll bar.  
  
## Syntax  
  
```  
  
      BOOL GetScrollInfo(  
   int nBar,  
   LPSCROLLINFO lpScrollInfo   
) throw();  
```
