# Compiler Warning (level 1) CS1691
'number' is not a valid warning number  
  
 A number that was passed to the [#pragma warning](../vs140/sharppragma-warning--csharp-reference-.md) preprocessor directive was not a valid warning number. Verify that the number represents a warning, not an error or another sequence of characters.
