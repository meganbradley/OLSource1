# CMFCTabCtrl::EnableActiveTabCloseButton
Shows or hides a Close button (**X**) on the active tab.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> to display the Close button on the active tab; <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> to display the Close button on the upper-right corner of the tab area. The default value is <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
## Requirements  
 **Header:** afxtabctrl.h
