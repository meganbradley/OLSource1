# &#39;Lib&#39; expected
A <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement specifies a library name without using the <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> keyword.  
  
 **Error ID:** BC30218  
  
### To correct this error  
  
-   Add the <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> keyword before the library name.
