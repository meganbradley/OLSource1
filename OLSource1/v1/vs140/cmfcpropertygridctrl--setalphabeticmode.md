# CMFCPropertyGridCtrl::SetAlphabeticMode
Sets or resets alphabetic mode.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
#### Parameters  
 [in] <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>  
 <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> to set alphabetic mode; <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> reset alphabetic mode. The default value is <CodeContentPlaceHolder>4\</CodeContentPlaceHolder>.  
  
## Remarks  
 When the property grid control is in alphabetic mode, the control sorts all the properties it contains by their property name.
