# CComObjectStack::m_hResFinalConstruct
Contains the `HRESULT` returned from calling `FinalConstruct` during construction of the `CComObjectStack` object.  
  
## Syntax  
  
```  
  
HRESULT m_hResFinalConstruct;  
```
