# CMFCTasksPaneTaskGroup::m_bIsSpecial
Determines whether the task group is *special* and whether the caption for a special task group should be identified by a different color.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Remarks  
 If your application is using the Windows XP visual theme and <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> is <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>, the framework calls <CodeContentPlaceHolder>3\</CodeContentPlaceHolder> with the <CodeContentPlaceHolder>4\</CodeContentPlaceHolder> flag. If <CodeContentPlaceHolder>5\</CodeContentPlaceHolder> is <CodeContentPlaceHolder>6\</CodeContentPlaceHolder>, the framework calls <CodeContentPlaceHolder>7\</CodeContentPlaceHolder> with the <CodeContentPlaceHolder>8\</CodeContentPlaceHolder> flag.
