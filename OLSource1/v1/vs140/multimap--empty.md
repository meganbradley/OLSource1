# multimap::empty
Tests if a multimap is empty.  
  
## Syntax  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
## Return Value  
 **true** if the multimap is empty; **false** if the multimap is nonempty.  
  
## Example

**The multimap m2 is empty.**
