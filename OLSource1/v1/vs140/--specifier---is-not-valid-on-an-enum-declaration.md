# &#39;&lt;specifier&gt;&#39; is not valid on an Enum declaration
You have used a specifier that is not valid on an <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> declaration.  
  
 **Error ID:** BC30396  
  
### To correct this error  
  
1.  Remove the specifier.
