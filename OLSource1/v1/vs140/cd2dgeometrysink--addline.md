# CD2DGeometrySink::AddLine
Creates a line segment between the current point and the specified end point and adds it to the geometry sink.  
  
## Syntax  
  
```  
void AddLine(  
   CD2DPointF point  
);  
```  
  
#### Parameters  
 `point`  
 The end point of the line to draw.  
  
## Requirements  
 **Header:** afxrendertarget.h
