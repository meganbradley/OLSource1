# Compiler Error C2019
expected preprocessor directive, found 'character'  
  
 The character followed a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder> sign but it is not the first letter of a preprocessor directive.  
  
 The following sample generates C2019:  
  
<CodeContentPlaceHolder>0\</CodeContentPlaceHolder>  
 Possible resolution:  
  
<CodeContentPlaceHolder>1\</CodeContentPlaceHolder>
