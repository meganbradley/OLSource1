# CFile::operator HANDLE
Use this operator to pass a handle to a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder> object to functions such as [ReadFileEx](http://msdn.microsoft.com/library/windows/desktop/aa365468) and [GetFileTime](http://msdn.microsoft.com/library/windows/desktop/ms724320) that expect a <CodeContentPlaceHolder>2\</CodeContentPlaceHolder>.
