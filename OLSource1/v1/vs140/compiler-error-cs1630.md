# Compiler Error CS1630
Invalid option 'option' for /errorreport; must be prompt, send, queue, or none  
  
 The command line option [/errorreport](../vs140/-errorreport--csharp-compiler-options-.md) must be followed by **prompt**, **send**, **queue**, or **none**, specifying what action you want taken when an internal compiler error occurs.
