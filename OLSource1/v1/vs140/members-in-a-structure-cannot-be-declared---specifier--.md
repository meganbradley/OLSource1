# Members in a Structure cannot be declared &#39;&lt;specifier&gt;&#39;
You have used a specifier that is not valid on a member within a <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> statement.  
  
 **Error ID:** BC30435  
  
### To correct this error  
  
-   Remove the specifier.
