# CMFCPropertyGridProperty::GetRect
Retrieves the bounding rectangle of a property.  
  
## Syntax  
  
```  
CRect GetRect() const;  
```  
  
## Return Value  
 A [CRect](../vs140/crect-class.md) object that describes the bounding rectangle.
