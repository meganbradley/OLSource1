# Compiler COM Global Functions
**Microsoft Specific**  
  
 The following routines are available:  
  
|Function|Description|  
|--------------|-----------------|  
|[_com_raise_error](../vs140/_com_raise_error.md)|Throws a [_com_error](../vs140/_com_error-class.md) in response to a failure.|  
|[_set_com_error_handler](../vs140/_set_com_error_handler.md)|Replaces the default function that is used for COM error-handling.|  
|[ConvertBSTRToString](../vs140/convertbstrtostring.md)|Converts a <CodeContentPlaceHolder>0\</CodeContentPlaceHolder> value to a **char \***.|  
|[ConvertStringToBSTR](../vs140/convertstringtobstr.md)|Converts a **char \*** value to a <CodeContentPlaceHolder>1\</CodeContentPlaceHolder>.|  
  
## END Microsoft Specific
